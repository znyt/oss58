　　胡梦浩和卫安不再被人打扰，少尉完成任务后也快速离开，来到了情报部。
<br/>　　这时候本该少数很少的狼牙情报部却恢复了以前的热闹，并且灯火通明，这里不少人都是连夜从段氏基地赶回来的，全部都在埋头工作。
<br/>　　陆玟丞紧皱着眉，没有去观望狼牙研究部的情况，虽然他知道过了今天狼牙基地将再次迎来大批人员入驻，但是眼下却有更重要的事。
<br/>　　“结果分析出来了！”忽然一个人出声，快速将一份结果递给陆玟丞，他的额头上满是汗水，显然是高度集中工作了很久。
<br/>　　陆玟丞立即接过来一看，忍不住大惊：“阴阳谷？”
<br/>　　那人擦了擦汗，道：“只能是阴阳谷，所有分析的结果都指向了阴阳谷，并且除了肖家的情况不知道，其余神秘家族的三家，全部派人去了阴阳谷！”
<br/>　　陆玟丞一下子觉得眼前一黑，惊悚道：“阴阳谷到底发生了什么，所有人都要去？长官也在那里！”
<br/>　　四周众人都倒吸一口凉气，作为情报部成员他们都极其敏锐，直觉告诉他们阴阳谷有大事发生。
<br/>　　“来人，联系狼牙战团！”陆玟丞当机立断的下令。
<br/>　　——————
<br/>　　阴阳谷之阴谷深处，高家的二十名精英很快找到了高家旧址，灰兔在带完路后跑的无影无踪，很快消失在众人面前。
<br/>　　“这兔子？”一名高家人迟疑道。
<br/>　　领队的人却双眼直勾勾的看着前方：“先别管这些土著生物，先占领旧址！”
<br/>　　哗啦啦！
<br/>　　二十名高家人立即前进，很快便全员进入高家的旧址内，眼前的迷雾顷刻间消失，整个阴谷的禁制也解除了一重。
<br/>　　悄然无息的改变在整个阴谷发生着，除了在水族旧址内的水族圣女，没有人察觉。
<br/>　　金色的鱼尾在溪水中摇摆着，弹起好几朵水花，晶莹的水滴被抛起又落下，在阳光下发着亮。
<br/>　　远处的水族族长鱼勇男、水族长老及几名水族成员，就站在那里看着，他们欲言又止，却又不敢靠近打扰鱼柔女的清闲时光。
<br/>　　自从进入阴谷以来，鱼柔女对他们而言就越来越陌生，越来越不像那个他们看着长大的女孩，随着时间的推移，或者更准确的说是随着阴谷的禁制解开的越多，鱼柔女就越来越不是鱼柔女。
<br/>　　而是圣女。
<br/>　　尤其是今天，鱼勇男等人不知道发生了什么，只觉得眼前的那条巨大的金色鱼尾好似比之前更耀眼更大，有种让人不敢直视的神圣感。
<br/>　　鱼柔女整个人也好似被渡上了一层金光，她就这么优哉游哉的趴在木栏上，整条鱼尾都在水中，画面妙不可言中却给了水族等人莫大的压力。
<br/>　　“你们觉得她像谁？”水族长老忽然揉了揉眼睛问道。
<br/>　　几名水族成员都没有说话，他们答不上来，但是他们知道绝对不像鱼柔女。
<br/>　　鱼勇男知道的多一些，之前一直没想通，只觉得妹妹越来越陌生，但今天经过长老的一提醒，他便顿时醒悟过来，整个人大惊失色的捂住嘴。
<br/>　　“我差点叫出来！”惊慌过后的鱼勇男小声道，瞪大了眼睛满是不可置信：“我是眼花了还是什么，我怎么觉得今天妹妹特别像……”
<br/>　　“嘘！”长老忽然打断鱼勇男，心有余悸的说道：“别再叫她妹妹，她不是你的妹妹。”
<br/>　　“圣女。”鱼勇男连忙改口。
<br/>　　这时的水族长老一双浑浊的双目闪过一道精光，意味深长的说道：“这个时代到底有什么不一样，末世降临，肖家灭门，水火回归，初代显灵……”
<br/>　　说着，长老闭上了眼，摇摇头转身往回走。
<br/>　　“长老？”鱼勇男追上去，有些手足无措。
<br/>　　“千万不要冒犯她。”长老又留下了一句话。
<br/>　　“圣女？”鱼勇男确认的问道。
<br/>　　“圣女，真正的圣女。”
<br/>　　——————
<br/>　　阳谷内，带着潮·湿热气的黑暗隧道里，楚云天步步艰难的走着，他一直在往前走，不知道走了多久也不知道走了多远。
<br/>　　只有一条通道一直往前往下，作为普通人的他不像前面的人能够快步前进，只能用普通人的脚力在这巨大的阳谷内摸索。
<br/>　　食物和水一点点减少，已经支撑不了多久，楚云天有很好的管理自制能力，根据食物的剩余数量能够大致摸索出他已经走了三天。
<br/>　　吃喝睡都在通道里，暗无天日。
<br/>　　当他又继续走了半天，将最后一餐食物解决掉后，楚云天有一瞬间的放弃念头，如果前方的道路还是长长的通道，那他可能就会死在这里。
<br/>　　这与待在一处地方不动不一样，不断的往前走，消耗了他大量的体能，哪怕每日用餐依旧只是维持生命的计量，也依然让他消耗过多。
<br/>　　只是在他冒出了放弃念头之时，前方漆黑一片的地方忽然变得宽广起来，楚云天停下脚步，小心翼翼的往前探了探。
<br/>　　那是一个巨大深坑！
<br/>　　笔直的通向地下不知道多深的地方，宽度更是惊人，弧度摸起来趋近于平缓！
<br/>　　这得多大？
<br/>　　大到让楚云天一开始还以为前方是悬崖！
<br/>　　没有灯光全部一片的漆黑，而楚云天只有普通人的视觉，根本无法在这里看到什么，他想要从旁边寻找边缘地带，却发现通道过来只有这个深坑，没有其他路。
<br/>　　“跳下去？”他愣住，紧接着将随身的一件重物抛下。
<br/>　　没有声音，过了很久都没有落地的声音传来，楚云天摇摇头，这种高度哪怕是新人类跳下去也会死。
<br/>　　就在他无计可施时，身后忽然传来一阵脚步声，飞快的靠近！
<br/>　　楚云天的心脏一下子提到了嗓子眼，他回头望去，却依然在一片黑暗中什么都看不到。
<br/>　　这时候脚步声越来越近，甚至已经走到了他的近前，不是一个人，脚步声并不快速急促，速度却惊人的一下子就来到了楚云天所站的地方。
<br/>　　来者是新人类，高阶新人类！
<br/>　　“一个老头。”一个男子的声音响起，听起来很年轻，还带着一丝惊讶：“好像是个普通人，怎么混进来的？”
