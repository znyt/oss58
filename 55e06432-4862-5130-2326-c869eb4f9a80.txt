        “因为，大少爷下班回来后，若是肚子饿了，可以一口气吃上两大碗饭！”刘婶笑着说。
<br/>
<br/>        童筱颖忍俊不禁道：“大哥原来也有小孩子的一面。”
<br/>
<br/>        “你怎么还叫大少爷为‘大哥’啊？大少奶奶，你应该叫他‘老公’才对。”刘婶打趣道。
<br/>
<br/>        童筱颖抿了抿唇，尴尬地笑了笑：“还不大习惯。”
<br/>
<br/>        刘婶很理解地点了点头，笑着说：“大少奶奶，你就放心吧！大少爷一定会好好珍惜你的。他的性子，我最清楚了。他跟他妈妈很像，认定了，就是一辈子。”
<br/>
<br/>        “他真的值得我去托付终身吗？”童筱颖看了一眼刘婶，试探性地问道。
<br/>
<br/>        刘婶微微一笑，笃定地回答道：“那当然！没有谁比大少爷更值得了！”
<br/>
<br/>        “刘婶，你就不怀疑我嫁给大少爷，是别有用心吗？”童筱颖开玩笑似地说。
<br/>
<br/>        她若是没记错的话，傅夜沉身边的那个助理的秘书，还怼她，说她是为了报复，才嫁给大哥的。
<br/>
<br/>        “我猜啊！不是你要嫁给大少爷的，恐怕是大少爷自己强娶你回来的。”刘婶笑道。
<br/>
<br/>        童筱颖忍不住也跟着笑了起来，然后跟刘婶说起那天傅夜沉派人将她从高铁站强行绑到民政局里，一边向她求婚，还顺便把结婚证给领了的事情。
<br/>
<br/>        刘婶从始至终都是说傅夜沉的好，就没有一句是坏话，还有更多的是劝她早些放下二少爷傅昼景那个旧爱，接受大少爷傅夜沉这个新欢。
<br/>
<br/>        每一次一到这个话题上，童筱颖只是在强颜欢笑。
<br/>
<br/>        她现在，需要的是时间。
<br/>
<br/>        刘婶也只是说到点上，就不再多说，怕她心里反感。
<br/>
<br/>        为了让傅夜沉中午能吃好，童筱颖还另外炒了一荤一素的菜，外加一份小点心，一起搭配着刘婶炖的鲍鱼汤。
<br/>
<br/>        饭菜盛入保温饭盒里后，童筱颖提着饭盒，在卧室里取了傅夜沉的车钥匙，开着他留在私宅里的备用车，去了傅氏集团。
<br/>
<br/>        傅氏集团大厦位于市中心的黄金地段，高耸的大厦遮住了正午的阳光。
<br/>
<br/>        童筱颖将车停入了大厦前坪的停车场里。
<br/>
<br/>        下车后，童筱颖提着保温饭盒，迈开脚步朝着大厦的正大门那边走去。
<br/>
<br/>        “小姐你好，请问你有什么事吗？”前台小姐挂着职业笑容，看向刚从门卫保安那儿签了出入记录来到她跟前的这个女人。
<br/>
<br/>        童筱颖上一次是被傅夜沉抱着从地下室坐电梯直达总裁办，这次，她一个人，没法坐那台电梯。
<br/>
<br/>        而且，她虽然三年前在这儿工作过，但这三年的时间里，她未触及过这栋大厦的人，一般大公司里的人员变动得也很快，所以她并不认识这位新来的前台小姐。
<br/>
<br/>        自然而然，这位前台小姐也不认识她。
<br/>
<br/>        没有工作证，她无法上楼，所以，她不得不来找这位前台小姐。
<br/>
<br/>        童筱颖看着前台小姐，微微一笑：“我找傅夜沉。”
<br/>
<br/>        谁知她声音刚落，四周来来往往的员工，纷纷向她投来了异样的目光。
<br/>
<br/>        前台小姐看着童筱颖的目光也已经带了一分鄙夷。
<br/>
<br/>        瞧着这位小姐的穿着打扮，一身高档品牌，还是限量款，就说明她应该是哪家的千金名媛。
<br/>
<br/>        既然是千金名媛，又岂会如此冒冒失失地来找傅总呢？
<br/>
<br/>        前台小姐很习以为常地将童筱颖归纳到傅夜沉的仰慕者当中，而这样的女人，在他们傅氏集团里，是极其不受欢迎的。
<br/>
<br/>        “请问你有预约吗？”前台小姐没好气地问道。
<br/>
<br/>        预约？！
<br/>
<br/>        童筱颖眨巴着眼睛，心想着要给傅夜沉打电话时，掏出手机后才意识到，她好像没有存大哥的号码。
<br/>
<br/>        去了澳洲后，傅昼景有给她换了新手机，以前的旧手机里倒是有大哥的手机号码，只不过她当时换手机的时候，心想着以后都不会再联系大哥了，就没有把大哥的手机号码复制过来。
<br/>
<br/>        这下，她觉得自己有些尴尬了。
<br/>
<br/>        童筱颖嗫嚅半晌，低声说：“没有。”
<br/>
<br/>        听到这个回答，前台小姐更加确定了自己的猜测。
<br/>
<br/>        她脸上挂着有礼而公式化的微笑，语气却十分冷淡地对童筱颖说道：“不好意思，小姐，如果你没有预约的话，我是不能让你进去的。”
<br/>
<br/>        “若是上不去的话，岂不是枉费我今天特意给他做了午餐。”童筱颖手里拎着保温饭盒，柳眉微蹙，不满地小声嘟囔道。
<br/>
<br/>        童筱颖本就长得很漂亮，虽然是素颜，但不施粉黛也天生丽质。
<br/>
<br/>        她后脑勺束着长及腰际的马尾，随着她摆头的动作，在身后轻轻地摇曳，为她那玲珑有致的身段增添了几分青春的气息。
