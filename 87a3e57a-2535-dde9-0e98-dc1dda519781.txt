        她童筱颖可以憋得住，他傅夜沉却熬得心里发慌。
<br/>
<br/>        他不懂，她为什么突然冷着他，而她却完全不给他任何跟她说话的机会。
<br/>
<br/>        好不容易等到他熬到了极限，想主动找她说话的时候，她却再次拖着行李箱从房间里走出来。
<br/>
<br/>        “昼景回来了，这段日子，多谢大哥的关照。”童筱颖在房门口遇上傅夜沉时，本不想理会他，但又考虑到自己这段时间住在这里，毕竟是承蒙了他的照顾，于是礼貌性地谢道。
<br/>
<br/>        傅夜沉嘴巴张了张，刚准备说什么的时候，放在裤袋里的手机振动了起来。
<br/>
<br/>        童筱颖瞄了一眼“滋滋”声的来源，再次看了一眼傅夜沉，向傅夜沉欠了欠身，手下拽着行李箱，转身往楼梯口那边走去。
<br/>
<br/>        傅夜沉并未接听电话，而是默默地目送童筱颖的离开。
<br/>
<br/>        童筱颖走出了别墅大门，下了台阶，傅昼景的车就停在了台阶下的前坪里。
<br/>
<br/>        当他看到童筱颖出来，立即疾步上前去帮她提行李箱。
<br/>
<br/>        “大哥没在家吗？”傅昼景接过行李箱后问道。
<br/>
<br/>        童筱颖不假思索地回答：“他在家。”
<br/>
<br/>        “那就怪了，他怎么不接我电话。”傅昼景纳闷地说道。
<br/>
<br/>        童筱颖这才想起，刚刚傅夜沉的手机在振动，兴许是昼景打过来的。
<br/>
<br/>        “可能他没听到吧！”童筱颖淡淡地说。
<br/>
<br/>        傅昼景微微一笑：“那我们先回家，回头，我再给大哥打电话。”
<br/>
<br/>        “嗯！”
<br/>
<br/>        “筱颍，以后我若不在家，你也不用再来大哥家住了。”
<br/>
<br/>        “嗯？”
<br/>
<br/>        “明天，我们搬家去滨江城最好的别墅区——‘花园别墅区’住，那里有保安二十四小时巡逻，很安全。而且，我给你请保姆，筱颍以后不用再出去工作，也不用再做任何家务。”傅昼景接着说道。
<br/>
<br/>        童筱颖不禁一脸好奇地问：“为什么突然要搬家？而且，还请保姆？”
<br/>
<br/>        “我接了个审计业务，别墅是对方送的，我写的你的名字。”傅昼景微笑道。
<br/>
<br/>        童筱颖听后，却只是在干笑，她不明白自己为什么无法发自内心地去笑。
<br/>
<br/>        “从今以后，我的筱颍，只负责貌美如花，我负责赚钱养家。”傅昼景伸出手来，将童筱颖抱入了怀中。
<br/>
<br/>        童筱颖抬起双手来，抱了抱傅昼景，心里却在莫名其妙地心虚。
<br/>
<br/>        她心虚的地方是，最近，昼景虽然忙，但是他赚的钱……
<br/>
<br/>        毕竟，能在这么短的时间内，还清一部分巨额债务，还能另外拥有一套别墅，除非他是一个收益与日俱增的大公司的老板。
<br/>
<br/>        显然，昼景不是……
<br/>
<br/>        除此以外，童筱颖还发现，昼景换了一台的豪车。
<br/>
<br/>        她虽然认不出这台车的牌子，也不知道这台车的价格，但这台车看起来似乎挺贵的样子。
<br/>
<br/>        童筱颖闭上了眼睛，不再让自己胡思乱想，她只想贪恋昼景怀里的这一丝温暖。
<br/>
<br/>        连华生的实验室里。
<br/>
<br/>        傅夜沉站在窗户边，看了许久。
<br/>
<br/>        也就只有连华生的这间实验室的落地窗前，才能将楼下前坪的动静看得一清二楚。
<br/>
<br/>        连华生一边做实验，一边调侃道：“阿沉，就算你望眼欲穿，大小姐也是你弟弟的老婆。我劝你，你还是别想了！”
<br/>
<br/>        “阿景换了一台价值五百万的豪车。”傅夜沉若有所思地说道。
<br/>
<br/>        连华生难以置信地抬眸瞥了傅夜沉一眼后，又继续淡定地做着自己手中的实验，口无遮拦地念叨道：“你弟弟是在替你打工吧！年薪应该不过百万，却能买到五百万的豪车。要么他在省吃俭用地存钱买车，要么他还有另外的你不知道的钱财来源。”
<br/>
<br/>        “如果，阿景坐牢了，筱颖会怎么办？”傅夜沉跟连华生说得上文不接下文。
<br/>
<br/>        连华生自然知道傅夜沉的重点不在傅昼景身上，于是顺着傅夜沉的心意回答道：“以我对大小姐的理解，以大小姐那倔强的性子。就算你弟弟坐牢了，大小姐哪怕再苦再累，她也会心甘情愿地等他一辈子。”
<br/>
<br/>        “所以，你觉得筱颖是真的很爱阿景吗？”傅夜沉面无表情地反问，但他的心病又犯了。
<br/>
<br/>        每一次，一想到筱颖爱的人是阿景，他的心就会隐隐作痛。
<br/>
<br/>        连华生咧嘴一笑：“这不是明摆着的事吗？她若不爱你弟弟，又为何会嫁给你弟弟？她若不爱你的弟弟，就不会在每次提及你弟弟的时候，笑得特别灿烂。不管我们如何说你弟弟的坏话，她可是一心护着你弟弟，永远都站在你弟弟那边，替你弟弟着想。”
