        &lt;h3&gt;第三卷第三百九十九章  人生如戏全靠演技&lt;/h3&gt;
<br/>
<br/>        寒伯安约了陆宇霆在上海最豪华的酒店用餐。
<br/>
<br/>        这一次多了俩个人，宫阳和苓希。
<br/>
<br/>        陆家和苓家果然有来往。
<br/>
<br/>        看吴润竹和苓希的关系似乎很亲密，俩个人好像认识的时间不短了，但苓希好像有些害怕吴润竹。
<br/>
<br/>        在他们这几个人中除了宫阳都认为李长卿和寒伯安的关系非同一般，宫阳不知道他们是怎么想的，但是宫阳能看出来寒伯安和李长卿在演戏。
<br/>
<br/>        寒伯安没有带安男来，宫阳就明白了他们的用意，寒伯安和李长卿都在变相的保护安男。
<br/>
<br/>        宫阳在，李长卿发挥不出来他的演技，他怕宫阳有别的想法，倒是宫阳将几个人的注意力完全的吸引到了她们三个女人的身上。
<br/>
<br/>        当几个人就餐完毕之后，宫阳说：“今天让寒总破费了，我晚上请大家一起去嗨歌。”
<br/>
<br/>        “你知道在座的都是什么人吗？你要请大家一起去嗨歌？”寒伯安笑着问宫阳。
<br/>
<br/>        “唉，什么身份我不清楚，我只清楚和我能嗨到一起，吴姐姐你说对不对？”宫阳笑着问吴润竹。
<br/>
<br/>        “你说的都对！”
<br/>
<br/>        “宫阳，你要真是请我们，我怕你要破产，就你那点工资不够塞牙缝的。”寒伯安看着宫阳。
<br/>
<br/>        寒伯安说得没错，宫阳的工资不高，况且他们家的家庭条件也一般，哪里有钱去那些高档的会所，像吴润竹和陆宇霆夫妇岂能进普通的会所？
<br/>
<br/>        一句话引起了吴润竹和苓希的好奇，他们还不知道宫阳是干什么的，以为宫阳也是哪个世家的姑娘。
<br/>
<br/>        “宫阳是做什么工作的？”吴润竹好奇的看着宫阳。
<br/>
<br/>        “我在某警、局上班，工资其实不高，但是我真的想请你们一起坐坐，我今天好不容易见到吴姐姐和苓希。”宫阳有些无奈，有钱人的生活她不懂。
<br/>
<br/>        “啊，宫阳，没有看出来啊，你还是警、察，学过格斗吗？”苓希的好奇心起来了。
<br/>
<br/>        “格斗什么啊，我就是在某部门的文员，你说的格斗什么的我都不会。”
<br/>
<br/>        “怎么可能，长卿都打不过她。”寒伯安又开口了。
<br/>
<br/>        “寒总，你一定是故意的，难道就因为我喜欢过长卿你就这样对我吗？”宫阳撅着嘴。
<br/>
<br/>        “不是，是我听说过苓希的身手也不错，你们俩人可以好好的切磋切磋。”寒伯安笑了。
<br/>
<br/>        “当真？”宫阳欢天喜地的看着苓希。
<br/>
<br/>        “练过，要不要一起切磋一下？”苓希倒是没有犹豫。
<br/>
<br/>        “这个，我........”
<br/>
<br/>        “宫阳，试试，如果你赢了，你可以提条件给我，就算是你要长卿，我也会把他送给你，输了嘛，你就今晚请我们去会所唱歌，喝酒！”寒伯安长臂一挥，搭在了宫阳的肩膀上。
<br/>
<br/>        吴润竹和陆宇霆看着寒伯安和宫阳的互动，都微微尴尬的笑了一下。
<br/>
<br/>        “你们不要介意，这才是他的本性暴露了，像那种前后任女朋友相见的修罗场几乎隔几天都会能发生，这些个女子无非就是贪图他的钱财。”苓希嗤之以鼻。
<br/>
<br/>        “你错了，我不但贪图他们的钱财，我还贪图他们的身子。”宫阳看了一眼李长卿，李长卿正一脸哀怨的看着她和寒伯安。
<br/>
<br/>        “哈哈哈，你这个个性我喜欢，要不，你就直接来我身边上班算了，近水楼台先得月啊，说不定我们就被你拆散攻陷了。”寒伯安的举动就更亲昵了，将宫阳搂了楼。
<br/>
<br/>        李长卿心在滴血。
<br/>
<br/>        寒伯安看了一眼李长卿，伸手拉过李长卿，对宫阳说：“我把我最亲近的人送给你怎么样？”
<br/>
<br/>        “说得比唱的好听，谁作证？”宫阳不屑一顾。
<br/>
<br/>        “你吴姐姐啊，苓希和陆总啊，他们都可以作证，前提是你可以赢了苓希，我不但将他送给你，给你一百万，我还可以聘请你来我公司，并且给你年薪百万，怎么样？”寒伯安诱惑着宫阳。
<br/>
<br/>        “苓希？”宫阳看着苓希，一脸求助。
<br/>
<br/>        “试试吧，宫阳，说不定你就赢了苓希呢。”吴润竹笑眯眯的对宫阳说，她看出来了，寒伯安对宫阳很感兴趣，她也对宫阳感兴趣。
<br/>
<br/>        “好吧，我就试试，年薪百万呢，我几十年都赚不到。”宫阳咬咬牙，这个数字太诱惑人了。
<br/>
<br/>        苓希和吴润竹相视一眼，吴润竹点点头。
<br/>
<br/>        “寒总，那就找个地方让他们切磋一下？”吴润竹笑着问寒伯安。
<br/>
<br/>        “长卿，让司机开车送我们回家，家里地方大，够她们俩人折腾的了。”寒伯安吩咐李长卿。
<br/>
<br/>        吴润竹听了寒伯安的话，心里一喜，寒伯安这是将他们当成了朋友看，才让他们去他家里，不错，这一次来上海值了。
<br/>
<br/>        寒伯安直接带着一行人到了他的庄园，亭台楼阁，花鸟和鸣，家里的佣人送上了香槟和清茶，寒伯安笑着对陆宇霆和吴润竹说：“陆总、吴总，请，我们今天来欣赏一下两位美女的对战。”
<br/>
<br/>        李长卿：“在我左边是代表寒家队的宫阳小美女，在我右边的是代表陆家队的苓希小美女，正义与邪恶已经开始较量，我们拭目以待看看谁是最后的大赢家！”
<br/>
<br/>        “去，还正义与邪恶呢，你以为你是蜘蛛侠呢？”寒伯安踢了李长卿一脚。
<br/>
<br/>        “那总得有说法吧，吴姐姐你说对不对？”李长卿闪身到了吴润竹的身边，避开了寒伯安的那一脚。
<br/>
<br/>        “就是一个小小的乐子，你搞得好像两军对垒。”吴润竹也笑着说。
<br/>
<br/>        “你看她们俩个人挤眉弄眼的，不给她们一点压力，她们不好好表现，那样就不好看了。”李长卿摇头晃脑。
<br/>
<br/>        陆宇霆摇了摇头，轻声对吴润竹说：“你看寒伯安和李长卿还是小孩子心性，竟然如此贪玩。”
<br/>
<br/>        “这样才好，你要是和那些老狐狸打交道会累死你。”吴润竹微微一笑。
<br/>
<br/>        宫阳和苓希已经你来我往，看得他们眼花缭乱，李长卿拍着手跺着脚：“阳阳小姐姐，加油，加油。苓希小姐姐，加油加油。”
<br/>
<br/>        “你到底给谁加油？”吴润竹看着蹦蹦跳跳的李长卿笑着说。
<br/>
<br/>        “管她呢，他们俩个人我谁都不能得罪，谁赢了对我都一样。不过我觉得宫阳会赢！你看，宫阳快赢了！”李长卿继续拍着手。
<br/>
<br/>        “吴姐姐，问你一个悄悄话，你认识甸城谷家的人吗？沧岚雪？”李长卿悄悄的问吴润竹。
<br/>
<br/>        “嗯，宫阳真的赢了。”吴润竹并没有回答李长卿的话，而是指着场上的俩个人。
<br/>
<br/>        “宫阳小姐姐，快来我怀里，来我怀里！”李长卿大笑着跑过去抱起了宫阳，寒伯安可是将他送给宫阳了。
<br/>
<br/>        “李长卿你给我过来!”寒伯安大喊李长卿。
<br/>
<br/>        李长卿放下宫阳，拉起苓希和宫阳的手，笑着对寒伯安说：“一百万我们三个人分了，哈哈哈！”
<br/>
<br/>        李长卿心花怒放，他家宫阳确实很厉害有没有？
