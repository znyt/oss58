　　就在段江伟抚摸着腰间太刀喃喃自语的时候，除却他外空无一人的屋内，忽然一个声音响起：“老天开了什么玩笑？”
<br/>　　段江伟心跳骤然一顿，声音响起在他身后，让他整个身躯都僵硬·起来，更是好似寒芒在背一般充满危险！
<br/>　　他没有第一时间回头，而是用他八阶巅峰的五感，感受着自己身后，他确定无论是之前还是现在，都感受不到自己身后有人！
<br/>　　段江伟闭上眼眸，隐去了眼中的无力感。
<br/>　　对方是在是太强大了，强大到来到自己近身，他却毫无所觉，甚至就在自己如此近的距离说话，他也感受不到任何人的气息靠近。
<br/>　　而这个声音……
<br/>　　段江伟上一回听到还是一年前，在初获这柄紫色太刀的时候。
<br/>　　他转身，望向了这位不请而来的神秘人。
<br/>　　此时站在段江伟面前的，是一个女人，一个没多少人见过，却神出鬼没的女人。
<br/>　　“好久不见。”段江伟开口，平淡的声音中隐藏着某种忌惮和恐惧：“高大小姐。”
<br/>　　来者，正是神秘家族高家的三大嫡系之一，高漫秋。
<br/>　　高漫秋依旧一身典雅至极的贵气华服，淡然的笑容好似千年不变，周身的气场充满不容侵犯的尊贵。
<br/>　　她就这么大方的站在段江伟面前，与之直视：“这一年段氏基地的发展，很可观。”
<br/>　　段江伟微微低头：“承蒙当初高家的赞助，让段氏基地一举进入十大基地之一。”
<br/>　　高漫秋轻轻一笑：“从金阳基地覆灭开始，十大基地倒的倒，乱的乱，就连上京基地当下都自顾不暇，也就为数不多的几家得以幸免，但却也大多保守停驻不前。”
<br/>　　段江伟洗耳恭听，虽然他心里很清楚高漫秋所说的那些基地是哪几个，但却不明白接下来这个神秘的女人会说什么。
<br/>　　“不过狼牙除外。”出乎段江伟的意料，高漫秋一上来就直接将狼牙摆在了明面上。
<br/>　　段江伟豁然抬头，眼中满是惊异：“您的意思是？”
<br/>　　高漫秋面上依旧带着淡笑：“上京乱，金阳灭，南都守，天阳颓，川域和段氏发展稳固都是因为背后有人，而狼牙则是异军突起，并且自始至终没有任何一家神秘家族伸手。”
<br/>　　段江伟瞳孔缩了缩，没有说话而是选择继续沉默。
<br/>　　“说起来，当初我来找你的时候，正当处于我名声最差之时。”高漫秋忽然说起了另一件事，面上的笑容也忽然变幻了味道：“你倒是一直很恭敬，难道不好奇传闻中的那些事么？”
<br/>　　高漫秋所说的传闻，正是白忧为了退婚，而让叶子博四处传播有关高漫秋臭名的那些，相当多的一部分都不堪入耳，但同时也引发了很多人对这个女人的臆想。
<br/>　　当她这么大大方方说出来的时候，段江伟不但没有任何心头雀跃感，反而恐惧感更深。
<br/>　　当他第一次见到这个女人，知道对方的真实身份时，就有了这种恐惧感。
<br/>　　因为这一点，与楚涵太像了！
<br/>　　楚涵臭名昭著，什么样的骂名都有，但段江伟与楚涵的接触次数可不少，很清楚楚涵此人本身与传闻中的差别之大。
<br/>　　这便是段江伟一直忌惮的地方。
<br/>　　高漫秋同样如此，至始至终她由内而外散发出来的气场，都与传闻太不相同。
<br/>　　这种反差往往映照着某种阴谋，与楚涵的状况不谋而合！
<br/>　　但段江伟对楚涵是忌惮，对高漫秋却是恐惧，因为两者之间的身份地位不是一个层次，更别说一年前与现在一样都让段江伟倍觉惊恐的可怕战力。
<br/>　　深不可测到无法试探！
<br/>　　看到段江伟没有说话，高漫秋自顾自的一笑，而后眯起眼睛望向窗外：“楚涵的反应就跟你们都不一样。”
<br/>　　此话一出，段江伟心脏一提，紧接着就剧烈跳动起来。
<br/>　　“他不但一眼就认出了我，还就着我的身份演了一出戏，小小的利用了我一把。”高漫秋说起这件事，嘴角挂着轻笑，而后微微偏头看向段江伟道：“但随后，他看我的眼神，就像是在看某个目标，好像我是一座山，他早晚会越过去。”
<br/>　　段江伟的心头涌·出惊涛骇浪，此时此刻的高漫秋当着他的面说出这番话，意欲何为？
<br/>　　“听说他在这次大尸潮一战中昏迷不醒了？”高漫秋看着他，带着深不可测的笑容问道。
<br/>　　段江伟迟疑了一秒，而后点头：“是。”
<br/>　　高漫秋眼底闪过微不可查的一丝·情绪：“原因不明么？”
<br/>　　段江伟不明白高漫秋专程来此，为何会盯着这件事发问，但还是如实回答：“是。”
<br/>　　“哦？”高漫秋眉头一挑，面上荡然而开一个灿烂笑容：“这么说来接下来的尸潮战，你段江伟将一展才华，他楚涵恐怕都参与不了？”
<br/>　　段江伟的眉头越皱越紧，此时此刻他要是还不明白这是离间，他也就白当这几年的基地首领了！
<br/>　　“高家，也与楚涵不和么？”他抽·搐着，问出了这么一句话。
<br/>　　“当然不是。”高漫秋的笑容更加灿烂，甚至在当下还有些温暖：“家弟与楚涵是至交。”
<br/>　　段江伟豁然抬头，眼中充满了迷雾，他承认他完全被高漫秋绕了进去，根本分辨不出这个女人究竟目的是什么。
<br/>　　只是当段江伟抬头后却发现，眼前却空无一人，前一秒还站在他面前的高漫秋，竟然已经消失不见！
<br/>　　“究竟想干什么？”段江伟紧皱眉宇，从未有过如此纠结的情绪。
<br/>　　高家是段氏基地的赞助者，可以说段氏基地有如今的一切，都脱不开高家，如果高家提出要求段江伟根本没有理由拒绝。
<br/>　　但高漫秋又什么都不说，只是来此与他看似闲聊了一番。
<br/>　　真的只是闲聊么？
<br/>　　堂堂神秘家族高家的嫡系大小姐，会专程来此处闲聊？
<br/>　　显然不是为了传达某些意思，就是为了打探什么消息！
<br/>　　段江伟揉了揉太阳穴，想不通也理不顺，但他知道经过高漫秋的这番话，之前好不容易接受了与楚涵公事的心境，有了动摇！
<br/>　　在段江伟不知道的某处屋顶，正要离开这座基地的高漫秋脚步顿住，颇有些意外的看着眼前挡住她去路的人：“这么巧？”
