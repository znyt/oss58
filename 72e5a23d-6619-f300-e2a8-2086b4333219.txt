        &lt;h3&gt;第三卷第二百三十六章  凌辰送宁玥回家&lt;/h3&gt;
<br/>
<br/>        宁玥提出要回家，凌辰是求之不得，他们的工作已经轰轰烈烈的展开了，刘诚之已经到了甸城，说好了他们这边要好好配合刘诚之的，但是因为有宁玥，凌辰他们总感觉束手束脚。
<br/>
<br/>        凌辰又找到董亚子：“我送宁玥回家，我们一起过去吧。”
<br/>
<br/>        “还要我继续当电灯泡？我告诉你，你自己心里是怎么想的你要好好把握，不要想着去借助第三方或者是第三个人。”
<br/>
<br/>        “你什么意思？”
<br/>
<br/>        “我什么意思你当然知道，你不喜欢宁玥，你就要明明白白的告诉她，告诉她你喜欢的人是宁雅，而不是宁玥，不要给她希望。”
<br/>
<br/>        “我有表现我喜欢她吗？我不过是因为她是宁雅的妹妹而对她有所照顾。”
<br/>
<br/>        “可是宁玥不会这么想的，宁玥会认为你是喜欢她的，如果你不明明白白的告诉她，你以后还会有更多的麻烦。我不会陪着你去杭州的。”
<br/>
<br/>        “这么无情的？”
<br/>
<br/>        “不是无情，而是你要明白你自己想要什么，自己去争取，不是依赖别人帮着你，尤其是感情上的事情。”
<br/>
<br/>        凌辰看了董亚子很久，然后郑重的点点头：“谢谢，我明白了！”
<br/>
<br/>        凌辰送宁玥回家，他想要给宁玥说，他喜欢的是宁雅而不是宁玥，但是宁玥并没有对他凌辰说过喜欢他的话，他怎么开口？
<br/>
<br/>        路上，宁玥表示了她到山西的感受，也感谢凌辰对她的照顾。
<br/>
<br/>        “凌辰哥哥，山西真是太好玩了，谢谢你陪了我这么些天。”
<br/>
<br/>        “应该的，因为你是宁雅的妹妹。”
<br/>
<br/>        “只是因为我是宁雅的妹妹？”
<br/>
<br/>        “是啊，宁雅的妹妹也是我的妹妹。”
<br/>
<br/>        “我不是十几岁的小女孩。”
<br/>
<br/>        “在我眼中叫我凌辰哥哥的都是十几岁的小女孩。宁雅只会叫我凌辰。”
<br/>
<br/>        “凌辰，我是宁玥，不是宁雅。”
<br/>
<br/>        “我当然知道你是宁雅的妹妹宁玥，要不然我会对你嘘寒问暖？”
<br/>
<br/>        “我是一个成年人。”
<br/>
<br/>        “大家都是成年人。”
<br/>
<br/>        “你拉过我的手，搂过我的腰。”
<br/>
<br/>        “我拉过手搂过腰的人多了去了。”
<br/>
<br/>        “渣男，你以为你能和宁雅在一起？”
<br/>
<br/>        “能不能在一起不是你说了算。”凌辰突然住口了，他刚刚说了什么？他只想着怎么样把宁玥推出去，却没有想到他说的话会给他带来什么样的后果。
<br/>
<br/>        “哼，我想要的我爸爸都会帮我拿到。”
<br/>
<br/>        凌辰再也不说话了，如果宁雅的父亲阻挠他和宁雅来往，除非宁雅能够脱离宁家，但是宁雅脱离宁家的可能性微乎其微。
<br/>
<br/>        宁玥见凌辰不说话了，她也不说话了，凌辰的心思她当然能看明白，可是当爸爸从陈家回来对凌辰大加赞赏的时候，她就动心了，能让父亲夸奖的人不多。
<br/>
<br/>        母亲也对凌辰很感兴趣：“有这么好的人才，不如让孩子们见见面，说不定就成了朋友了。”
<br/>
<br/>        “他们明天会来家里，让宁玥见见。”宁雅的父亲完全忘记了刘诚之父亲对他说的话。
<br/>
<br/>        也许女人都喜欢那种酷酷的冷冷的男人？宁玥一眼就看上了凌辰，几个人正聊得欢，姐姐宁雅回来了。
<br/>
<br/>        凌辰看姐姐的眼神让她嫉妒，后来刘诚之和董亚子说凌辰是宁雅的男朋友她心里就很不舒服，姐姐这么多年都没有男朋友，刘诚之他们真会开玩笑，爸爸可是让她先见的凌辰。
<br/>
<br/>        凌辰看着冷冰冰，其实心底是很温暖的人呢。
<br/>
<br/>        宁玥被凌辰护送到了宁家。
<br/>
<br/>        宁爸爸宁少天很高兴，凌辰这个孩子倒真的是不错，有始有终，值得深交。
<br/>
<br/>        “凌少，宁玥这孩子一向被娇惯坏了，也只有你才能容忍她，这些天给你添麻烦了吧？”宁少天知道凌辰送女儿回来了，早早下班回到了家里。
<br/>
<br/>        “不，宁玥知书达理，对人热情，脾气也很好，没有给我添什么麻烦。”凌辰微笑着说。
<br/>
<br/>        “哈哈哈哈，到底是年轻人在一起，脾气还能合得来。我这每天都被她给气死了。”宁少天的语气是满满的都是宠溺的味道。
<br/>
<br/>        凌辰心里想，不知道宁爸爸对宁雅是不是也如对宁玥这么好？
<br/>
<br/>        “宁伯伯，宁雅回来了吗？”凌辰心里想着宁雅，嘴里很诚实的就问出来了。
<br/>
<br/>        宁少天一愣：“宁雅？哦，宁雅和你们同一天走的，还没有回来呢。”
<br/>
<br/>        “嗯，她最近还回来吗？”
<br/>
<br/>        “最近也不知道能不能回来，这孩子心大，出门了就很少回家，又去了陈子昂的公司，和陈子昂的关系很好，就更不想回家了，啊哈哈哈哈。”
<br/>
<br/>        “你们和陈子昂谈成合作意向了吗？”
<br/>
<br/>        “还没有呢，陈子昂明天会安排人过来和我们商讨合作意向，这也是我叫宁玥回来的原因，宁玥也应该参与公司的事情了。”
<br/>
<br/>        凌辰心里一疼，宁雅能力那么强，到别人公司去上班了，宁玥明明一点经验也没有就可以参与到公司商务对接，人和人真的没有办法相比。
<br/>
<br/>        “宁玥之前有参加过公司的事情吗？”
<br/>
<br/>        “没有，这一次不是陈子昂的公司嘛，宁雅和陈子昂关系不错，我们之前又聊过，我就是想给宁玥一个锻炼的机会。”
<br/>
<br/>        “嗯，确实应该给年轻人机会，这样才能成长起来，宁玥很聪明的，相信她一定能能够很快能成为伯伯您的得力助手的。”
<br/>
<br/>        虽然董亚子说不愿意和凌辰一起到杭州，但是因为工作的原因，董亚子还是到了杭州，不过没有来宁家，原本她和宁家也不熟悉。
<br/>
<br/>        凌辰坐了一会准备告辞，宁少天说：“明天宁玥会和陈子昂的人商谈合作事宜，晚上我做东，请凌少和陈子昂公司的人一起吃顿饭，凌少务必参加。”
<br/>
<br/>        “哦，伯伯不必客气，你们商务上的事情，我不好参与，改天吧。”
<br/>
<br/>        “据我所知，你和他们公司也有商务往来，我们就一起坐一坐，我晚上安排人过去接你，你是和董亚子一起来的？”宁少天知道董亚子也来了，宁玥给他发了信息。
<br/>
<br/>        “是的，我们最近要视察这边的市场。”
<br/>
<br/>        “那晚上我让宁玥去接你，就这样说定了啊，不见不散。”
<br/>
<br/>        “那恭敬不如从命，谢谢伯伯，晚上见。”
<br/>
<br/>        ........
<br/>
<br/>        宁雅被陈子昂派回家和宁家讨论合作的意向，如果能够谈成，那么他们就会合作，宁家每年会从陈子昂公司走不少的流水，这对宁家来说也是很可观的一笔收入。
<br/>
<br/>        就如陈子昂所说，谁会和钱过不去，在正常合理的范围内，赚取合理的利润，保证双方共同稳定的成长。
<br/>
<br/>        这个社会是一个多元的相互协作的社会，社会的发展离不开各行各业之间的相互协作和互补，陈子昂的公司以研发软件为核心，但是她代理的硬件设备也很多，而宁家是一个生产型企业，有着自己的核心产品，如果双方能够达成合作是互惠互利的。
<br/>
<br/>        宁雅并没有回家，她住在了酒店，现在的她代表的是他们的公司。
<br/>
<br/>        当然，宁家也并不知道陈子昂派过来商谈合作的人是宁雅，宁雅用的是公司的手机号码。
<br/>
<br/>        宁雅到达酒店不久，就接到了宁家打来的电话，邀请晚上一起共进晚餐，吃饭的地方就定在宁雅住的这个酒店，宁雅不想参加晚宴不行了。
