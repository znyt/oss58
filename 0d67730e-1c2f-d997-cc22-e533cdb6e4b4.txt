　　小黄袋中飞出来的是什么，既然是一只又一只的蚊子，不同普通的蚊子，这些袋子中飞出来的蚊子通体白色纹路，每一只都足足有一公分大小。
<br/>　　这些白蚊的口器长而尖锐，可以轻松的刺入人的皮肤之中。
<br/>　　此时，鹰钩鼻男子的眼前足足悬浮了一百只这样的白蚊。
<br/>　　“都去吧宝贝，主人在这里等着你们回来”
<br/>　　鹰钩鼻男子随意的挥了挥手，霎时间，这些白蚊好像能听的懂人言，一个眨眼，既然消失在街道的人山人海中。
<br/>　　飞行的速度非常的快，比普通蚊子不知道要快多少倍。
<br/>　　眼前的一百只白蚊消失人海后，鹰钩鼻男子并没有就此罢休，只见他的手掌一摊开，不知何时，他的掌心中骤然出现了一只飞行小生物，十公分大小，通体银白，月光映照下，既然灿灿生辉，非常的圣洁。
<br/>　　最为明显的是小生物有着一对刺吸式的口器，长长尖尖宛如一把银针插在嘴角上，寒光凛然。
<br/>　　很显然，这又是一只别具一格的蚊子，要是被人看到，一定会惊呼一声:我靠，好大的蚊子。
<br/>　　足足十公分的蚊子还真是非常的少见，最关键的他手中这只蚊子非常的独特，头部，腹部，双翅，三足，都银白条纹覆盖，显得一点都不丑陋，恶心，反而给人圣洁好看的错觉，就好像从动漫中飞出来的。
<br/>　　它跟世人常见的蚊子有着截然不同的区别，不会让人心生反感，甚至想拥有，第一感觉就是漂亮好看，唯一让人触目惊心的就是银白蚊子长长尖尖的口器，坚硬异常，寒光流转，那么长的口器要是刺入人的身体中，那么吸的就不是一点点血液了。
<br/>　　仿佛能将人吸掉半条命去。
<br/>　　就这样，这只十公分银白色的蚊子安静乖巧的趴在男子的掌心中，非常的听话。
<br/>　　鹰钩鼻男子此时的眼神很温柔，就像在看自己的孩子，充满父爱，：“这里是个好地方，满大街的人供你吸噬血液，能吸多少，就看你自己了”
<br/>　　只瞧十公分银白色蚊子缓缓的悬浮在了男子的面前，长长尖尖的口器点了点作为回应，非常的人性化。
<br/>　　如果涂小安看到这只大蚊子，一定会非常的惊讶，因为这只是觉醒类的蚊子。
<br/>　　一只蚊子也能觉醒，这也是日了狗了。
<br/>　　大千世界何其之大，万兽林立，各种品种的生物不胜其多，蚊子不过是世界上最为卑微的一种生物，这种生物甚至是没有自我意识的，生存只不过是出自于本能驱使。
<br/>　　换一句话说，蚊子是没有灵智的，从出生到自然死亡，也不过区区一周，但鹰钩鼻男子眼前悬浮的这只大蚊子却是觉醒类的生物，开启了灵智，它聪明到可以听懂人的语言，甚至还能跟你交流，这是非常不可思议的。
<br/>　　“去吧，好好的去饱餐一顿”
<br/>　　鹰钩鼻男子笑的淡淡开口，眼眸闪过一抹冷然。
<br/>　　他的话音刚落，就见这只十公分的银白蚊子灿灿生辉的飞走了，悄然间的钻进了人海之中，它的体型很大，但飞行的速度更快，好像一抹流光。
<br/>　　区区蚊子，飞行速度快到肉眼都难以捕抓，让人瞠目结舌。
<br/>　　白镇人民依然在节日的欢欣雀跃中，殊不知一场厄运悄然的来临。
<br/>　　这只十公分的银白蚊子飞入人海中，好像鲤鱼如海，嗡嗡嗡的发生兴奋的声音。
<br/>　　因为这满街都是属于它的食物。
<br/>　　此时，一位捕蛇人摆着一个蛇摊，他的旁边有几个铁笼，每个笼中都装着一条不错品种的毒蛇，作为买卖。
<br/>　　今日很多捕蛇人会将自己家中的毒蛇拿出来，在白镇夜市买卖，趁着节日人山人海，赚上一笔。
<br/>　　这位捕蛇人在卖力的吆喝，附近也围过来不少人，他高兴的从笼中拿出一条头大呈三角形，颈细，正背有20个大方斑的毒蛇出来。
<br/>　　“来来来，各位老板看一看，我这条尖吻蝮蛇可是昨日刚从佘山捕抓到的，是成年罕见品种的毒蛇，今日蛇王节，市价五万RMB，今天只要一万，你们就可以带走了”
<br/>　　他手中这条毒蛇其实到很常见，不少人都认的出来，说尖吻蝮蛇有人或许不熟悉，但说五步蛇都没人陌生了。
<br/>　　这种蛇，在本国是非常常见，也是毒性很强的毒蛇。
<br/>　　不过真正按照罕见品种来算，这条蛇到算不上，不是说指它的毒性弱，而是想要这种毒蛇很简单，到处都有的卖。
<br/>　　而且价格也就数千便够了。
<br/>　　显然，这位捕蛇人做起生意，也是有一套。
<br/>　　今天外地人也很多，能宰一个算一个。
<br/>　　围观的也都是一些外地人，一般白镇本地人，对他这种摊位连看的兴趣都没有。
<br/>　　“老板，你一条蛇就要一万，你抢劫啊”
<br/>　　人群中一位年轻人不满的开口。
<br/>　　“小哥，一看你就不是白镇人，我们白镇一条蛇十几万都很常见，我现在只收一万，你还嫌贵啊，这条蛇可是蛇王大人开过光的”
<br/>　　这人也是满嘴的胡说八道。
<br/>　　此言一出，众人发傻的看着他，呆若木鸡。
<br/>　　蛇开光，这是什么鬼。
<br/>　　“切，只听过佛珠开光，我还是第一次听到蛇还能开光的”
<br/>　　怎么滴，将你这条蛇挂在身上，还能消灾除病不成。
<br/>　　他这是在侮辱广大群众的智商呀，要是这话都信了，那也是脑子被驴给踢了。
<br/>　　纷纷的议论声传来，这捕蛇人也是不在意，开始介绍这条蛇的特点跟毒性。
<br/>　　他喊价是一万，但要是有人肯出五千，他还能买一送一，这种蛇在佘山太常见了，可谓不值一提。
<br/>　　捕蛇人双手抱着蛇，不断的喷吐沫，这条五步蛇也是乖巧，连蛇信都不带吐的，非常听话。
<br/>　　但就在此时，这条五步蛇的蛇眸微微冒起了凶光，它蛇眸中见到远处一抹微亮的白光迅速飞了过来，已一个隐晦的方位，不偏不倚的落到了捕蛇人的脖子上。
<br/>　　这位捕蛇人满嘴吐沫的推销自己手中的五步蛇，丝毫没有察觉到什么异样。
