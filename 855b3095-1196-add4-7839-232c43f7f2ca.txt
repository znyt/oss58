　　叶少阳苦笑，看着瓜瓜道：“那你要问他自己。”
<br/>　　望着陈悦紧张而又满怀期待的目光，瓜瓜心软了，而且，被她这样抱着的感觉还挺舒服的，当下笑道：“好吧，我们总算是有缘一场，暂时叫你妈妈好了，将来要是你恢复记忆，你就做我干妈吧。”
<br/>　　陈悦苦笑，将瓜瓜搂紧。
<br/>　　叶少阳无奈摇了摇头，出来跟大伙见面。
<br/>　　大家聚在一起商量了下，目标是京城，这个没有问题，那里毕竟是政治中心，而且之前沈强也说过，九星叠气阵的阵法核心就在京城，想要破阵而出，自然要去京城。
<br/>　　不过在具体意见上，大伙闹了分歧，叶少阳的意思是偷偷过去，出其不意，小马的想法则是一路攻城略地，横扫过去，路上不断攻打州府，让更多的邪物觉醒，壮大他们的势力，至少在人数上可以不吃亏。
<br/>　　最后碧清和腾永清认为，两人说的都有道理，倒是可以分头进行，于是简单分配了一下，叶少阳和碧清一起轻装上阵，偷偷潜入京城，先看看情况，小马则继续当他的大力鬼王，一路杀到京城去。然后双方在京城汇合。
<br/>　　陈悦不想跟那么多陌生人一起，自然还是叶少阳。
<br/>　　叶少阳拿出两张灵符，用自己的血在上面画了符印，交给小马，“这张符半个月内有效，等你到了京城，烧了这张符，然后待在原地不动，我能感觉到你的位置，到时候过去找你。你们当心点，对了，永清你就跟着小马吧，多个帮手，免得路上发生意外。”
<br/>　　当初一起对付七奶奶的时候，腾永清跟小马也相处过，彼此很熟，听叶少阳这么说，欣然接受。叶少阳又跟小马的团队们道了别，约定京城相见，然后彼此分开，赶着两辆马车先走——车夫吓破了胆，不愿跟随，叶少阳一人拍了一巴掌，两个车夫也现出了真身，一个是树妖，一个是黑鱼精。
<br/>　　两人觉醒之后，还想闹事，叶少阳早有准备，将两张画好的灵符分别贴住他们，念咒化解了两人的戾气，两人缓了半天，总算想起了自己是谁——跟别的人一样，都是在山里修行得好好的被抓来的。
<br/>　　叶少阳一扫眼看到陈悦在边上围观，笑着问她：“看到没有？”
<br/>　　陈悦一言不发，回到马车里去。
<br/>　　两人既然觉醒，为了能够回到真实世界，自然也加入了叶少阳的阵营，为叶少阳继续赶马车，一行人继额上路，马车行驶了一天，天黑之前，路过一个驿站，于是停车吃饭。
<br/>　　这几个人里，只有陈悦和叶少阳是吃饭的（猪妖是个特例）。陈悦一个劲给瓜瓜夹菜，瓜瓜又不想让她难过，只有憋着吃了。叶少阳看得想笑。
<br/>　　朱智慧，黑鱼精，还有树妖三人结伴在门外坐着望天，感怀自己可怜的身世，越聊越委屈，最后弄了一壶酒三个人喝起来。
<br/>　　陈悦把瓜瓜拉到自己房间去睡了。因为店小人多，房间不够，给叶少阳四人的只有两间房，让叶少阳意外的是，碧清选择了自己而不是陈悦来同房。
<br/>　　叶少阳回房间的时候，碧清正在泡澡：古代没有浴缸，但是有木桶澡盆，效果也是一样的。
<br/>　　碧清脱光了坐在里头，在热水中浸泡着，俏脸被升腾的热气蒸得微红，平添了几分妩媚。
<br/>　　“你为什么要跟我住，你跟陈悦两个姑娘一起住不是更好吗？”
<br/>　　“她总把我当成弟媳，磨叽的很，我不喜欢，反正我又不用睡觉，睡你的就是了。”
<br/>　　叶少阳无奈，脱下外衣，躺到床上去，目光不自觉地落在她露在水面之外的两个香肩上，笑道：“你知道你现在的样子，让我想起了四个字，知道什么吗？”
<br/>　　“出水芙蓉。”碧清眨了眨眼睛。
<br/>　　“额，”叶少阳摸了摸鼻子，道，“还真不是，我想的是……水煮莲藕，哈哈。挺清淡好吃的一道汤。”
<br/>　　碧清横了他一眼，突然邪魅一笑，“你想吃吗？”
<br/>　　“啥？”
<br/>　　“想不想吃我。”
<br/>　　“我怕被你打。”叶少阳呵呵傻笑。
<br/>　　碧清撇撇嘴，“没情趣。”
<br/>　　叶少阳双手枕在脑后，想着心事，碧清突然开口，问道：“叶少阳，你说这个陈悦，到底是什么来头，要说是邪物，为什么身上没有邪气，也无法觉醒，如果是人类……嗯，那她至少也该有自己的记忆，你说呢？”
<br/>　　“不知道。”
<br/>　　这个问题，也是叶少阳一直想不通的，听陈悦说话什么的，偶尔也会用到几个现代的词，说明八成是跟自己一个时代的人，也是被圣灵会绑来的，但她却没有过去的记忆……
<br/>　　“我总觉得，她的来历肯定非同一般，不是一个普通人。叶少阳，你觉得怎样才能帮她恢复记忆？”
<br/>　　“我哪知道啊，说什么她都不信。”
<br/>　　“不，她是怀疑的，只是内心抗拒吧，因为如果失去了现在的身份，那她就一无所有了——她失去了真正的记忆，如果现在也是假的，你让她相信什么呢？找不到一点寄托。”
<br/>　　叶少阳皱起眉头，仔细想，碧清说得还真挺有道理。
<br/>　　碧清接着分析：“想让她相信这一切是假的，必须给她证据，让她无法反驳的。”
<br/>　　叶少阳耸耸肩，道：“你没法叫醒一个装睡的人，她内心排斥，就算我们所有人都告诉她这是假的，她也选择不信。”
<br/>　　碧清摇摇头，“叶少阳，如果我告诉你，这世界才是真的，你在人间生活的二十多年经历，不过是别人给你灌输的记忆，你信不信呢？”
<br/>　　“开什么玩笑。”
<br/>　　“对啊，因为这记忆显得太真实了，就算你身边所有人一起告诉你那是假的，你恐怕也不会相信吧。”
<br/>　　叶少阳心中一动，认真思索起这个问题，是啊，自己这一路走来的经历……就算所有人都说它是假的，自己也不会相信。
<br/>　　“我明白了，你是说，陈悦现在就是这种情况？”
<br/>　　“没错，她现在的记忆，虽然是别人灌输的，但全都是关于这个世界的，她自然不会轻易相信别人说的。”
<br/>(本章完)
