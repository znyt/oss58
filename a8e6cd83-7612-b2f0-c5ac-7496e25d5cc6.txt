　　轰隆隆！
<br/>　　剧烈的震动，伴随着猛烈的爆响，整个神图空间都在震颤。
<br/>　　所有的人都惊恐不已，无数的苍穹神宫弟子吓的四处逃窜。
<br/>　　苏莫的额头之上，流出了大量的汗水，虚界神图上布置的防御阵法，已经层层的奔溃，他根本阻止不住了。
<br/>　　虚界神图，也到了毁灭的边缘。
<br/>　　一旦虚界神图毁灭，其内的一切存在都将随之毁灭。
<br/>　　苏莫的心中没有恐惧，有的仅仅是悲哀和不甘。
<br/>　　难道就这样死了吗？
<br/>　　父亲、夕儿、朋友，所有的人都和他一起毁灭！
<br/>　　他还有很多事情没有完成，青璇没有救出，芊寻月没有救活！
<br/>　　他实在是不甘心！
<br/>　　就在所有人惊恐不已之时，就在苏莫心有不甘之时，已经千疮百孔的神图空间，突然平静了下来。
<br/>　　“过去了？”苏莫面色一怔，难道地煞之气过去！
<br/>　　其他人亦是神情怔怔，没事了吗？
<br/>　　苏莫意念一探，立刻便探知了外界的情况，地煞之气的确已经过去了！
<br/>　　随即，苏莫稍稍检查了一番神图空间中，并没有人员伤亡之后，便身形一闪，离开了神图空间。
<br/>　　外界，一片虚无毁灭般的景象，苏莫出现之后，一把将虚界神图抓在了手中。
<br/>　　此刻的虚界神图，外表已经有些残破，但并没有遭到太严重的毁灭。
<br/>　　只差一点，若是地煞之气再强一分，怕是虚界神图就保不住了！
<br/>　　苏莫眸光扫视四方，方圆数万里之内已经完全毁灭，变成了一片虚无之地，连下方的海水也消失无踪。
<br/>　　他身处之地的下方，是一眼望不到尽头的深渊。
<br/>　　苏莫叹了一口气，这一次还是没能斩杀巫天御，对方既然敢用出这么恐怖的地煞之气，肯定也有把握逃得性命。
<br/>　　不过，他还有机会，那就是空间通道。
<br/>　　两个月的试炼之期也没多久了，巫天御以及所有巫族的天才，总要离开吧！
<br/>　　只要他堵住通道，就能以逸待劳。
<br/>　　少倾，苏莫离开了此地，向之前的小岛飞去。
<br/>　　不过，他刚刚飞了没多久，便迎到了秦不死三兄弟。
<br/>　　“苏莫，你没事吧？”秦不死急速来到苏莫身前，面带惊疑的问道。
<br/>　　之前的爆炸，即便相隔数十万里，他们亦是看的清清楚楚，感受到了那恐怖的威势。
<br/>　　“没事！”苏莫摇了摇头，沉声问道：“战利品收集了吗？”
<br/>　　“收集了，都在这枚储物戒呢！”秦不死点了点头，递给了苏莫一枚储物戒。
<br/>　　这些资源对苏莫有用，但对于秦不死来说，已经没有半点用处了。
<br/>　　他的修为与资源无关，完全是苍穹世界的限制，唯有离开苍穹世界，他才能突破。
<br/>　　所以，这些资源，对于他来说可有可无。
<br/>　　既然苏莫让他收集，他也没有懈怠。
<br/>　　“那些斩杀的巫族之人，体内神丹有没有收集？”苏莫再次问道。
<br/>　　“没有！”秦不死摇了摇头，神丹在他的理解之中，根本没有任何用处。
<br/>　　“我们走吧！”
<br/>　　苏莫叹了口气，随即返回了小岛所在，将一些巫族天才尸体内的神丹挖了出来，才最终离开。
<br/>　　他的攻击太狂暴，留下的神丹不多，只有不到四十余枚，只有灵剑杀死的人，神丹才能保留下来。
<br/>　　至于先前一剑斩杀两百余人，那些人基本上全部变成了渣，有神丹保留下来的人，寥寥无几。
<br/>　　苏莫和秦不死三人说了一下自己接下来的打算，几人便径直向南域而去。
<br/>　　巫族试炼者从南域降临，必定会从南域离开，通道将在原地开启。
<br/>　　几人一路急行，秦不死和苏莫两人，有一句没一句的聊着。
<br/>　　“那名强大的巫族天才，没有斩杀或擒拿吗？”秦不死问道，他指的自然是巫天御。
<br/>　　“没有，估计让他逃了！”苏莫叹道，不过他绝不会放过巫天御，就算对方交出宏青璇，他也不会放过。
<br/>　　“此人实力超强，估计是巫族不可多得的天才，若是能擒下他，我们的把握便更大三分！”秦不死叹道，他的实力不如巫天御，也就只能靠苏莫了。
<br/>　　“我会尽力的！”苏莫点了点头。
<br/>　　“之前我又擒拿了几名巫族之人，现在我手上已经有十四人了！”秦不死道。
<br/>　　“嗯！”苏莫点了点头，有十余人已经够了，再多也没有多大效果了！
<br/>　　这时，便听秦不死继续道：“其中一人并不是试炼者，三弟说是黄泉魔宗的宗主！”
<br/>　　秦不死沉睡了近千年，根本不知道什么黄泉魔宗，也就是随口一说而已。
<br/>　　但是，此言听在苏莫的耳中，却无异于惊雷炸响。
<br/>　　“什么？你擒拿了巫森？”苏莫面色一怔，立刻停下了身形，满脸的错愕之色。
<br/>　　之前他只顾着杀人，根本就没注意到巫森！
<br/>　　秦不死三人亦是停了下来，光头大汉大声道：“苏莫，的确是巫森，换了一副皮囊，但瞒不过我！”
<br/>　　光头大汉曾经见过巫森，对于巫森的气息很了解。
<br/>　　“呵呵，把他交给我！”苏莫面上露出了一丝冷笑，这真是意外之喜啊。
<br/>　　他没想到巫森居然被秦不死擒拿住了，这简直就是得来全不费功夫啊。
<br/>　　擒获了巫森，就能找到巫山，也能因此救出李峰。
<br/>　　秦不死闻言点了点头，随即手中光芒一闪，空间一阵波动，一名消瘦的老者出现在了其手中。
<br/>　　这名老者全身浴血，整个胸膛都被洞穿了，气息孱弱无比，正是巫森。
<br/>　　“他的战魂和修为都被我封住了，想逃都逃不了！”秦不死道，他也是才知道，苍穹世界除了他之外，原来还有一位武帝，而且是巫族的后人。
<br/>　　“苏……苏莫！”乍一出现的巫森，见到眼前的苏莫，顿时惊骇欲绝。
<br/>　　他可是清楚，自己若是落到了苏莫手中，绝对的有死无生。
<br/>　　“呵呵！”苏莫见此顿时冷笑一声，一把便扣住了巫森的脖子，将对方抓了过来。
<br/>　　“三位前辈，我们稍后再赶路！”手提巫森，苏莫身形一闪，便向远处一座岛屿飞了过去。
<br/>　　他们目前还没有离开西海的范围，无垠的海面上，有不少微型的小岛。
<br/>　　飞上小岛，苏莫一头扎进了一处山林之中。
