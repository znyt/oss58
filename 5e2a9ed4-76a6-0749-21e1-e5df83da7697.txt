　　自从那诡异的丧尸头领出现，不断的一惊一乍让指挥室内众人都手忙脚乱起来，完全不知道该往哪里看，很多人甚至也把那只丧尸头领的移动轨迹跟丢了！
<br/>　　太过于混乱的状况让参谋部的很多人都极其不适应，同时还带着一股羞愧感，这是他们参谋部成立以来参与的各种大小型战役中，节奏最混乱的一次。
<br/>　　战斗的节奏一旦失控，就代表着离失败不远！
<br/>　　这不是楚涵一个人的战争，而是整个狼牙战团的战争，哪怕他已经用了最快速度去追击那只丧尸头领，还不忘带上神之狙击陈少爷来打配合。
<br/>　　但并不是所有人都能跟上楚涵快到极致的反应，若他们与楚涵的节奏相差太大，正常战役也早晚会失去平衡，这时候就算楚涵和陈少爷战胜了那只丧尸首领也无济于事，重要的是整场战役的胜利。
<br/>　　“各司其职！”何峰当机立断的担任起了总指挥官，一道道命令立即下达：“迷宫战区各就各位，参谋部所有人用最快速度分析数据！”
<br/>　　参谋部的成员马上行动起来，早在前两天的持续战斗中，他们便已经有了分配，每个人负责一小块迷宫战区的情况，及时汇报给何峰以便调整。
<br/>　　这时候再次进入这种分工合作，他们很快就找准了自己的位置。
<br/>　　“一区至十区的丧尸已经得到控制，街道清理程度到达了九成！”
<br/>　　“三十一区至四十区的丧尸清理缓慢，需要支援！”
<br/>　　“十一区至二十区的丧尸清理进度到达一半！”
<br/>　　“二十一区至三十区的丧尸清理进度到达六成！”
<br/>　　“……”
<br/>　　听着各个区域的情况，何峰的大脑高速运转，及时调整的命令也随之下达：“十一至三十区的队伍合并，立即前往三十一至四十区清理丧尸！一至十区的丧尸不用清理了，那里的队伍从十一区开始逐步往后清理丧尸，进度到达八成后马上进入下一个区域！全体人员展开快速清理模式，不要给丧尸头领任何机会！”
<br/>　　这一调整完全针对了那只丧尸头领的能力，既然能够唤醒被影响行动的丧尸群，那么何峰的想法便是将聚集的丧尸群以最快的速度变成分散的点，这样一来哪怕丧尸头领到达了那个区域想要唤醒同伴，那些丧尸也并非成群结队而是分开的游走状态。
<br/>　　只要丧尸群得以分散，那就不再能对狼牙战团造成影响！
<br/>　　参谋部便是以这样的反应在拼命追赶楚涵的节奏，只要将前后差距拉近，他们便还有机会反败为胜。
<br/>　　——————
<br/>　　丧尸头领有着极高的战力和智商，它在楚涵刚进入迷宫区的时候就知道有人追来过来，谨慎为上的它不再大张旗鼓的行动，而是立即隐藏了身形，一边不断将所过之处的丧尸群唤醒，一边朝着丧尸密集的区域而去。
<br/>　　而每当丧尸群被唤醒的时候，变灰发出阵阵狂暴的嘶吼声，这些声音时而出现在东面，时而出现在南面，非常不稳定，这也让丧尸头领的位置很难被定位，其移动轨迹也完全无法推测。
<br/>　　所以当楚涵一路狂飙进入了迷宫战区之后，反而跟丢了丧尸头领，并且还时不时的被醒过来的丧尸群包围，加之恶劣的周遭环境，黏糊糊近乎全是丧尸尸体和血液的地面，让楚涵的行动变得越发受阻。
<br/>　　此时用最快速度朝着迷宫区内跑去的陈少爷，则是已经来到了一处高层房屋的楼顶，这里视野很广不说，最重要的是周围大·片都是还未得到清理的区域，从陈少爷的角度往下看，近乎到处都是密密麻麻的丧尸。
<br/>　　丧尸多的地方，一定是那只丧尸头领的首要目的地！
<br/>　　凭借陈少爷一个人想要这么快到达这里自然不可能，就在他刚跑下指挥高台后没多久，蒋天庆就追了上来，并且一路用极其老练的侦查手段，开了挂似得带着陈少爷来到这里。
<br/>　　在陈少爷快速在楼顶做准备工作的时候，蒋天庆也没闲着，立即在周围大·片的地带穿梭。
<br/>　　这名狼牙最年轻的中将，此时此刻就这样进入了疯狂的侦查状态，其恐怖无比的侦查及反侦查能力，更是在丧尸头领之前将对方发现！
<br/>　　不仅如此，蒋天庆还顺带找到了一路追击而来，但却没能第一时间找到丧尸头领的楚涵！
<br/>　　两人刚一碰头就二话不说进入合作状态，蒋天庆调头就走，楚涵快速跟上，一前一后维持着稳定不变的距离，在迷宫战区的各种诡异道路一闪而过，默契的程度堪称史上之最。
<br/>　　蒋天庆带的路时而为矮房屋顶，时而为某些建筑物的前后门，总之其对这片地域的熟悉程度，简直像是从小在这里生活了十八年！
<br/>　　于是蒋天庆的能力爆发之下，楚涵与丧尸头领的距离越来越近！
<br/>　　丧尸头领的速度极快，零零总总已经唤醒了一百五十万的丧尸，眼看就已经踏入了这片未清理区域，光是这里就近乎有一百万丧尸。
<br/>　　它感受着前方不远处那大·片丧尸密集的地带，穿梭在各个不易被察觉的道路上时，表情诡异的带着一丝得逞和畅快，似乎将追来的楚涵耍了一遭过后心情很好。
<br/>　　刷！
<br/>　　它一个跳跃，从地面一举来到了一座两层楼高的屋顶，紧接着脚步不停继续朝着更高也更接近丧尸密集地带的地方而去。
<br/>　　这是一大·片被苏行改造，由各种建筑物连接而成的迷宫墙，丧尸头领就在这些房屋顶快速奔跑，只剩下最后两道迷宫墙，它便能到达目的地！
<br/>　　刷！
<br/>　　它的膝盖毫不犹豫的一曲一伸，朝着前方对面的更高迷宫墙跳去，惊人的弹跳力让它一下子就跳出了一半的距离。
<br/>　　但就在这个弧形跳跃进入到下落过程的时候，忽的！
<br/>　　一个身影强行出现，那是从对面房屋之中的某个窗户跳出来的楚涵！
<br/>　　此时的丧尸头领就在楚涵的正前方偏上的位置，看起来似乎处于劣势，但一个在下落，一个则是在上升，两人必将在半空中相遇。
<br/>　　而无论是角度还是高度，蒋天庆选择的这个窗户这个楼层，配合着楚涵的速度和爆发力，巧合无比的能让他与丧尸头领靠近之时，高度将会完全一致！
