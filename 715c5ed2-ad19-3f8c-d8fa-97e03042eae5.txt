　　一个时辰后，萧逸结束修炼。
<br/>　　紫炎狮王的妖兽精血已被全部吸收。
<br/>　　可喜的是，修罗战体第一层已经练成了。
<br/>　　萧逸站了起来，明显感觉到自己的肉体充满了力量，狠狠一拳轰出。
<br/>　　“轰”的一声巨响，旁边一个灵气屏障激荡了几下。
<br/>　　“好强。”萧逸倒吸一口凉气。
<br/>　　刚才那一拳，他没有使用真气和武技，仅仅是靠自己一身力气打出。
<br/>　　“单凭肉体力量，我便有先天九重的战力，修罗战体果然强大。”萧逸暗暗赞叹了一声。
<br/>　　一般的武者，真气才是最主要的实力来源。
<br/>　　若没了真气，便会实力大降，发挥不出原本应有的战力。
<br/>　　这也是为什么武者在外历练战斗时，一般不敢完全耗光真气，否则将会很危险。
<br/>　　可如今的萧逸，就算没有真气，没有灵器，没有其他任何手段，单凭肉体，就能和一头先天九重妖兽打个不分上下。
<br/>　　“若不算灵器，肉体力量才是我如今的最强战力。呵，第一层已经如此厉害了，若是练成第二层....”萧逸暗暗想道。
<br/>　　当然，他也只是想想。
<br/>　　修罗战体，需要的妖兽精血太多了。
<br/>　　这一次是靠着狂血玄君留下来的六级妖兽紫炎狮王的精血，才一举练成第一层。
<br/>　　“妖兽身上，除了内丹外，血液也蕴含庞大的力量，精血乃是全身血液中最精华的寥寥数十滴。”
<br/>　　“紫炎狮王乃是地元境实力的妖兽，它的全部精血才足够我将第一层练至圆满。”
<br/>　　“若是其它三级妖兽，只有先天境实力，也不知要杀多少头才够，更别说第二层所需的妖兽精血更多了。”
<br/>　　萧逸自语了几声，摇了摇头。
<br/>　　修罗战体共七层，每一层都需要妖兽精血才能修炼。而层数越高，所需的妖兽精血能量也会越多。
<br/>　　这时，萧逸体内忽然一阵躁动。
<br/>　　“嗯？”萧逸一惊，内视一番。
<br/>　　躁动，竟来源于体内的武魂。
<br/>　　原本一直呆头呆脑的控火兽，此刻竟忽然活跃起来，好像有什么东西正涌入它里面。
<br/>　　“是残暴气息，不是被消灭掉了吗？”萧逸脸色大变，但忽然又发现了什么，“不对，不是残暴气息，是一股单纯的力量。”
<br/>　　当控火兽停下活跃后，萧逸感受了一番，那是他的武魂，他自然清楚它身上发生了什么。
<br/>　　“原来如此。”萧逸自语道，“残暴气息来源于紫炎狮王的精血，本身就带有它的力量。残暴的力量被太阴太阳之眼吸收后，剩下的精纯力量全部输送到控火兽身上了。”
<br/>　　就像之前依依被杨伏的毒匕武魂刺中，太阴太阳之眼将毒素灭掉了，剩下的力量则输送到控火兽武魂身上。
<br/>　　可是，太阴太阳之眼不是只能吸收武魂力量吗？怎么此刻连残暴气息也一并吸收了？
<br/>　　半晌，萧逸还是想不明白，手中一动，操控出一股火焰凝聚掌心。
<br/>　　这时，他变得更加惊讶。
<br/>　　因为，火焰竟然是紫色的。
<br/>　　“难道...”萧逸忽然想到了一个可怕的念头。
<br/>　　紫炎狮王可是六级妖兽中有名的妖兽，特别是它操控的紫晶灵炎，更是一种极为强横的火焰。
<br/>　　任何东西一旦被紫炎沾上，便会如附骨之疽，抹也抹不去，水浇不灭，风吹不熄，只有将物体完全烧尽，才会自动熄灭。
<br/>　　萧逸试了试，从乾坤袋中拿出一块紫云精铁。
<br/>　　紫色火焰打到紫云精铁上，立刻紧紧吸附住，不出数秒，整块紫云精铁被烧成灰烬。
<br/>　　“嘶。”萧逸倒吸一口凉气，“竟然真的是紫晶灵炎。”
<br/>　　紫云精铁可是非常坚硬的东西，以往他曾用火焰燃烧过，足足半个时辰才完全融化。
<br/>　　可如今，在紫晶灵炎的焚烧下，却是数秒时间都撑不住。
<br/>　　“控火兽竟然拥有了操控紫晶灵炎的能力。”萧逸不禁皱起了眉头。
<br/>　　这世间有很多种类的火焰，有强有弱。
<br/>　　像萧家大长老，便能操纵一种名为‘流金离火’的火焰。
<br/>　　流金离火，温度远高于一般火焰，攻击力极强；在炼药术，也让大长老焚烧药材的速度大增，炼药能力大大加强。
<br/>　　可这是因为大长老的武魂便是流金离火，火属性武魂。
<br/>　　而控火兽武魂，只会操纵普通的火焰，还是最弱小的那种，用来烧柴还差不多。
<br/>　　紫晶灵炎，则是紫炎狮王才拥有的特殊本事，比流金离火厉害百倍不止。
<br/>　　控火兽怎么也会用了？
<br/>　　萧逸思索了一下，大致有一些猜测。
<br/>　　太阴太阳之眼，本身就能吸收别人的武魂力量，从而提升自己的武魂品阶。
<br/>　　或许，控火兽武魂属于妖兽武魂。
<br/>　　而紫炎狮王也属于妖兽，它的残暴气息上，自然也拥有着它的能力。
<br/>　　太阴太阳之眼吸收掉残暴气息后，将对萧逸身体不利的残暴力量灭掉，剩下的紫炎狮王的能力，则灌输到控火兽身上了。
<br/>　　简单来说，太阴太阳之眼让控火兽获得了操控紫炎的能力。
<br/>　　萧逸不禁一笑，“控火兽呆头呆脑的，虽然精通控火，奈何本身是低级妖兽，只拥有普通火焰。这下倒好，它有了紫晶灵炎，必是威力大增。”
<br/>　　“呼。”，萧逸深呼吸一口气，自语道，“也该离去了。”
<br/>　　说罢，对着卧榻再鞠了一躬，便转身离去。
<br/>　　出了洞府，看着那透明的巨大灵气屏障，萧逸摸了摸，那是一层坚固的屏障。
<br/>　　原本，这个屏障的作用除了屏蔽洞府外，还有着防御的作用。
<br/>　　除非是比狂血玄君更强的人物，否则是不能跨越屏障进入洞府的。
<br/>　　只不过，天星雷鳄动了手脚，这才让猎妖师们能进入罢了。
<br/>　　如今，天星雷鳄凝聚出来的‘分身’已经被消灭，这层灵气屏障便再次恢复了作用。
<br/>　　以后，谁也无法进入洞府。
<br/>　　萧逸也明显感觉到，身上的幽魂面具已经失效。
<br/>　　若他下次再想来，就要等到他有足够的实力强行破除屏障了。
<br/>　　洞府内，除了天星雷鳄外，还有无数雷鸣荒鳄。不过，当年狂血玄君既然没杀，恐怕也是有原因的，萧逸也没敢杀，免得对禁制有影响。
<br/>　　几个呼吸间，萧逸已经离去。
<br/>　　这次的洞府之行，可谓收获颇丰。
<br/>　　一批高品阶的天材地宝，幽魂面具、血戮剑两样宝物。
<br/>　　还有修罗战体这本炼体功法，更兼将第一层炼至圆满。
<br/>　　.....
<br/>　　破玄城，猎妖殿。
<br/>　　萧逸走到任务处，递交了自己的令牌。
<br/>　　猎妖殿人员拿出一本任务手册，对照了几眼，而后归还给萧逸。
<br/>　　“这个任务已经由发布者结束并缴纳赏金了。”猎妖殿人员礼貌地说道。
<br/>　　萧逸点点头，他知道应该是铁达大叔等人回到破玄城后把任务完结了。
<br/>　　猎妖殿人员继续道，“这是个三级的招募任务，你已完成，获得十点任务点。另外，你的等级是一级猎妖师，若你有足够的实力，可以凭此晋升为三级猎妖师。”
<br/>　　“三级猎妖师？”萧逸眼前一亮。
<br/>　　他可不想接低级任务浪费时间，若是能提升等级自然是好的。
<br/>　　完成三级任务后，便有资格晋升三级猎妖师了。
<br/>　　“我需要做什么吗？”萧逸问道。
<br/>　　“很简单。”猎妖殿人员说道，“你要证明自己拥有三级猎妖师的能力。”
<br/>　　“怎么证明？”萧逸皱眉问道。
<br/>　　猎妖殿人员解释道，“最简单的办法自然是展露气息了，但我只是普通的猎妖殿人员，实力不强，你的气息若太强，我怕是无法判断的。”
<br/>　　萧逸皱眉问道，“你们猎妖殿不是有更强的人员吗？”
<br/>　　猎妖殿人员笑道，“本来是有一位执事负责这一项的，不过他恰好有事外出了。执法队的人员倒是够强，可他们没义务替你服务。”
<br/>　　萧逸撇撇嘴，这猎妖殿的规矩还真是麻烦。
<br/>　　“你们执事什么时候回来？”
<br/>　　“说不准，可能数天，可能几个星期。我们总执事倒是有空，但你的权限可见不到他。”
<br/>　　“这...”萧逸无语了，问道，“我是先天境的话，有能力成为三级猎妖师了吧。”
<br/>　　“能是能，只是我无法判断你是否先天境....”
<br/>　　“嘭”的一声，萧逸身上猛然出现了一层真气外衣。
<br/>　　“真气外衣？”猎妖殿人员一惊。
<br/>　　“这能够证明了吗？”萧逸问道。
<br/>　　“当然。”猎妖殿人员点点头，立马接过萧逸的令牌。
<br/>　　先天境不一定拥有真气外衣，但拥有真气外衣的，必然是先天境，而且还是其中的天才。
