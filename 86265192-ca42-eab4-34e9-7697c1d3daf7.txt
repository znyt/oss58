        只不过才三天的时间，他两就成了学校里公认的一对。
<br/>
<br/>        连星儿主动找明司寒当面谈起了这件事情。
<br/>
<br/>        学校树林的大榕树下，阳光被树叶撕碎，稀疏地落在石子路上，光片闪闪。
<br/>
<br/>        明司寒双手抱臂，身着白色衬衫，黑色休闲裤，白色跑鞋，颀长的身躯，背靠着树干，又加之完美到无死角的侧脸，隔着老远的距离，就透着一股帅气逼人的感觉。
<br/>
<br/>        连星儿走过去后，直接开门见山地说道：“你能不能跟大家解释一下，我们不是那种关系？”
<br/>
<br/>        “什么那种关系？”明司寒扬了扬眉梢，意味深长地反问。
<br/>
<br/>        连星儿瘪了瘪嘴，一脸不高兴地说道：“就是男女朋友关系啊！我又不是你女朋友。”
<br/>
<br/>        “我们都是未婚夫妻的关系了，男女朋友关系这个问题，没什么好解释的。”明司寒微微一笑。
<br/>
<br/>        连星儿仍旧不乐意：“反正我不管，你得去解释。我不想当你的未婚妻，也没想过要当你的女朋友。”
<br/>
<br/>        她态度坚决到让他有些痛心。
<br/>
<br/>        不经意间，明司寒瞥见连星儿的手腕上未戴他送给她的那条手链，而是依旧戴着那条不值钱的破水晶手链，顿时又莫名其妙地感到恼火。
<br/>
<br/>        “我送给你的那条手链，你为什么不戴？”明司寒没好气地质问。
<br/>
<br/>        连星儿却是一脸茫然：“你什么时候送我手链了？”
<br/>
<br/>        “我们彼此的家长，一起吃‘联姻同意宴’的那天。”明司寒额冒青筋。
<br/>
<br/>        连星儿这才想起这件事，不禁赧然一笑：“不好意思哈！我把它放家里的包包里了，不记得去看。”
<br/>
<br/>        别说他明司寒送的礼，她没拆开过，就连爹地傅夜沉送的那些价值上千万的礼物，都被她“压箱底”了。
<br/>
<br/>        “以你傅家少小姐的身份，居然会看上这种不值钱的水晶手链？”明司寒一脸不悦。
<br/>
<br/>        连星儿抬起手来，拨弄了一下自己左手腕上的粉水晶手链，笑不拢嘴道：“这可是我朋友亲手为我做的，在我心里，就是无价之宝。”
<br/>
<br/>        明司寒也不知道自己是哪里来的醋意，怒吼道：“我看你就是喜欢洛羽！”
<br/>
<br/>        “对啊！”连星儿不假思索地肯定他的说法。
<br/>
<br/>        明司寒不禁双手握拳，蹙着眉头，只身离去。
<br/>
<br/>        男生寝室里，这几天邢乐游和张子晋，还有仲正浩都在调侃明司寒是不是学古筝去了。
<br/>
<br/>        因为，明司寒回来的时候，十指手指头上绑着创可贴，就连打键盘写代码都有些生痛。
<br/>
<br/>        “司寒，你这几天是做什么去了？把手搞成这个样子？”调侃完后，邢乐游凑到明司寒的身旁关心道。
<br/>
<br/>        明司寒正拿着药水给自己的手指上药，有些心不在焉地说道：“是不是女孩子都比较注重一样东西的意义，而不在乎它的价值？”
<br/>
<br/>        “那你得看什么样子的女孩子了。部分女孩子，会比较注重一样东西的意义所在。还有一部分了，确实会比较看重一样东西的价值。有的女孩子拜金，有的女孩子不拜金。”邢乐游接着说道。
<br/>
<br/>        张子晋这个时候补充了一句：“司寒，那个连星儿又让你闹心了吗？”
<br/>
<br/>        仲正浩随声应和：“十有八九是那丫头吧！”
<br/>
<br/>        寝室里，他们三看明司寒对连星儿的那事，已经心知肚明了。
<br/>
<br/>        明司寒十有八九是喜欢人家连星儿。
<br/>
<br/>        至于连星儿……
<br/>
<br/>        他们心里没底，凭感觉来讲，人家丫头似乎对明司寒没那种感情。
<br/>
<br/>        明司寒不吭声，心里只是想着，等自己亲手把手链做好后，一定会让那丫头爱不释手。
<br/>
<br/>        这天晚上，他从珠宝首饰工作坊里出来，将自己打造好的手链放在小礼盒中便塞入了裤兜里。
<br/>
<br/>        这条手链，他用的是铂金材质，上面的点缀是星型粉钻。
<br/>
<br/>        如果他没记错的话，连星儿喜欢的颜色应该是粉色，毕竟，平常，他见她穿粉色的衣服比较居多。
<br/>
<br/>        他亲手做的手链，这下，连星儿总没话可说了吧！
<br/>
<br/>        明司寒心里正得意的时候，身后突然传来了一阵急促得不像常人的脚步声。
<br/>
<br/>        这是一条小巷，回学校的捷径，平常一到七八点就鲜有人活动，而此刻已经十点了，除了满月的光辉铺洒时，野猫会喵喵直叫，而更多的时候都是静谧无声。
<br/>
<br/>        明司寒警惕地皱了皱眉，故作淡定地继续往前走。
<br/>
<br/>        当他感觉到身后有微风袭来之际，他一个侧身，微微弯腰，将背后偷袭的人来了个过肩摔。
<br/>
<br/>        刚解决掉一个不知哪里冒出来的黑衣人，又有另外一个黑衣人手持电棒向他挥了过来。
