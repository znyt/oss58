　　接过长老身份玉牌后，慕清霄暗骂一声老狐狸，但脸上却没有表现出来，微行虚礼，道：“多谢掌门。”
<br/>　　听他改口，涵素真人非常满意。
<br/>　　“等明日弟子受封时，便在天墉城中宣布你的身份，等一会本座就命人带你去看一看住处，你自己选一座偏殿。”
<br/>　　“有劳。”
<br/>　　“不必客气，将天墉城当成自己的家就好，行了，你们都去殿外等候吧。”
<br/>　　闻言，慕清霄微微点头，率先向殿外走去，原本他想要求天墉城的强大剑技，思索过后，最终将想法压了下去。
<br/>　　正所谓无功不受禄，他对天墉城也没有丝毫的贡献，现在要求反倒会留下不好的印象。
<br/>　　来到殿门口，慕清霄就将腰间的佩饰取下，随手将凝丹长老身份玉佩挂了上去，更加的衬托出他的高贵不凡。
<br/>　　约莫十分钟后，一名身穿弟子服饰的年轻女子，与七名男弟子走了过来。
<br/>　　可能是修行过的缘故，女弟子容貌颇为清秀，男弟子也算是俊逸，身上都透着一股出尘的气质。
<br/>　　其中那名女弟子，在各位长老身上打量一番，当看到慕清霄腰间的凝丹玉佩时，清秀的脸蛋上，浮现惊愕之色。
<br/>　　不仅是这名女弟子，其余七名男弟子也注意到了慕清霄的存在，站在长老群众，简直格格不入，想不注意到都难。
<br/>　　凝丹长老不是糟老头么？如此年轻俊逸的凝丹长老，简直前所未闻。
<br/>　　只是，当他们看到凝丹玉佩时，看向慕清霄的目光，都变得尊敬起来。
<br/>　　容不得他们不尊敬，要知道，凝丹长老可是管辖者整个凝丹堂的资源，这些弟子往后可都需要和他打交道。
<br/>　　那名清秀女弟子愣神过后，快步走了上来，眼眸中满是恭敬，行礼道：“请问，可是慕长老。”
<br/>　　“正是。”
<br/>　　慕清霄微微点头，打量了她一眼，半步筑基，资质一般般吧。
<br/>　　“侍女小兰，见过慕长老，接下来由小兰带领您去挑选住处，请随我来。”
<br/>　　说着，小兰便转身离去，慕清霄则紧随其后。
<br/>　　普通长老们，望着慕清霄随美女弟子离去的背影，又看了看自己身前的青年，面面相觑，待遇怎么就相差如此悬殊呢？
<br/>　　偏殿，坐落在天墉城西侧的宫殿群。
<br/>　　其中设备应有尽有，只有天墉城中，尊贵的长老才有住的资格，至于普通长老，最多也就比普通弟子住的好一些。
<br/>　　两个时辰后，跟在小兰身后，在西侧转悠了一圈，慕清霄最后选择了，坐落在昆仑最边缘的一座宫殿。
<br/>　　宫殿坐落在山崖边缘，确实是好地方，距离天墉城中央腹地，却是偏的有点远。
<br/>　　若非如此，这种好地方，怎么可能留给他。
<br/>　　虽然偏僻了点，但正合慕清霄的意思，风景优美，站在凉亭中，还能欣赏昆仑周边风景，实在是一处绝佳的景地。
<br/>　　至于侍女小兰，也随慕清霄留在宫殿中，照顾他的衣食起居，便是做待女的每日任务。
<br/>　　甚至，只要慕清霄想，要了她的身子都不是问题，想必小兰也非常愿意。
<br/>　　因为，整个天墉城，除了掌门与执剑长老外，就属凝丹长老权利最高，哪怕其他长老，权利都有所不如。
<br/>　　更何况，慕清霄还如此年轻。
<br/>　　找到住处后，慕清霄就迅速熟悉起其中的环境，客厅卧室都有许多，最重要的是，其中有独立的炼丹室。
<br/>　　需要炼丹的时候，直接让弟子将灵药送来就行，炼制好后，也有弟子来取，非常方便。
<br/>　　对于住处，慕清霄非常满意，挑不出任何毛病。
<br/>　　傍晚，满天星辰衬托着天上的圆月，慕清霄则思绪万千。
<br/>　　“掠夺任务①：掠夺烛龙之鳞，成功奖励：500000掠夺积分，失败惩罚：死亡！”
<br/>　　“掠夺任务②：掠夺龙渊七凶剑之一‘焚寂’，成功奖励：300000掠夺积分，失败惩罚：修为跌落一层。”
<br/>　　“掠夺任务③：掠夺玉衡，成功奖励：200000掠夺积分，失败惩罚：修为跌落五重。”
<br/>　　这是他来到古剑奇谭位面后，所接到的三个任务，奖励非常丰富，甚至可以说是一笔巨款，整整的一百万掠夺积分。
<br/>　　有了这一百万掠夺积分，他就能给妻子们兑换最顶级的修炼心法，但是，奖励越是丰富，往往风险就越大。
<br/>　　以上三个任务，除了当时在神雕位面，与龙儿相遇时接到的任务，危险系数绝对是有史以来之最。
<br/>　　烛龙之鳞，焚寂，玉衡，三样东西，哪一样不是宝物，但它们的源头都与一个人有关，或者说与一位曾经的仙有关。
<br/>　　欧阳少恭！
<br/>　　欧阳少恭的前身是太子长琴，太古时的三界第一乐师，榣山奏乐怡情时，结识水虺悭臾并成为好友。
<br/>　　悭臾修成应龙后因触犯天条逃入不周山，求得烛龙之子钟鼓的庇护，天帝派共工、祝融与太子长琴前往捉拿。
<br/>　　太子长琴弹奏让钟鼓入睡，却因突然看见黑龙金色眼瞳，认出昔日好友，惊异之下忘记奏乐，钟鼓醒来，与水火之神战斗而致天柱倾塌。
<br/>　　至此长琴太子被贬为凡人，永世不得为仙，落凡后寡亲缘情缘，不管哪一世皆为孤独之命。
<br/>　　因失去命魂无法再入轮回，只能通过不断渡魂存活于世，太子长琴被贬为凡人，永远不能再拥有仙籍。
<br/>　　但太子长琴被贬投胎时，因眷恋瑶山不忍离去，他的命魂四魄被角离以血涂之阵强行引入焚寂剑中而陷入沉睡。
<br/>　　不甘之下，太子长琴选择渡魂。
<br/>　　渡魂稍有不慎即魂飞魄散，每一次渡魂也会损失混乱一部分记忆。
<br/>　　在欧阳少恭淡定睿智的外表之下，隐藏着执着疯狂的一面。
<br/>　　无论渡魂多少次，欧阳少恭终没有办法在修仙之事上真正大成，但是执着的他从未放弃向孤独的命运抗争。
<br/>　　对生死和孤独感到超乎常人的痛苦，最终认为人的意识和生死是一件只能带来痛苦和麻烦的东西，没有思维而永恒地活着才是一种理想的状态。
<br/>　　欧阳少恭第一次渡魂仅是出于求生本能，自己并无清晰的意识，以角越身份活下来后，常呆望焚寂，焚寂被封印后，“角越”投炉，是对自身魂魄的追寻与绝望。
<br/>　　投炉身死，魂魄凭本能继续求活，直到渡魂几次后，这二魂三魄才渐渐清醒，明白自己是什么，为什么。
<br/>　　余下的二魂三魄也从此无法再入轮回，几千年后少恭寻到血涂之阵秘法和焚寂所在，夺回焚寂另一半魂魄之时，被韩云溪的母亲转移入儿子体内。
<br/>　　这就是古剑奇谭位面的开端。
<br/>　　叹息长琴太子命运的同时，慕清霄也有些无奈，现在距离剧情开始，最起码还有五六年的时间。
<br/>　　那么，剩下的唯有等待了。
