　　“楚风眠，你可要小心一点，这燕碧性格，残忍无比，这次你让她没了面子，这次武斗大会上，她肯定会动用手段来对付你的！”
<br/>　　寒月漓在一旁，看到楚风眠不以为然的样子，突然出声解释道。

<br/>　　“在混战中分到的武者，很有可能便是跟他们有所关系的，混战之中，九人合力，围攻一人的话，就算是实力再强，也有被直接淘汰的可能。”

<br/>　　寒月漓解释了起来。
<br/>　　擂台战，的确是公正无比，全凭力量。

<br/>　　在强大，天资在妖孽，在碰到九位天才围攻的时刻，也很有可能被击败。
<br/>　　历届之中的强者，也有着不少人是被针对，倒在了这混战之中。
<br/>　　如今以这燕碧的性格，楚风眠居然不给他的面子，他肯定会前来对付楚风眠的，最好对付楚风眠的机会，便是在擂台战上。

<br/>　　这些人想要从中捣鬼，并不是一件难事。
<br/>　　“她若是敢来捣鬼，便是她死的时候。”
<br/>　　楚风眠平静的开口道。
<br/>　　他的对手可不少，夜天君恨他恨到极致，不用这燕碧动手，夜天君也会选择对付楚风眠。
<br/>　　敌人众多，他也不怕，在绝对的实力面前，一切的阴谋诡计都没有任何的意义。
<br/>　　这样的议论声，并不少。
<br/>　　楚风眠跟寒月漓，坐在广场的一处，闭眼休息了下来。
<br/>　　在这广场上，像是楚风眠这样坐着休息的武者，数不胜数。
<br/>　　在武斗大会之前，众人都想要将自己的状态，达到了最为完美的地步，来参加这武斗大会的考核。
<br/>　　时间一分一秒的过去。
<br/>　　在这武王城汇总聚集的武者数量也是越来越多，各大宗门，武道世家，隐秘势力的强者，都纷纷出现了。
<br/>　　基本上北域百绝榜上的人物，如今已经是全部到齐，只是那北域百绝榜上的第一人，风无心，还没有出现。
<br/>　　“轰隆！”
<br/>　　数日之后。
<br/>　　就在这武王城中，已经是人声鼎沸，这武斗大会，即将开启的一刻。
<br/>　　一道身影，突然从天而降。
<br/>　　这一道身影，身材高大，脸上刀削斧凿的男子，此人的脸色，平静到极致。
<br/>　　这种平静，并非是故意装出来的，而是他真正的内心，便是无悲无喜，极为平静的性格。
<br/>　　“风无心！”
<br/>　　“风无心出现了！”
<br/>　　一道道武者的视线，也都是看了过去，看向这突然出现的男子。
<br/>　　此人，便是北域百绝榜的榜首，风无心。
<br/>　　“在最后关头，终于出现了。”
<br/>　　楚风眠也是打量了一眼这风无心，这风无心一眼望过去，看起来只像是一个平平无奇之辈，只是心态，无悲无喜。
<br/>　　可是楚风眠的心中清楚，这种心态，才是最为可怕的，泰山崩于前而不变色，虽然这风无心的内心，显然并非是自己锻炼出来的，但是这种境界，也是绝对完美的境界。
<br/>　　一个无懈可击的心态，比起拥有先天神体的天才，都丝毫不差。
<br/>　　此人，隐藏极深。
<br/>　　这风无心，看到楚风眠的目光，似乎也是瞥了楚风眠一眼。
<br/>　　虽然仅仅是一瞬间，但是楚风眠清楚，这风无心，已经是注意到了他的存在。
<br/>　　如此敏锐，倒是有些出乎楚风眠的意料。
<br/>　　“轰隆隆！”
<br/>　　就在几乎同时，一声巨响，突然响彻而起。
<br/>　　就在这广场背后，一扇巨大无比的铜门，突然打开，这铜门的背后，便是这一次武斗大会的会场。
<br/>　　武斗大会，虽然是在武王城举办，但是却并非是就在武王城中举行，而是在另外一方空间之中。
<br/>　　武王城，只是一座大门，如今这通往会场的大门，已经开启。

<br/>　　“我们进去！”
<br/>　　来到这里的无数武者，都不是第一次来参加这武斗大会，看到大门打开，一个个涌入其中。
<br/>　　无数人浪，都向着其中挤了进去。
<br/>　　楚风眠跟寒月漓，也是同时起身，一起向着其中，直接身形一闪，冲了进去。
<br/>　　进入到这大门中，顿时楚风眠的面前，开始天旋地转。
<br/>　　这便是要进入到，另外一处空间的变化，这大门，其实便是一座传送法阵。

<br/>　　这里，的确是最为适合开始武斗大会的地方，简直是一个完美的地方。
<br/>　　“这一次武斗大会，就由我们通天阁来举办！”
<br/>　　在这最高的擂台之上，一尊老者，就站在这里，这是一尊实力深不见底的老古董，一眼望过去，此人的实力，只怕是要比起冰漓长老还要强大。
<br/>　　也是一尊超越了生死境的人物，甚至是在生死台阶上，都有着极为可怕的实力。
