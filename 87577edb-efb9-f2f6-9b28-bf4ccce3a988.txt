        这根刺也成为了她此刻唯一的一点理智。

<br/>        昼景爱她吗？
<br/>
<br/>        到底爱不爱？
<br/>
<br/>        她好想知道他心底的答案是什么。
<br/>
<br/>        蓦地，傅夜沉眸光深邃，情不自禁地抬起另一只手，缓缓地凑到了她脸上的那具白羽假面眼镜上。
<br/>
<br/>        突然间，他好想看看她的真实容颜。
<br/>
<br/>        可这样的念头，却被他及时扼杀在摇篮里。

<br/>        不能再对这个女人动任何心思了！
<br/>
<br/>        他爱的女人是“漫雪”，他不能花心！！
<br/>
<br/>        傅夜沉抬起的那只手，猝不及防地在童筱颖的脖颈后用力一击。
<br/>
<br/>        童筱颖瞬间两眼发黑眼晕，无力地软瘫在傅夜沉怀里昏了过去。
<br/>
<br/>        回到私宅后……
<br/>
<br/>        傅夜沉凝眉问道：“她到底怎么呢？”
<br/>
<br/>        连华生看着手里的检验报告，唉声叹气地回答道：“检查结果出来了，她被人下了那种比较特殊的药。不同于常规的那种药，而且因为下的分量多了一些，而胃里已经消化吸收了，所以导致她现在浑身无力，但又……”
<br/>
<br/>        “又什么？”
<br/>
<br/>        “待会她醒来后，需要的是这张床，以及一个男人就行了。只有等她体内的药效都散了，就没事了！”连华生睐了傅夜沉一眼，眉飞色舞地说。
<br/>
<br/>        傅夜沉顿时额冒青筋：“连华生，你个庸医！”
<br/>
<br/>        “你若是不上她，那就把她留给我好了。”连华生勾了勾唇，眸子里像是窜出了一团火光般，意味犹存地打量着床上的女人，伸出舌头来舔了舔唇瓣，嘶溜一声，道，“我在她身上，闻到了处子的味道。”
<br/>
<br/>        傅夜沉瞪了连华生一眼，冷斥：“滚！”
<br/>
<br/>        “OK！你等着！”连华生忍俊不禁起来，转身从自己的医药箱里拿出了一支注射器，嘴角微扬，得意地问，“这下，你觉得，是我来呢？还是你上呢？”
<br/>
<br/>        傅夜沉看着连华生，瞬间黑脸了。
<br/>
<br/>        翌日……
<br/>
<br/>        童筱颖再次睁开双眼的时候，天都已经亮了。
<br/>
<br/>        天花板上吊着的是一盏仿古百烛灯，四角有花纹繁复的金色装饰线条点缀。
<br/>
<br/>        这里是哪里？
<br/>
<br/>        童筱颖撑起身子坐了起来，有些头疼地揉了揉太阳穴。
<br/>
<br/>        她身上盖的被褥是烫金色的真丝被，大床软如鹅毛，大得出奇。
<br/>
<br/>        房间里的陈设，全是那种欧洲皇室的仿古风格，金碧辉煌的色调，让人产生一种时空交错的错觉。
<br/>
<br/>        “姑娘，你醒了啊！早餐已经给你备好了。”
<br/>
<br/>        那边只有门框垂着珠帘的大门外，一个中年妇女手里端着餐盘，一手撩开珠帘，走了进来。
<br/>
<br/>        童筱颖只觉这个中年妇女令她觉得眼熟，努力一番回忆后，她才猛然想起，这个中年妇女就是上次她救了的那个男人家的保姆。
<br/>
<br/>        她来过这里！
<br/>
<br/>        所以，这里是……
<br/>
<br/>        那个男人的家！
<br/>
<br/>        童筱颖下意识地在脸上摸了摸，发现自己的假面眼镜不见了后，心里顿时一阵忐忑不安。
<br/>
<br/>        刘婶见她在找那具白羽假面眼镜，于是将手里的餐盘放到矮柜上后，从床头柜的抽屉里拿出了那副白羽假面眼镜，递了过去。

<br/>        “小谢，你长得这么漂亮，为什么要戴着这副面具呢？”

<br/>        刘婶笑得两眼眯成了缝。
<br/>
<br/>        童筱颖接过白羽假面眼镜后，连忙重新戴回了脸上。
<br/>
<br/>        “你昨晚出了一身的汗，身上全是一些脏东西，大少爷让我给你洗个澡。所以，我才把你的面具给摘下来的。”刘婶和蔼可亲地说。

<br/>        她还真就没认出，面前这个小谢，就是上次大少爷从外带回来的那个未婚妻。

<br/>        刘婶虽然没认出童筱颖来，但是童筱颖却记得刘婶。
<br/>
<br/>        “那你家大少爷呢？”童筱颖试探性地问了一句。

<br/>        刘婶微笑道：“在楼下和连医生一起吃早餐了，小谢要不要也下去跟他们一起吃？”

<br/>        “是这样啊！”童筱颖半信半疑地应了声，难道是她因为感冒发烧还未痊愈的缘故吗？
<br/>
<br/>        楼下的餐厅里。
<br/>
<br/>        连华生一边将培根夹入生菜里还抹了花生酱，一边抬眸打量着坐在长桌对面的傅夜沉。

<br/>        “我看你是不敢吧！是怕把她的面具摘下来后，因此更加爱她了！”连华生勾唇一笑。

<br/>        以前这位冷面阎王，怎么逗，他都是面无表情，喜怒不形于色。

<br/>        可是，自从他回国后，拉着他在“夜惑”里邂逅了这位“小谢”姑娘……
