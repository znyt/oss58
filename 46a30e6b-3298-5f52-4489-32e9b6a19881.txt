　　　　终于在无数观众的期待下，在休息了将近30分钟后，双方的十王者战最终局盲选战终于是正式开始。
<br/>　　“好了，刚刚接到后台的消息，双方已经是准备好了，十王者战最终战盲选战正式开始！”随着米粒的一声大喊，现场那块巨大的LED显示屏上，顿时出现了选人画面。
<br/>　　在大家的注视下，双方直接进入到了选人缓解，十个人都没有在第一时间确定英雄，而是在那里随意的点击着，有的放在提莫的头像上，有的则是放在剑圣、德邦这几个英雄的身上。
<br/>　　每当看到双方亮出这些英雄的时候，即使知道他们不会选择，但是现场还有现场外的那将近20万的观众还是会爆发出一阵巨大的欢呼声。
<br/>　　“别闹了好吧，根本不可能选提莫的！”解说席上的米粒，看着场上的选人，这个时候也是在那里笑着说道。
<br/>　　“这边两边的中单率先确定好英雄了，都选择了发条，在这最终局当中，两人居然都选择了发条这个英雄。”突然间随着两道锁定英雄的声音响起，林雅楠立刻开口说道。
<br/>　　只见场上叶帝还有金相赫两人的选人位置，他们两个已经确定好了发条这个英雄。
<br/>　　看到这一幕，现场的观众顿时再次爆发出了一阵巨大的欢呼声。
<br/>　　“一样的英雄，双方选择了一样的中单，在这关键局中，两人还是都选择了发条这个英雄，不得不说发条虽然是个万金油的英雄，但是在这种局势当中，这也是一个最值得信赖的英雄，这一场估计是要打到大后期了。”米粒跟着开口说道。
<br/>　　“6666666666666，王和王的对决，最终战都拿出了发条，这可有意思了，我现在又希望这一局出现单杀，又不希望出现中路单杀。”
<br/>　　“我也是如此，中路单杀对方就代表着一个神话的终结，一个传奇的诞生，不过我现在还是更期待两个神话在一起。”
<br/>　　“果然没米粒他们料中了，在这种关键局中，中路还是选择了发条这样的刷线英雄，这个英雄到了后期的伤害还是非常的高的。”
<br/>　　“越是一样的英雄，才越能展现出双方的实力来，这一局中路有看头了，就是不知道到底会不会出现单杀。”
<br/>　　看到中路选择的英雄，众多观众顿时兴奋的议论了起来，虽然不是劫那种能秀的英雄，但是能够看到这种相同的英雄出现，大家还是要更兴奋一些的。
<br/>　　而在中路确定好英雄之后，双方的辅助紧接着也是确定好了各自的英雄，两边的辅助这一刻居然都是选择了锤石。
<br/>　　“我的天，两边的辅助都是锤石，又是一样的英雄，这一局对决真的有意思了。”看着双方的辅助，米粒连忙大声的喊道。
<br/>　　“霄神选择锤石我倒是不意外，没想到long居然也选择了这个英雄，在我的印象里long用锤石的次数可是屈指可数的，看来这一阵子他肯定是专门修炼了一下这个英雄，故意在等待这一局的盲选战的，这一局辅助的对决倒也是一个亮点。”林雅楠跟着开口。
<br/>　　看到辅助也选择了一样的英雄，这一刻场下的观众更加的热情了起来，一个个仿若不要命般的在那里嘶吼着。
<br/>　　就是连那些职业选手也不例外，从来没有一场比赛像现在这样仅仅在选人的时候就让他们感觉如此的热血沸腾。
<br/>　　下一刻双方的上单也都确定好了各自的英雄，只不过这一次双方倒是没有选择相同的英雄，蓝队这边蒋念选择了石头人，而红队这边的god则是选择了泰坦。
<br/>　　“这一边红队选择了上单泰坦，蓝队这边选择的则是石头人，单论这一手的话，我感觉蓝队这边占了一些上风。”米粒再次开口说道。
<br/>　　“没错，石头人这个英雄是可以带球冲锋的，只要石头人的大招能够大起来两人，那么发条的大招就可以无缝连接，这到了后期对于红队这边绝对是毁灭性的打击的。”林雅楠再次开口。
<br/>　　“而且石头人这个英雄在前期和泰坦这个英雄对线的时候，也不会吃太多亏的，这一局如果真打到后期的话，红队这边很危险。”米粒继续开口。
<br/>　　看到双方的这个选人，众多职业选手这个时候也都是皱起了眉头，按照双方这几个位置的选人，双方绝对是要打后期的，但是打后期的话石头人和发条这对组合，确实是要强过发条和泰坦的。
<br/>　　场上的时间缓缓的走着，当时间还剩下最后5秒钟的时候，双方最后两个位置的英雄，在这一刻也终于是确定了下来。
<br/>　　“红队这边楚白衣选择了女警这个英雄，而蓝队这边的emp没有出乎雅楠的预料，还是自信的选择了他的本命英雄圣枪游侠。”米粒大声开口喊道。
<br/>　　“蓝队这边的打野夜雪狂歌则是没有选择他最拿手的男枪，反而是选择了盲僧这个英雄，红队这边的伟神也是选择了盲僧，看来大家的观点还是非常一致的，在这种大赛中，盲僧这个英雄的节奏带动还是要远远强于男枪的。”林雅楠跟着说道。
<br/>　　“没错，往往越是到了关键性的比赛，双方的打野便是越会选择盲僧这样的英雄，大家都说盲僧这个英雄到了后期没用，但是那也要分在谁的手里，真正高手的盲僧，即使到了后期，也是能够打出毁天灭地般的连招的，只要回旋踢踢回来一个c位，那么这一波就是他们胜利。”米粒点头说道。
<br/>　　最终红队这边选择了：上单泰坦、打野盲僧、中路发条、下路则是女警和锤石。
<br/>　　而蓝队这边则是选择了：上单石头人、打野盲僧、中路发条、下路则是圣枪游侠和锤石这两个英雄。
<br/>　　双方在三个位置上选择了相同的英雄，看到双方的阵容终于确定下来后，现场的气氛再吃一次热闹了起来，人浪也是被众人玩了起来，那个样子看起来好不壮观！
