<br/>阿尔法随意打开了一个玉盒，楞了一下，竟然是魔核！虽然才是四阶火属性魔核，但是却已经价值不菲了。<br/>　　<br/>　　阿尔法又随手打开了一个，三阶金属性魔核，看样子这些盒子都是一些中低阶的魔核了。<br/>　　<br/>　　魔核是魔兽死亡时精华凝聚而成，虽然每只魔兽死亡理论上都会凝聚出魔核，但是魔核的出现几率却十不足一。往往越高阶的魔兽，出现魔核的几率越大，但是高阶魔兽又异常强大，智慧也很高，被击杀的几率也很低，往往在临死时会选择自爆，与敌人同归于尽。所以这个世界上高阶魔核的存在少之又少，一颗7阶以上的魔核都是千万金难求的，往往都是以物换物。更不要说顶级9阶的魔核了，据说存世的也不会超过5颗，其中一颗还在东大陆的教皇手里。<br/>　　<br/>　　“振邦，这些魔核你都收起来吧！也许以后你会用的到。”阿尔法将手中的玉盒随意的抛给了李振邦。<br/>　　<br/>　　“这难道就是爱德华要隐藏的秘密吗？”李振邦疑惑的问道。<br/>　　<br/>　　“有可能吧！这些魔核数量虽然不多，但是价值已经非常高昂了。很多大家族都会储存一定量的魔核备用。尤其是中低阶魔核，对家族低级成员的修炼会有很大的好处。你脖子上的那枚6阶光明属性的魔核项链可是对身体有着非常大的好处的，光明属性的魔核可是只有东大陆才会有的，在西大陆上可是堪比8阶魔核的价值了。”<br/>　　<br/>　　李振邦拽出了胸前6阶光明魔核项链，长大了嘴巴，没想到杨冰老师送给自己了一件这么珍贵的礼物！<br/>　　<br/>　　“既然没有什么东西了，我们就走吧！”欧米伽开口说道，这里毕竟是爱德华的藏宝阁，不是久留之处。<br/>　　<br/>　　阿尔法点了点头，刚抬起腿，却有种异样的感觉。那是一种身为大盗的直觉，仿佛自己如果就这样离开，会遗憾终生的感觉。<br/>　　<br/>　　“等一等，也许这只是爱德华的一个障眼法！”<br/>　　<br/>　　“障眼法？什么意思？”欧米伽不解的问道。<br/>　　<br/>　　“是不是说，这里其实是一个假的宝藏，虽然很吸引人，但是为了掩藏另一个更加珍贵的宝藏？弃车保帅？”李振邦若有所思的问道。<br/>　　<br/>　　“振邦，以后跟着我混吧！我发现你很有做大盗的潜质。”阿尔法看着李振邦的眼神中闪烁着赞许的光芒。<br/>　　<br/>　　“阿尔法姐姐，你就别开玩笑了。我很清楚自己几斤几两，我这点实力，还是不太适合这个高风险高回报的很有前途的职业。”<br/>　　<br/>　　“那就太可惜了！”阿尔法摇了摇头，并没有勉强李振邦，只是惋惜的摸了摸李振邦的脑袋。<br/>　　<br/>　　阿尔法在密室内仔细观瞧，没有放过任何一个细节。她可以肯定，密室中不会再有任何暗格或者暗道了。<br/>　　<br/>　　三人又回到了四层仔细观瞧依然没有收获，阿尔法相信自己的直觉是不会有错的，可是第四层和暗格似乎都没有再值得注意的地方了。<br/>　　<br/>　　“阿尔法姐姐，你确定这里还会有暗格或者密室吗？”<br/>　　<br/>　　“我的直觉告诉我应该还有，而且比我们看到的更有价值。”阿尔法很肯定的说道，圣级强者的直觉往往都是很准的。<br/>　　<br/>　　李振邦仔细的回忆着爱德华带着他们进来的全部细节，突然灵光一现。<br/>　　<br/>　　“有没有可能在一层？”<br/>　　<br/>　　“一层？怎么可能呢？我们都是从一层走过来的！”欧米伽疑惑的说道。<br/>　　<br/>　　“如果这一层都没有的话，那就很可能是在第一层啊！最危险的地方就是最安全的地方。我刚才仔细回想了一下，爱德华带我们进来之后，并没有在一层逗留，直接下到了二层。一开始我们都以为应该是第一层的东西比较普通，大家都看不上眼，所以才下到第二层。可是第二层的入口也未免太过于明显了。一般人如果偷盗，看到第一层的金币珠宝可能就会偷走了。可是如果看到还有第二层，那么正常人都会想着怎么进入二层，因为按照常理二层要比一层更有价值，而事实也确实如此。第三层就相对要隐蔽一些，第四层就更加隐蔽了，而那道暗格即便是阿尔法姐姐也费了点周折才打开。既然都找到了暗格，那还会有人回到第一层去找宝贝吗？偷完好东西，肯定是第一时间逃走才对啊！”<br/>　　<br/>　　“振邦说的有道理，咱们上去看看。振邦，你真的不再考虑考虑跟着我混了？”<br/>　　<br/>　　“阿尔法姐姐，你饶了我吧！就我这小身板，除了给你们拖后腿以外没啥作用了。”李振邦一脸无辜的说道。<br/>　　<br/>　　“算了，不强求你了，强扭得瓜也不甜，咱们先去一层看看。”<br/>　　<br/>　　事实果然如李振邦所料，第一层果然有一间密室。虽然说是密室，可给人的感觉更像是一个未知的空间。<br/>　　<br/>　　开启的方
