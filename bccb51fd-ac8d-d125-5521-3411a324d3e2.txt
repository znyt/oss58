　　“黑云学教下的通缉令？”
<br/>　　横天殿主的话语落下，就连承风殿主亦皱起了眉头。
<br/>　　承风殿主眯了眯眼，似是想起了什么。
<br/>　　“黑云学教的副院长，你们学教，能发布通缉令我知晓。”
<br/>　　“但，其余五殿的通缉令，同样由你黑云学教发出，随后五殿殿主代由发布？”
<br/>　　“老夫没记错的话，你这位副院长可是相交遍布中域，江流火、白月、金绝玄等人，皆是你的好友。”
<br/>　　“皆是好友？”萧逸闻言，直视副院长的目光，已然变得冰冷瘆人。
<br/>　　江流火、金绝玄、白月、荆无等一个个主殿主以及总殿之内任职的副殿主，竟皆是副院长的好友？
<br/>　　萧逸认得这些个主殿主和副殿主，是因为这些人本就是中域之内成名多年的强者，而且也是各殿功绩累累的有名之人。
<br/>　　他以前便有查探各大势力，还有一些声名斐然的强者情报的习惯。
<br/>　　其目的，便是更快更容易的了解中域之内的事情。
<br/>　　且，他本就身兼风刹殿武者的身份，查阅这些情报本就容易。
<br/>　　故，他一眼就能认得这些个主殿主和副殿主。
<br/>　　但，他也只是认得，大概知晓这些人的身份，要说深入了解，那自然不可能。
<br/>　　反倒是承风殿主，他在风刹殿总殿之内，任职总殿主多年了。
<br/>　　知晓的情报，还有对这些各殿主殿主、副殿主，自然了解得更多。
<br/>　　此话从承风殿主口中，那便自然不会错了。
<br/>　　这时，副院长感受到萧逸冰冷的目光，连声道，“萧逸，你不要误会。”
<br/>　　“你也曾是黑云学教弟子，该知道，我们学教本就与各殿有交情。”
<br/>　　“我们学教，本就时常替各殿完成一些棘手的任务，也能在各殿中发布通缉令。”
<br/>　　对，这一点，萧逸自然知晓。
<br/>　　黑云学教与各殿，有着联系。
<br/>　　其原因，便是副院长乃是药尊殿的十大殿主之一。
<br/>　　炼药师，本就是朋友遍天下；而药尊殿方面，虽和黑魔殿、风刹殿等，不是同一联盟殿，但总的来说和各殿的关系都不错。
<br/>　　故黑云学教本身，可以当作一个势力，或者一支猎妖队，却接这些任务，并完成。
<br/>　　而同时，也因其关系，故黑云学教方面，有一定的特权。
<br/>　　如通缉令的补发。
<br/>　　正常的通缉令，应该是先发布，随后再由武者去完成。
<br/>　　而黑云学教方面，萧逸记得，如当年的花灵宗之事，便是先完成了任务，通缉令后面再补发的。
<br/>　　以黑云学教的特殊性，能在各殿发通缉令，甚至各殿的主殿主等有名的殿主都亲自相助，确实有可能。
<br/>　　只是…
<br/>　　“副院长。”萧逸冷眼凝视着副院长。
<br/>　　“我劝你不要骗我，我既在黑云学教待过，你自然也知道我的性格和本事。”
<br/>　　萧逸的脸色，异常认真，语气，也异常凝重。
<br/>　　副院长闻言，脸上微不可察地闪过一丝复杂，但还是沉声道，“我没骗你。”
<br/>　　萧逸冷声道，“那请你告诉我，黑云学教发布通缉令也就罢了，为何要在六龙地域那边也发布？”
<br/>　　“黑云地域内，便有各殿的主殿。”
<br/>　　“你大可直接在黑云地域的药尊殿、天机殿、黑魔殿等各殿直接发布。”
<br/>　　“为何偏偏在黑云地域发一份，六龙地域发一份，别的地域也发一份，随后更是由各位主殿主替你代发？”
<br/>　　“而且，这些个替黑云学教代发通缉令的主殿主，还皆是你的好友。”
<br/>　　萧逸深知副院长此人为人不错，但论及圆滑，同样厉害。
<br/>　　所以副院长的解释，他不会信全，亦直接质疑起来。
<br/>　　若要发通缉令，为何要各主殿主代发，为何黑云学教不光明正大地发布？
<br/>　　“不过是巧合罢了。”副院长沉声回答道。
<br/>　　“药尊殿主殿，确实丢了一批珍稀无比的天材地宝。”
<br/>　　“而最有嫌疑的，便是你身后这群朋友，他们平日里行踪低调，又恰好在那附近出现过。”
<br/>　　“那批天材地宝，本是我们学教帮忙护送去药尊殿。”
<br/>　　“自然的，丢失了，便先禀报药尊殿，而事关重大，便也直接下了通缉令。”
<br/>　　“至于其余各殿的通缉令。”副院长淡淡一笑。
<br/>　　“你这些朋友的本事，确实了不起。”
<br/>　　“药尊殿的通缉令，还有主殿执法队，根本奈何不得。”
<br/>　　“故我们学教，只得在另外的各殿也发布通缉令。”
<br/>　　“只是老夫平素交友不少，这才恰好这些主殿主们都来相助罢了。”
<br/>　　“当真？”萧逸皱了皱眉。
<br/>　　“当真。”副院长确定地点了点头，“你那些朋友，确实本事厉害。”
<br/>　　“若非承风殿主和横天殿主提前出现，并布下大阵，围困于此；恐怕单凭我们六殿之前的追击，都奈何不得。”
<br/>　　“另外，我知道你在怀疑些什么。”副院长认真地说道。
<br/>　　“可你曾是黑云学教弟子，便也该知道我们学教上至长老，下至一众弟子的为人如何。”
<br/>　　“云渊长老、讳执事、黑玄长老等等，还有童烨、秦翊等等一众弟子，谁会是那等胡乱安插罪名的小人？”
<br/>　　“又有谁会做那等欲加之罪的事？”
<br/>　　“学教既发出通缉令，自有其缘由。”
<br/>　　萧逸皱了皱眉，副院长的一席话，倒是顺理成章，并无什么值得怀疑的地方。
<br/>　　“唉。”边说着，副院长叹了口气。
<br/>　　“你身后那些朋友之事，我看在你的份上，可以当作误会，撤销掉通缉令。”
<br/>　　“那些丢失的天材地宝，我也会寻办法替你解决。”
<br/>　　“只是，你…”副院长指了指萧逸，手指颤了颤，“你刚才不应该这般冲动。”
<br/>　　“金绝玄，乃是天机殿主殿殿主，你当众将其击杀，其后果…”
<br/>　　“若你还当我是副院长，信得过我这个前辈，便随我去一趟天机殿赔罪。”
<br/>　　“我保证，即便拼了命，也会替你周旋到底。”
<br/>　　副院长眼中，尽是真挚。
<br/>　　萧逸的眉头，已然紧皱，因为他发现，副院长眼中的真挚，作不了假。
<br/>　　副院长言语中的那抹关心，也绝对作不了假。
<br/>　　“萧逸，放下剑吧。”副院长这时沉声道。
<br/>　　“随我去一趟天机殿，之后的事，交给我。”
<br/>　　萧逸闻言，皱着眉。
<br/>　　恰在此时。
<br/>　　下方白冰雪，蓦地喃喃自语，“黑云学教？”
<br/>　　白冰雪似是想到了什么，瞬间闪身至剑姬前辈身旁。
<br/>　　“师尊，黑云学教，前些天我们遇到那个泼辣女子时，她好像说过。”
<br/>　　“嗯？”剑姬前辈闻言，也似是想到了什么，皱了皱眉。
<br/>　　这时，副院长的脸色，陡然一变。
<br/>　　虽然副院长反应极快，这抹脸色一闪而过。
<br/>　　但萧逸敏锐下，还是捕捉到了。
