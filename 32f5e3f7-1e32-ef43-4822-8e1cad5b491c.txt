　　声音落下的同时，叶尘猛然伸手一抓，
<br/>　　轰！
<br/>　　一道巨大的金色手掌，直接朝柯莱特教皇狠狠抓落。
<br/>　　“哼！”
<br/>　　柯莱特顿时瞳孔一缩，之前身上的老迈气息，瞬间消失的无影无踪，脚下猛然一顿，下一刻已经退到数百米开外，险而又险的躲过了叶尘这一抓。

<br/>　　而那些西方教廷的教徒们，也全都精神大振，一个个重新燃起了希望。
<br/>　　“我就是嘛，只要教皇大人出手，这小子又能如何？”
<br/>　　“虽然这少年实力堪称恐怖，教皇大人也未必就输于他！”
<br/>　　“教皇大人万岁！”

<br/>　　众人纷纷欢呼起来，一些高手甚至开始跃跃欲试，准备联手围攻叶尘。

<br/>　　天空上，信心倍增的柯莱特，忍不住狂笑了一声，然而没等他的话说完，脸色顿时再次狂变，
<br/>　　只见，那道硕大的金色巨掌，仿佛穿梭空间一般，竟然如影随形的跟了过来，一把将柯莱特死死擒住！

<br/>　　老教皇柯莱特，顿时发出了一道犹如杀猪般的惨叫声。
<br/>　　“区区一只蝼蚁，也敢在本帝面前狂吠？真是不知死活！”
<br/>　　叶尘轻而易举的擒住柯莱特之后，又跟着随手一拉，便瞬间将逃离到千米之外的柯莱特，活生生拉扯到跟前。
<br/>　　周围众人顿时个个两眼瞪得来到，诺大的广场之上，更是死一般的寂静！
<br/>　　就连王硕和韩颖玉两人，一时间也忘记了逃命，不可思议的望着空中那道，犹如战神般的少年身影。
<br/>　　“老教皇之前叫他……叶狂仙，难道他就是……”
<br/>　　王硕脸色惨白，哆里哆嗦的说了一句，眼眸深处浮现出一抹，浓浓的敬畏之色。
<br/>　　身为华夏人，尤其又是京都八大家族中的嫡子，他又岂会不知道那个名字？
<br/>　　叶狂仙！
<br/>　　当初一人横压京都八大家族，是华夏武道界第一人。
<br/>　　之后灭高丽，屠岛国，横扫东南亚六国，不久前，更是将当时第一大国的米国，逼的举国投降！
<br/>　　如今已经是公认的全球第一人！
<br/>　　他就是传说！
<br/>　　是神话！
<br/>　　是华夏国的守护神！
<br/>　　他如果早知道叶尘的身份，就算是借给他一百个胆子，之前也不敢那么做。
<br/>　　一想到自己之前的所作所为，王硕顿时一阵后怕，背上冷汗直冒。
<br/>　　而一旁的韩颖玉，美眸之中则露出一抹异样的神采，望着空中犹如战神般的身影，低声喃喃道：
<br/>　　“原来他就是那位大名鼎鼎的叶神仙啊，没想到竟然如此年轻，长得也这么帅……”
<br/>　　……
<br/>　　另外一边，
<br/>　　老教皇被叶尘轻松擒住之后，顿时吓得脸都绿了。
<br/>　　直到此时此刻，他才意识到，自己是多么的无知可笑！
<br/>　　“叶仙人！我错了，求您、求您不要杀我……我教廷和神界，一直都有联系，只要您不杀我，我一定可以帮您找到神界的入口……”

<br/>　　面对老教皇凄惨的哀求声，叶尘却嘴角微微一扬，露出一抹戏谑之色，冷冷道：
<br/>　　“我给过你机会，可惜你不懂得珍惜，现在，已经不需要了！”
<br/>　　说完这话，叶尘手掌猛然用力一握，
<br/>　　“彭！”
<br/>　　可怜这柯莱特，西方教廷真正的主宰，统治欧洲大陆近两百余年的无上强者，竟然被叶尘随手捏爆！

<br/>　　他们心中至高无上的神祇，无所不能的上帝使者，竟然被人如此轻易的虐杀了！
<br/>　　他们的信仰，也随之彻底崩溃……
<br/>　　“叶狂仙！你竟然、竟然敢杀教皇大人，神界是绝对不会放过你的！”
<br/>　　一直跟随在柯莱特身后的那名红衣大主教，自知自己也难逃一劫，顿时大吼了一声，继而如同疯了一般，朝叶尘扑了过来。

<br/>　　那名红衣大主教，冲到叶尘跟前，整个人身上，竟然迸发出一道耀眼的白色光芒，继而直接爆炸开来，瞬间将叶尘淹没在了其中……

<br/>　　等那道璀璨白芒消失之后，众人顿时全都傻眼了。
