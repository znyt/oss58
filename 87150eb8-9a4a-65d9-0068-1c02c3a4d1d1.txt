　　楚风眠掌控龙之本源，一切龙族的隐秘，在他的双眼之中，都无从遁形。
<br/>　　现在的楚风眠，可以说是天下间一切龙族的克星。
<br/>　　纵然是这老者实力强横到了极点，可他依然无法逃过楚风眠的双眼。
<br/>　　“回答我的问题！小辈，你为何要见我！”
<br/>　　那老者看向楚风眠，怒声喝道，这声音之中的杀意，毫不掩饰。
<br/>　　楚风眠的目光之中，带给了他的一种无比不舒服的感觉。
<br/>　　只要是接下来楚风眠的回答，不令他满意的话，这老者就会立刻出手，轰杀楚风眠。
<br/>　　楚风眠听到这老者话中的杀意，脸色依然没有任何的变化，他既然是敢逼迫出这老者现身，也不会怕了他。
<br/>　　这老者的实力的确是强横无边，甚至这一道化身拥有的力量，都不在楚风眠之下。
<br/>　　不用这老者真身现身，单单是这以一道化身，都可以给楚风眠带来极大的威胁。
<br/>　　不过楚风眠想要全身而退的话，哪怕是这老者真身到来，也拦不住楚风眠。
<br/>　　“我见你，只是想要跟你做一个交易。”
<br/>　　楚风眠从容不迫的说道，像是一个生意人一般。
<br/>　　“我没有什么可以跟你做交易的，现在离开龙仙商会，忘记今日的事，不然你得死！”
<br/>　　那老者盯着楚风眠开口道，眼神中的杀意，丝毫不减，隐隐约约蕴含的杀意，已经是忍耐不住了。
<br/>　　“是吗？不用这么着急赶人，我倒是觉得我们可以聊聊。”
<br/>　　楚风眠嘴角浮现出几分笑容道。
<br/>　　“例如聊聊，逆天大劫的事。”
<br/>　　“万龙灭世！”
<br/>　　在提到逆天大劫四个字的时候，这老者脸色大变，刹那之间所有力量爆发而出，眨眼之间向着楚风眠轰击而去。
<br/>　　这老者一出手，爆发出来的威力，简直惊天动地，比起那五位金龙卫加起来的力量，都要强横无数倍。
<br/>　　庞大的龙力，化为了一道道远古巨龙，咆哮着向着楚风眠轰击过来。
<br/>　　甚至隐隐约约之间，都可以感受到一丝丝始祖天龙之力，蕴含在其中。
<br/>　　吼！
<br/>　　在这万龙凝聚现世的一刻，仿佛是古老的万龙之国降临了。
<br/>　　这老者一出手，甚至仿佛恢复了龙族巅峰，万龙之国，诸天龙族统御世间的一幕。
<br/>　　一条条灭世之龙，轰击着楚风眠，这每一条灭世之龙都比起寻常半步天人境强者的全力一击都毫不逊色。
<br/>　　楚风眠看到这一条条灭世之龙的到来，都是不为所动，龙力凝聚在楚风眠的掌心之中，突然一掌打出。
<br/>　　这龙力打出的一刻，也同样是在空中，化为了一条条的灭世之龙。
<br/>　　楚风眠打出的，同样是这一招万龙灭世。
<br/>　　这一招乃是在无数龙族武技之中，都极为高深莫测的一招，真正的天人境龙族，都未必可以领悟的一招。
<br/>　　任何的龙族武技，都是从始祖九式之中分化而出，这一招万龙灭世，也是如此，属于始祖第五式的一部分。
<br/>　　现在楚风眠已经是将始祖九式的前六式，全部领悟熟练，哪怕是始祖第五式，楚风眠都随意施展。
<br/>　　更何况这从始祖第五式之中衍生而出的万龙灭世，这一招楚风眠随手施展，信手拈来。
<br/>　　一条条灭世之中从空中碰撞，刹那间交锋在一起，不相上下。
<br/>　　尹龙仙在一旁看到这一幕，心中震撼不已，她背后这一位老者实力的强横，她可是一清二楚。
<br/>　　哪怕是现在苍龙宗之中最为古老的老祖，最厉害的一位强者，都远远不是这老者这一道化身的对手。
<br/>　　可眼前楚风眠居然是轻而易举的，抵挡住着这灭世之龙。
<br/>　　“始祖天龙血脉？你到底是什么人？你不是苍龙宗的人？”
<br/>　　那位老者看到楚风眠随手抵挡住了这灭世之龙，也是惊讶不已的说道。
<br/>　　灭世之龙这一招，可是从始祖第五式之中衍生而出的龙族武技，天人境龙族都未必可以领悟而出。
<br/>　　这老者也是曾经有所奇遇，才参悟出了这一招，凭借这一招，别说是半步天人境武者，哪怕是真正的天人境强者，都可以横扫。
<br/>　　眼下这虽然是他的一道化身，可横扫半步天人境都不成问题，现在居然没有杀了楚风眠。
<br/>　　更是令他震惊的是，楚风眠居然也可以打出了这灭世之龙，这除非是有着始祖天龙血脉，不然不可能做得到。
<br/>　　“始祖天龙血脉？看来你也是太久没有动手了，这都察觉不到？”
<br/>　　楚风眠看向那老者，嘴角一动，突然一掌轰击过去。
<br/>　　这一掌之中，根本没有蕴含的灵力，只是随手一掌。
<br/>　　可当看到这一掌的时候，那老者却是突然愣住了，他的眼中好像是看到了什么不可置信的事情一样，甚至是都忘记了躲闪，直接是被这一掌打飞了出去。
<br/>　　“师尊！”
<br/>　　尹龙仙急忙飞过来，看着那被打飞出去的老者。
<br/>　　“龙之本源！这是龙之本源！太初之气！怎么可能？怎么可能？”
<br/>　　那老者根本没有理会尹龙仙，而是一脸呆滞的喃喃自语，仿佛是受到了惊吓，被吓傻了一样。
<br/>　　“你不可能做到的事，他人未必做不到。”
<br/>　　楚风眠重新坐回了椅子上，倒了一杯茶喝了一口说道。
<br/>　　“如何？现在我们可以聊聊了吧，我可以帮你化解你的问题，甚至可以助你踏入天人境，如何？”
<br/>　　“太初之气！龙之本源，这真的可以有人成功？”
<br/>　　那一位老者还有些没有反应过来，在听到楚风眠的话后，还有些神志不清。
<br/>　　“亲眼所见，你还有怀疑？”
<br/>　　楚风眠淡然开口道。
<br/>　　“不是怀疑，只是一时无法相信罢了，没想到余生真的有机会见到度过逆天大劫之人。”
<br/>　　老者说了一句，语气之中的怒火也平息了下来，声音之中无比感慨。
<br/>　　逆天大劫，乃是他一生的心结，没想到今日居然因为楚风眠所解。
<br/>　　“终究想亲眼所见，楚公子，来我洞府，一叙如何？”
<br/>　　这老者看了楚风眠一眼，语气之中已经是无比客气了，没有了之前的那般杀意。
<br/>　　“那就见一见吧。”
<br/>　　楚风眠点了点头道。
