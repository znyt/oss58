<br/>    步留痕作为天庭之中的成员，还是知道一些天庭没有端掉的瑶池暗中势力的，步留痕戴着黑色的斗笠，走进一处红袖招酒馆之中，步留痕走进一个隔间之中，待得小厮进入隔间中，询问道：“不知这位客官想要吃点什么？”
<br/>    
<br/>        步留痕摘下斗笠，看向这个小厮，直接道：“你应该认得这张脸，让李红衣来见我！”
<br/>    
<br/>        小厮看到步留痕的脸之时就已经认出了步留痕，小厮道：“红衣大人会不会见你，我可不确定，不过，我可以帮你通报一声。”
<br/>    
<br/>        不过一会儿，门外传来轻轻的脚步声，一个抱着琵琶的红衣女子走了进来，赤足落地，三寸生香，红衣女子扫了步留痕一眼，笑道：“呦呵呵，这不是天庭的仙官，步留痕吗？哦，不对，你现在是天仙修为，现在应该算是天庭率领仙官的仙将。堂堂仙将，就是赤龙星河天庭也不多，怎么沦落到了这般地步？”
<br/>    
<br/>        步留痕道：“我也没有想到，叱咤玉龙星河的李红衣，居然只是一个卖笑的普通女人！你应该知道我现在的处境，我现在没空和你说风凉话，我现在只想知道一点，那就是瑶池可不可以帮我一把，让我安全离开赤龙星河。”
<br/>    
<br/>        李红衣笑道：“凡事都有着代价，你也知道，天庭现在封锁了整个赤龙星河，想要将你从赤龙星河带走，除非是能够攻破赤龙星河和孽龙星河的入口，瑶池，可没有办法带领大军随意进入孽龙星河。”
<br/>    
<br/>        步留痕道：“我有关于天庭的情报，关于造化仙果的！”
<br/>    
<br/>        李红衣的眼中闪过一抹精光，“怪不得天庭要不计一切代价追杀你，原来你知道了造化仙果的秘密，造化仙果，可是能够直接提升一个境界！什么级别的造化仙果？”
<br/>    
<br/>        步留痕道：“只是地仙服用的，你也知道赤龙星河只是一个小地方。”
<br/>    
<br/>        李红衣道：“只是地仙服用的造化仙果可没有什么用处，众多周知，强行提升境界成就的天仙，可完全不是真正的天仙的对手，只是说，凌驾于所有地仙之上而已，这样的造化仙果，根本不可能让瑶池冒
<br/>    
<br/>        着和孽龙星河其中的势力开战的代价去救你。”
<br/>    
<br/>        步留痕道：“就没得商量吗？”
<br/>    
<br/>        李红衣伸手，拿出一壶酒，给步留痕倒了一杯酒，然后笑道：“其实，也不是没有任何办法。”
<br/>    
<br/>        步留痕听到李红衣这么说，伸手端起李红衣倒得酒，一口饮尽，然后道：“还请阁下教我。”
<br/>    
<br/>        李红衣淡淡道：“我有四个徒弟，都死在了一个人的手上，你猜，那个人是谁？”
<br/>    
<br/>        步留痕问道：“是谁？”
<br/>    
<br/>        李红衣道：“除了你，还能有谁，我是真的没有想到，派遣四个地仙巅峰的弟子去杀你，竟然还被你杀了，直到现在才明白，原来你，早就是天仙修为。”
<br/>    
<br/>        步留痕忽然感觉到一阵头疼，自己是天庭的人，手底下瑶池的鲜血可是沾染了无数，求助瑶池，原来也是死路一条。
<br/>    
<br/>        他看向周围，可以感受得到周围有着人在靠近，而且修为都不低，至少是地仙！
<br/>    
<br/>        “所以说，你之所以和我说这么多，就是为了拖住我，方便你布置这些杀我的手段？”
<br/>    
<br/>        李红衣道：“鼎鼎有名，整个赤龙星河步法顶尖的天仙，是要废一番功夫！”
<br/>    
<br/>        步留痕伸手将酒杯摔碎，看了周围，一眼，已经有着三柄仙剑朝着他飞来！
<br/>    
<br/>        步留痕起身，抬腿之间将周围三个仙人踢的咳血！
<br/>    
<br/>        步留痕再次看向李红衣，却是发现李红衣并没有动手！
<br/>    
<br/>        “你若是不动手，可没有那么容易留住我！”
<br/>    
<br/>        李红衣微微一笑，然后道：“我可没有兴趣给你陪葬，这里是天庭的地盘，若是我暴露了，同样只有死路一条，我只需要，让你暴露在天庭的眼中就好，就像是现在一样。”
<br/>    
<br/>        李红衣道完，已经抽身离开了此处，而周围瑶池的地仙，已经是再次朝着步留痕围攻而来！
<br/>    
<br/>        步留痕口中喝道：“八步登龙！”
<br/>    
<br/>        他踏出一步，神威赫赫，脚下荡漾出一阵涟漪，直接将整个酒馆震碎，同时震碎的，还有着一位瑶池的地仙！
<br/>    
<br/>        步留痕看向周围，喝道：“李红衣已经抛弃了你们了，难道你们就不想活着吗？”

<br/>        周围有地仙冷笑道：“我的女儿，可是死在你的手里，只要能够杀了你，那么什么都不重要了！”

<br/>        步留痕知道不好，这里的人，根本就是留下来的死士！

<br/>        战斗的巨大动静果然引发了周围天庭巡天军的关注，而步留痕，也是在此刻落入了天庭的眼中，“是步留痕，干净禀报守星将大人，前来抓捕步留痕！”

<br/>        步留痕此刻深知，等到守星将赶到此处，自己必死无疑，现在，唯有重新杀回赤龙星河，隐藏起来！

<br/>        步留痕脚下神威赫赫，一圈圈涟漪弥漫周天，每一步落下，周围的天地都是一阵晃荡！
