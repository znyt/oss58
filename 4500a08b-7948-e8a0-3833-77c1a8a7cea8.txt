        只不过了，连星儿发现，自己对这小狼狗还是挺忌惮的。
<br/>
<br/>        她现在只是希望，他能记得她的好，不要老想着攻击或者偷袭她就好了。
<br/>
<br/>        离开首饰店，明司寒的生日礼物仍旧没有着落，这又让连星儿发起愁来。
<br/>
<br/>        这是她平生第一次给人挑生日礼物犯难了。
<br/>
<br/>        只因她完全不知道自己该送他明司寒什么样的生日礼物比较好。
<br/>
<br/>        直到连星儿路过一家“陶艺DIY店”，抱着随便看看的心态，才好奇地走了进去。
<br/>
<br/>        女店长见到是新顾客，于是主动迎上前来询问：“您好，请问您是需要做陶艺当礼物送人吗？”
<br/>
<br/>        “嗯，是送给未婚夫的。”连星儿坦然道。
<br/>
<br/>        女店长下意识地睐了连星儿身后的郎安一眼，不禁会心一笑：“那就送您自己动手做的杯子吧！你未婚夫，一定会喜欢。我们这儿还有专门是陶艺师傅手把手的教您。”
<br/>
<br/>        连星儿一高兴，立马就跟着店长去办理了一个上千元的套餐。
<br/>
<br/>        当连星儿坐在转台前在一个陶艺师傅的受教下做杯子的时候，郎安就坐在她身边看着。
<br/>
<br/>        而这个时候，店子的橱窗外，有人拿着手机正对着他俩拍照。
<br/>
<br/>        因为做完杯子后，还要釉下彩，再烧结。
<br/>
<br/>        连星儿索性多给了这店长一笔费用，让她能在店内打烊的时候继续完成自己的手工杯。
<br/>
<br/>        临海大学内，男生寝室里，明司寒正在看自己公司正准备做的一款游戏的策划案。
<br/>
<br/>        仲正浩从外回来后，利索地搬了条椅子往明司寒身边坐下后，直接夺走了明司寒手中的文件，放置在书桌上后，一边掏出手机，一边说道：“寒哥，我这人虽然花心，但我好歹是一个一个谈，但是，连星儿她这种一脚踏两船的行为，我就看不惯了！这事，你必须得知道。”
<br/>
<br/>        明司寒不明所以然地看着仲正浩，只见他点开手机相册后的第一张相片，就是连星儿拉着郎安的手的画面。
<br/>
<br/>        不，不是手，而是手腕。
<br/>
<br/>        她抓着他的手腕，脸上还带着笑容。
<br/>
<br/>        仲正浩接着义愤填膺地说道：“我今天陪我一个正在追求的妹子逛街，结果，我就看到连星儿拉着这个男生的手从我前方经过。然后我灵机一动，立马就拍了下来。然后，我陪妹子吃了晚饭，准备回来的时候，路过一家陶艺DIY店，又发现这两人竟然在一起做陶艺。”
<br/>
<br/>        “……”明司寒看着仲正浩手机里的相片，听着仲正浩的话，只是眉头紧皱，默不吭声。
<br/>
<br/>        仲正浩却仍旧没完没了起来：“她连星儿就算长得再漂亮，也不应该在跟你有婚约的情况下，跟别的男生暧昧吧！这又是逛街又是拉手又是玩陶艺的。想想，就觉得她是有多花心了。寒哥，为了你以后的幸福，你还是跟她解除婚约的好！这种女人，婚后出轨的概率极高！”
<br/>
<br/>        话到这里，仲正浩见明司寒一声不吭，终于闭嘴了。
<br/>
<br/>        此刻，明司寒忽然间回想起，起初在图书馆里所看到的画面。
<br/>
<br/>        他之所以在图书馆往上走，是因为他从图书馆里借完书出来遇到了连星儿的室友顾千凝。
<br/>
<br/>        顾千凝说：“星儿还在上面，你快去找她吧！我就不打扰你们二人了！”
<br/>
<br/>        那个时候，他听着顾千凝的话，还沾沾自喜来着。
<br/>
<br/>        结果，只是在楼梯间里，他才上一层楼，就撞见他们两楼在了一起。
<br/>
<br/>        难道，连星儿这么快就从洛羽那儿移情别恋到了那个名叫郎安的男生身上了吗？
<br/>
<br/>        明司寒心里很不是滋味，但又觉得自己没有任何理由去责怪或者是指着连星儿那丫头的任何不是。
<br/>
<br/>        毕竟，一开始，她便摆明了她自己的态度。
<br/>
<br/>        那就是她不喜欢他，联姻是受两个大家族的所迫，她还想着跟他解除婚约。
<br/>
<br/>        一个要跟自己解除婚约的女孩，他又有什么资格去说她的不是？
<br/>
<br/>        明司寒忽然间眉开眼笑，让仲正浩摸不着头绪。
<br/>
<br/>        星期五的这天晚上，连星儿没有回寝室，也没有回滨江城，更没去临海的傅家大庄园的祖爷爷家。
<br/>
<br/>        这是明司寒主动去了解到的，所以，他鬼使神差地给连星儿打了通电话。
<br/>
<br/>        此刻，晚上十一点半。
<br/>
<br/>        连星儿正在给自己做的杯子上彩釉。
<br/>
<br/>        她做了两个杯子，一个杯子上她画了自己，另一个杯子上，她画的是明司寒。
<br/>
<br/>        因为陶艺店里之前有师傅在讲课，所以她将手机调成了静音状态。
<br/>
<br/>        明司寒电话过来时，连星儿的手机放在了手提包里，她没听到，也就没有接听。
