　　第638章
<br/>　　船队排成一字长蛇阵，分开水浪顺流而下。程大雷站在首船的船头，身披血月披风，手里攥着匹夫剑，身后立着背斧的银眸。
<br/>　　单雄信立在左边，手里握着金顶枣阳槊，意气风发不可一世。在单雄信旁边立的是阿喜。这次行动本不用阿喜亲自出马，蛤蟆城但凡有行动，往往是他将喜字队得到的情报递过去，最多派一两名喜字队的成员配合。
<br/>　　但这次不同，这次是程大雷亲自行动，蛤蟆城也担心他有个闪失，所以喜字队的队长亲自出马。
<br/>　　“大当家，那黑风寨就在大河之畔，依山而立，属下已经绘了一份草图，我们趁夜发动攻击，当有七成把握。”
<br/>　　黑风寨的兵马数量要多过程大雷，兵力比达到一比三。不过，这次行动出动了程大雷和单雄信，还有个力大无穷的银眸跟着，其实也无多少风险。
<br/>　　唯一的问题是蛤蟆城不擅长水战，程大雷并没有积累这方面经验。不过这方面问题不大，程大雷也没有准备与黑风寨打水战的准备。集中兵力夜袭，打对方措手不及，己方派出三员大将，想来不会遇到困难。
<br/>　　在距离黑风寨五里处，船队选一弯曲的河道靠岸，再进就要担心被黑风寨望风的探子发现。当然，阿喜已提前派出人手，埋伏在黑风寨方圆，这个时间他们当已开始清理黑风寨外围的望风人员。
<br/>　　停船靠岸，战士从船上下来，在河滩上集结完毕。人人身穿黑色皮甲，表情肃穆，兵器是统一的制式大刀。
<br/>　　这一千人来自关鱼率领的鱼字军，由关鱼亲自传授的无极刀，而关鱼一手调教出来的队伍，也继承了关鱼的性格特点。沉默寡言，目中无人并且很猛。
<br/>　　现在一千人立在程大雷面前，没有任何窃窃私语的声音，千人一面，千人一体。在蛤蟆城内，若单论纪律性，鱼字军当排第一。
<br/>　　程大雷亲自练兵的机会不多，他多是作为精神象征存在。此刻看到眼前这一幕，程大雷满意的点点头，挥挥手道：“出发。”
<br/>　　留下一批人看守船队，其他人随着程大雷向黑风寨挺近。首先要穿过一片密林，才能靠近黑风寨的大本营。
<br/>　　此刻夜已经黑了，风呼呼的刮着，密林树梢碰撞，风中似乎隐藏着某种怪物的声响。
<br/>　　这不是『似乎』的幻觉，而是却又其事，程大雷侧了侧头，这声音乃是哭声。
<br/>　　伸手不见五指的密林中，突然响起哭声，程大雷也是一激灵，起了一身鸡皮疙瘩。
<br/>　　“这林子里会不会有某种狐狸精什么的？”程大雷道。
<br/>　　“我去看看。”
<br/>　　单雄信低声道了一句，提着大槊向声音传来的方向去了。
<br/>　　程大雷撇撇嘴：怎么半点大将风度也没有，这种事明明可以派一个小校去探探的么。
<br/>　　贸然前往，若遭了敌人埋伏，便会决定一场战斗的胜败。如此看来，单雄信想要领兵，还需一番历练呐。不过，他本身就是武艺强，但没有什么谋略之能。
<br/>　　程大雷担心他出事，挥挥手令士兵暂且停步，自己和银眸向前面去了。
<br/>　　拨开密林，前面出现一片空地，杨国忠抱着小妹躲在树上，在大树下几只豺狗正发出阵阵吼声。
<br/>　　哭声就是杨国忠发出的，他醒来后发现自己躺在一片滩涂上，周围一片漆黑，身上是又湿又冷。也不知道自己身在何处，只好抱着小妹，深一脚浅一脚的往外走。
<br/>　　这也恰是野兽出来觅食的时候，杨国忠就被几只豺狗围上了。亏得他见机得快，抱着妹妹三下两下上了一棵大树。但这几只豺狗明显不想放弃口边的食物，口中呜呜的叫着，几次纵跃，越来越接近树上的杨国忠。
<br/>　　杨国忠心慌意乱，若幼年时曾学几日武艺，断不会落到今日下场。可无奈何，百无一用是书生，自己抓着树干，心里已知道撑不了多久。情不自禁口放悲声，心中想若此刻谁会出面救我一条性命，当结草衔环报答救命之恩。
<br/>　　相比之下，他怀中抱着的杨小妹倒是镇定些，此刻睁大眼睛，骨碌碌转着。
<br/>　　恰在此刻，程大雷三人先后赶到。杨国忠一见忙道：“好汉，快救我一救。”
<br/>　　说罢，杨国忠就后悔了，实在是这三人都奇形怪状，看着不像好人。那小姑娘模样倒是俊俏些，但身后背着一柄大斧，如何看如何都有些古怪。
<br/>　　程大雷听见这话，顺势查看了杨国忠的信息，看罢之后大吃一惊。
<br/>　　“单二爷，快快将这几只畜生赶散。”
<br/>　　单雄信挥舞着金顶枣阳槊扑了出去，他还是托大了。这几头豺狗论速度论力量都不是人类可以抗衡，也就是仗着手中金顶枣阳槊利害，才没有吃亏。这几头畜生，也不想放弃嘴边的美食，有一只绕到他身后，朝他后勃颈扑来。
<br/>　　银眸反应很快，上前一步抓住豺狗的尾巴，狠狠一甩，直接将它砸死在一块山石上，顿时脑浆迸裂。
<br/>　　这一幕却是吓惨了杨国忠，谁能拖起一只百十斤重的豺狗，在空中抡个圈。这女孩看似瘦弱，但不是天上下凡的神人，便是黄泉中的魔王。
<br/>　　程大雷三人各自出力，总算将几头畜生赶跑。杨国忠踉跄从树上掉下来，膝盖仍然有些发软。程大雷赶忙将他扶起，口中道：“啊哈哈，先生如何行此大礼，不至于，实在不至于。”
<br/>　　“大王饶命，大王饶命。我们身上没有金银，只要大王肯绕我们兄妹一命，来世当牛做马报答大王。”
<br/>　　程大雷摸摸鼻子，心道：这个……他怎么知道我是山大王，我脸上又没字。
<br/>　　他委实小看了自己，自己生得虎背熊腰，凶神恶煞，无论如何看，都不像是好人。
<br/>　　程大雷倒不是很在意杨国忠，他手下虽然缺少谋士，但他毕竟不是一个很有上进心的人。他的注意力还是落在那小女孩身上。
<br/>　　黑暗中，两人大眼瞪小眼，各自沉默。
<br/>　　这位便是那羞花的美人么，年纪……委实有些小呐。
<br/>(本章完)
