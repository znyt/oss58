　　布克尔特的童年充满着黑暗，老西伦科特不承认他们母子俩的存在，他们只能住在肮脏、潮湿的贫民窟中，艰难度日，直到布克尔特的妈妈生病死去。
<br/>　　对于布克尔特来说，他的妈妈就是他最温暖的港湾，也是最值得他怀念的人，所以他用自己妈妈的生日来作为密码，很容易理解。
<br/>　　宋萱这次收集到的资料很齐全，其中就包括他母亲的信息。
<br/>　　只是布克尔特的母亲是个中国人，被布克尔特用作密码有可能是他妈妈阳历生日，也有可能是他妈妈的阴历生日；不过现在罗昊只剩下最后一次尝试的机会，他最后还是决定使用布克尔特妈妈的阳历生日。
<br/>　　“这是我们最后一次尝试的机会。”
<br/>　　罗昊看了眼身边正在积极做着防御的战术中队队员，说道：“如果失败，我们就立刻撤退，再寻找营救机会！”
<br/>　　安全屋中所有战术中队队员都很清楚，如果这次营救不能成功，他们想要再找到这样的营救机会就会很难，而且难保布克尔特不会再利用耍什么阴谋诡计，到时候就要步步受制于他了。
<br/>　　慕容飞脸色有些难看，但最后还是点了点头。
<br/>　　因为慕容飞自己也很清楚，如果不能在第一时间撤离，他们所有人都有可能都会栽在这里，这样他们就连以后救佟千雪的机会都不会再有。
<br/>　　“准备突围！”
<br/>　　齐凡对着周围其他战术中队队员下令。
<br/>　　孙涛、司徒空、林晓阳、周柏手中分别扣住了闪光震撼弹和手雷，周立也是把轻机枪的子弹袋重新更换了一个全新的，随时准备进行突围。
<br/>　　罗昊右手握住快枪套中的uSP.45手枪，左手伸向钢板门上的密码盘，深吸一口气，然后在密码盘上按下了布克尔特妈妈的生日，并且按下了确认键。
<br/>　　“PASS！”
<br/>　　“嘀”的一声，系统通过了罗昊输入的密码。
<br/>　　咔嗒！咔嗒！咔嗒！
<br/>　　钢板门完成了解锁。
<br/>　　“怎么会这样，为什么会这样！？”
<br/>　　布克尔特难以置信的看着地下室门口的方向，他想不明白为什么龙隐部队的人能猜出他设置的密码！
<br/>　　“看来你输了。”佟千雪对布克尔特说道。
<br/>　　“输？”布克尔特摇了摇头，说道：“我不会输！别忘了，我手里还有你这个人质！只要你还在我手中，我就还是有胜算的！”
<br/>　　布克尔特将佟千雪抓到自己面前，左手勾住她的脖子，右手握着手枪顶在她的太阳穴上，看着地下室门口的方向。
<br/>　　罗昊左手握着强光战术手电，右手握着uSP.45手枪，沿着楼梯缓步走进地下室，他身后，梅庆伟、齐凡等人一边对着安全屋外的CiA行动队射击，一边向地下室撤退，周立对着林子中正在朝安全屋靠近的CiA行动队员甩出一颗手雷，扫出一个长点射后，也是跟着撤入地下室，然后把地下室的钢板门关闭锁死。
<br/>　　林子中的CiA行动队总算是冲进了安全屋，可是留给他们的只是满地的子弹壳和那扇已经关闭并且重新锁上的脱氧钢板门。
<br/>　　Royce，前海豹突击队队员，退役后加入CiA，他是这次行动队的队长。
<br/>　　Royce走到脱氧钢板门前，检查了一下脱氧钢板门上的密码系统，说道：“联系HQ（headquarters，总部），把我们这边的情况告诉他，让他想办法联系到布克尔特得到这个密码系统的密码。”
<br/>　　“明白！”
<br/>　　一名CiA行动队员点头走到一旁，跟总部进行联系。
<br/>　　地下室中，罗昊握着手枪正在跟布克尔特进行对峙。
<br/>　　“你们能猜到我设置的密码，进入到这里来真是有些出乎我的意料。”布克尔特把手枪顶在佟千雪太阳穴上，藏在佟千雪身后对罗昊说道。
<br/>　　慕容飞拔出手枪，拉开击锤，走到罗昊身边，双手持枪瞄准，对着布克尔特大声说道：“布克尔特，你现在立刻放了她！”
<br/>　　“慕容……”佟千雪红着眼睛看着慕容飞。
<br/>　　“你当我傻吗？”布克尔特开口说道：“放了她？我只要放开她，下一秒就会有至少两颗子弹击中我脑袋！”
<br/>　　“那你觉得你藏在她的身后就能躲一辈子吗？”罗昊跨前一小步，眼睛四下环顾，寻找着进攻的机会。
<br/>　　“我不需要躲一辈子，我只要等到外面的人把地下室的门打开，一切就都结束了！”布克尔特回答道：“你们的枪都很快，可是我的枪也不慢，不如我们就拿这个女人的命来赌一把，到底谁的枪更快一点？”
<br/>　　“慕容，救我……”佟千雪带着哭腔对慕容飞说道。
<br/>　　“无常，保持冷静！”
<br/>　　罗昊瞥了眼随时准备冲上去的慕容飞，说道：“你的冲动很有可能会害死佟千雪！冰刃，把无常换下去，他现在的状态不适合营救行动！”
<br/>　　“我可以！”
<br/>　　慕容飞手心中渗出了一些汗水，他扣在扳机上的右手食指也在轻轻颤抖。
<br/>　　“立刻退下去，这是命令！”
<br/>　　罗昊的语气很严厉，没有任何可以商量的余地。
<br/>　　齐凡走上前，拍了拍慕容飞的肩膀，说道：“相信我们，我们一定会把佟医生救下来！”
<br/>　　慕容飞看了眼罗昊，又看了眼齐凡，最终还是把手指从扳机上放开，移到扳机护圈外，垂下手枪，退到了齐凡身后。
<br/>　　叮铃铃……，叮铃铃……
<br/>　　布克尔特放在桌子上的卫星电话响了起来，是赫克托打给他的电话。
<br/>　　罗昊看了眼响个不停的卫星电话，也明白了这个电话是什么用意，这对于罗昊来说可能是个机会，也有很大概率是唯一的机会！
<br/>　　叮铃铃……，叮铃铃……
<br/>　　桌子上的卫星电话还在响着。
<br/>　　罗昊猜出这个电话的用意，布克尔特更是没有道理不知道，他甚至知道这个电话是谁打来的。
<br/>　　布克尔特是个很冷静的人，但是现在这急促的电话铃声却是令得他不再那么冷静。
<br/>　　对峙很耗费精力，如果没有密码，外面的CiA想要用焊枪切开地下室的门至少需要一个小时，布克尔特不敢保证自己能不能在眼下这种情况下再坚持一个小时！
