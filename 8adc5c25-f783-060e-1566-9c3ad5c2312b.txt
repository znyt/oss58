　　当混沌道君从荒古禁地深渊出来时，身边随着一道身影，正是大成圣体。道君终究是没有复活女帝的哥哥，但他却与女帝达成协议，道君提供突破修为的方法，女帝加入混沌神殿。
<br/>　　如今的大成圣体，身上哪有什么红毛，又哪有之前那硕大的锁链，如今的他，身体已经恢复到了最巅峰的状态，全身圣血汩汩，气势逼人，力量惊世，若说此时与大帝叫板，绝对有人相信。
<br/>　　这才是大成圣体该有的样子，至于之前那样，只是个神志不清的怪物罢了。
<br/>　　“道君，我们如今欲往何处？”
<br/>　　身后，大成圣体凌天恭敬道。他神色间有些疑惑，不知道这天地间怎会有如此恐怖的人物存在，又疑惑这般恐怖的人物又想要做什么。
<br/>　　他亲自见到眼前这人伸手间伸缩时光长河，岁月中的画面，就仿佛看电影一般。一念动而苍穹震，而且，最让人奇怪的是，天地竟然没有降下天劫来劈死他，真是恐怖！
<br/>　　困扰自己无数年的圣体诅咒，竟然让这人一言就破了，当时人情况如今还历历在目，道君口吐一个“破”字，他身体之间的枷锁便全部消失，诅咒也不见了，全身恢复到巅峰状态，真是恐怖！
<br/>　　言出法随！
<br/>　　“现在么，当然是再次招人了，混沌神殿中如此多资源，就只有这么几个人，怎么足够？更何况……”道君说道这里，停顿了一下。
<br/>　　“更何况什么？”圣体凌天疑惑道。
<br/>　　“更何况作为主神，又怎么会只有这么几个小弟，哪个主神手下没有千万小弟，犹如蝗虫过界，天灾降临，掠夺万千世界。”
<br/>　　“主神，是什么？”
<br/>　　道君瞥了他一样，淡淡道：“你迟早会知道。”
<br/>　　语罢，道君不再多言，一步踏出，两人便消失不见，紧接着，一股谁也不能察觉的莫名波动由无尽虚空中传出，扩散到整个宇宙。
<br/>　　……
<br/>　　一间豪华的屋子里。
<br/>　　“可恶的哥哥，我早晚要镇压你，叫你不理我！”
<br/>　　一个十七八岁的少女，皱了皱那可爱的小嫩鼻，露出小小的虎牙，做出恶狠狠的样子，恨恨道。但话里却透露着一种调皮的意味，嘴角微微翘起，像是想起了她口中的哥哥被镇压的模样。
<br/>　　这少女身穿紫衣，有一种贵不可言的气质，她露出微笑时，有一种别样的风清，眼波流转，顾盼生辉，左脸颊有一个小酒窝，看起来非常动人与俏皮。
<br/>　　紫衣少女极其美丽，眼睛很大，眼眸很长，带给人一股灵气，看起来非常活泼与俏皮，给人一种灵动的感觉。
<br/>　　“哥哥已经神体小成了，想要镇压他，好像有些困难呀。”少女好似想到了什么，有些为难道，“不过，我是不会放弃的。”
<br/>　　“要怎么办？呃……”紫衣少女一步走出，突然就消失在房间之中，声音也随着消失。
<br/>　　房间中空空如也，就好似从来都每人来过一般。
<br/>　　……
<br/>　　一座山峰之上，星光璀璨，好似银星闪耀，耀眼无比，星辰垂落，与周边其他山峰形成鲜明的对比。
<br/>　　此山峰名为星峰，乃太玄门一百零八峰的主峰。
<br/>　　星峰上的一片丛林之间，山峦秀丽，景色优美，薄雾如烟，缭绕在山顶，山脚下则流水潺潺，古木伴老藤，甚是幽静。
<br/>　　“叮叮咚咚！……”
<br/>　　芳草间，一蓝衣男子正在抚琴，双手浮动，如蝴蝶翩翩，给人以灵动而又轻盈的感觉，十指划动琴弦，乐声优美而动听，让人心灵宁静。
<br/>　　男子身材修长，不过二十三四岁的样子，一身水蓝色的衣衫随风飘舞，给人以柔和灵动的感觉。
<br/>　　他身上有一种独特的气质，如流动的云，似浮动的风，有一种缥缈轻灵的气质。
<br/>　　仔细倾听，便会听出他此刻的琴声虽然悠扬悦耳，但却也有一种被命运束缚，挣扎不断，始终逃不脱的感觉，有一种苦涩的味道在里面。
<br/>　　“命运啊，挣不断，逃不脱，难道我一生真的只能为他人做嫁衣？不，我不甘心！”男子心中苦涩，但也有一种坚定不移的信念在其中。
<br/>　　不知过了多久，男子起身，轻轻抱起眼前的琴瑟，转身向山顶走去。
<br/>　　男子刚走两步，便消失在山林之间，谁也没有发现这里的情况。
<br/>　　……
<br/>　　“打！打死他，给我使劲的打！往死里打！”
<br/>　　一个面目凶狞的男子对着地上被他一脚踹倒的年轻男子恶狠狠喝道。
<br/>　　他身后的几个面目凶恶的壮汉急忙上前，对着地上的男子一阵拳打脚踢，丝毫没有留情。
<br/>　　“砰！咚……”
<br/>　　暴打声不断响起，地上的男子只有十一二岁的样子，他身体清瘦，一身破烂衣衫，仿佛一个小乞丐一般。他面目坚定，紧紧地盯着旁边的男子，仿佛没有感到身体的疼痛一般，一点也不像其他人那样，哭喊出来。
<br/>　　“敢偷宝爷的东西，你小子活腻歪了，给我使劲打！”男子恶狠狠道。
<br/>　　一阵暴打之后，小乞丐身上的气息渐渐衰弱了，好似随时可能熄灭似的。
<br/>　　“毛爷，他好像快要不行了。”一个人突然说道。
<br/>　　旁边的男子一看，好像真是如此，便吐了一口唾沐，转身大声喝道：“走，管他生死！”
<br/>　　接着，几人便消失在小巷之间。
<br/>　　不知过了多久，地上的少年突然动了动，他缓缓睁开那沉重而又疲劳的双眼，挣扎着站了起来，他的目光坚定而又明亮。
<br/>　　“我不能倒下，妹妹还等着我回去呢。”少年喃喃道。
<br/>　　他向着四周看了看，发现没人，便找了一个方向，摇摇晃晃地向着远处走去。刚走没几步，他便消失在小巷间，一阵风吹过，卷起地上的几片枯叶。
<br/>　　……
<br/>　　山林间，一个刚杀死一个敌人的少年消失不见。
<br/>　　楼房里，一身雪白衣裙的秀丽女子消失不见。
<br/>　　大街上，一个负剑修士消失，谁也没有发觉什么不对。
<br/>　　一座古老的墓地之中，一个无良道士消失，只留下一道断断续续的声音：“无量他妈个天尊……”
<br/>　　洞府里，一个完美得不像话的女子消失。
<br/>　　高空上，一只金翅大鹏消失。
<br/>　　一个头戴白纱遮面的女子消失。
<br/>　　两个被追杀得重伤的男子突然消失，只留下一众敌人面面相觑……
<br/>　　……
<br/>　　这些消失的人，全部在在同一时刻，来到一片灰蒙蒙的神殿之中，还来不及探听情况，便有一股信息传入他们的脑海之中，紧接着，一道神秘的声音在他们脑海之中响起：
<br/>　　“欢迎来到混沌神殿，幸运或者即将不幸的人、妖、佛、魔，又或者其他。仙，有所不能，混沌神殿，无所不能！”
<br/>　　……
