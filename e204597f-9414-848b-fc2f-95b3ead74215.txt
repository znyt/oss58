
<br/>　　<br/>　　    “等会，别动！”<br/>　　<br/>　　    陈三岁对面冰层里冻着的是一具尸体，青年，三十岁左右，身上穿着套墨绿色的军装，右胳膊上印着黑底白色的万字符，冰层冻得不是很厚，里面的尸体看的比较清楚，五官清晰可见，甚至连尸体上的毛孔都一清二楚。<br/>　　<br/>　　    上面王惊蛰和孔良不知道怎么回事，就出声问了一嘴，陈三岁回道：“下面有具尸体，好像是二战时的人，看衣服应该是纳粹的士兵”<br/>　　<br/>　　    王惊蛰当即一愣，孔良顿时惊呼道：“那个传闻是真的？当年，真的有纳粹士兵押送着大批珍宝来到了藏区深入到喜马拉雅山中，给藏到了这里？”<br/>　　<br/>　　    “那群小鬼子，目的出来了啊……”王惊蛰说道。<br/>　　<br/>　　    陈三岁舔了舔嘴唇，咬牙说道：“哥，我的民族气节告诉我，落在这九百六十万土地上的东西，那就是咱的了，别说别人不能碰了，小鬼子肯定更不行啊，我觉得咱是不是要黑他们一把？”<br/>　　<br/>　　    “你先上来再说吧，他们的问题等会研究，我的民族气节也烧得正旺呢，更何况他们对咱也是居心叵测的，我这人心眼小，有仇等不了多久，必须当场得报！”<br/>　　<br/>　　    民族气节必须是有的，这是旧恨，一天前发生的事算是新仇，新仇旧恨一起算的话，王惊蛰是注定得让三井美奈这一伙小鬼子，遗恨在喜马拉雅山上的，绝对没跑！<br/>　　<br/>　　    陈三岁被拉上来后，三人继续往洞深处走，有了先前这个意外的教训，孔良改用登山杖先来探路了，走几步敲几下，如果冰层下面有空鼓，敲出来的动静一听就能听的出。<br/>　　<br/>　　    这么一来，速度是慢了，但脚底下的安全就能有保证了，山内不知山外事，大概走了几个小时走走歇歇后，洞外已经是午夜之后了。<br/>　　<br/>　　    山洞里的路明显是感觉着有个坡度的，盘旋着一直向下，这么一走的话，基本就能品出来了，当他们从洞中出来的那一刻，势必要接近天坑的底部了，只是谁也想不到迎接他们的会是怎样一副天地。<br/>　　<br/>　　    当清晨的第一缕光洒向喜马拉雅山脉的时候，天还没有亮起。<br/>　　<br/>　　    天坑附近，昨夜在此露营的三井美奈已经起来半天了，正在简单的洗漱着，附近是小林昙一和三井良河正在收拾着行装。<br/>　　<br/>　　    那个昨天带领他们来到天坑的藏民，一晚上都睡在了外面，此时正匍匐在地上朝着天坑的方向，吟诵着晦涩难懂的藏经。<br/>　　<br/>　　    队伍收拾完行装，小林昙一走过去继续和藏民交涉，但这一回无论他怎么说，不管许下什么承诺，始终都晃着脑袋，不肯在为他们带路了。<br/>　　<br/>　　    小林昙一无奈，跟三井美奈解释道：“他说带我们来已经是对神明最大的不敬了，所以不可能在领着我们进入天坑里了，什么条件都说不通，看来只能由我们自己去找入口了”<br/>　　<br/>　　    三井美奈说道：“只要有入口就行，最多浪费点寻找的时间，这个天坑四周的面积很大，那我们分出两队人从两个方向同时环绕着走，哪一方先找到就从哪个口子进”<br/>　　<br/>　　    “也行，只好这样了……”<br/>　　<br/>　　    三井小叶这时担忧的问道：“吉野他们还没有跟过来，迷路是不可能的，他们也带了导航，唯一的可能就是出事了”<br/>　　<br/>　　    三井良河不可置信的说道：“他们带着枪，就去对付三个人，还干不掉？”<br/>　　<br/>　　    “也不一定是这么回事，你别忘了还有那头大型的犬类和马熊……”<br/>　　<br/>　　    三井美奈摆手说道：“这个问题暂时先别考虑了，我们还是寻找入口要紧，马上分成两队，现在就开始顺着山坡下去！”<br/>　　<br/>　　    于此同时，当第一缕阳光落向喜马拉雅山的时候，在洞中的王惊蛰忽然感觉到了一股暖风从前方吹了过来，很有种春风拂面的意思，脚下的冰层也渐渐的消失了，变成了湿润的土地，孔良反应很快的点头说道：“估计我们马上就要出去了，小心一点”<br/>　　<br/>　　    “呼……”陈三岁吐了口气，扣着眼屎说道：“总算是要出来了，走了一将近一天一夜了，脚底板都要磨破了，出去后吃点东西眯一觉吧，不然我都怕要猝死了”<br/>　　<br/>　　    这个山洞走的，说实话都要把人给走崩溃了，脚下冰层太滑，还得时刻提防着冰层断裂，走的累精神更累，一路走下来人始终绷紧了神经在加上小心翼翼的，确实都要把人给揍崩溃了，片刻后当他们看见一缕光芒透在了洞口的地面上时，三人终于长长的舒了口气，那种空气中弥漫着的泥土的芬芳，很是让人陶醉不已。<br/>　　<br/>　　    “踏踏踏，踏踏踏……”三人一路快步，几步就走了出去，站在洞口向着远处眺望，几乎集体就瞠目结舌的张着嘴巴瞪起了眼睛。<br/>　　<br/>　　    之前在天坑上方朝下望的时候，映入眼帘的只是大片被绿植覆盖着景象，这就跟你从高空俯瞰森林差不多，但是此时当他们深处天坑底部的时候，才真正的明白了什么叫壮观两字。<br/>　　<br/>　　    他们站着的洞口距离地面大概二十米左右，从这里眺望远处，首先看见的就是一大片古老的原始森林，所谓的古老就是三人在外界根本都没有见过或者熟识的树木，从上方看不出来这些古老的树木有多高，但从下面看的话几乎有很多差不多都有七八曾甚至十层楼以上了，参天大树开枝散叶，一棵古树的面积铺展开来后几乎比一个篮球场还要大了。<br/>　　<br/>　　    地上是一片厚厚的灌木和草丛，不时还有鸟兽从中掠过，这里的景象整个一人间仙境的感觉，看的不免让人忍不住要一声惊叹，果真还有世外桃源这一说。<br/>　　<br/>　　    从天坑上向下望，大概感觉这里的面积可能会有几个足球场那么大，但等他们深入到天坑底部的时候，才发现之前的判断应该是差了不少，这天坑底下在实际上，应该还要更为广阔了一些。
