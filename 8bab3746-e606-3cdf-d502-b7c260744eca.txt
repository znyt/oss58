　　八龙焚火炉祭出，漂浮于金光坑洞之上。
<br/>　　“吸。”萧逸低喝一声。
<br/>　　道道金光，被吸摄入内。
<br/>　　萧逸一边操控着八龙焚火炉，一边感知着里头情况。
<br/>　　金光力量进入八龙焚火炉中，暂时未有异动。
<br/>　　萧逸见状，点了点头，随后加快了吸摄的速度。
<br/>　　霎时间，金光以极快的速度涌入炉内。
<br/>　　八龙焚火炉，看似并不多大，但里头自成空间，可装纳东西的范围可不小。
<br/>　　自成空间这一效果，一般圣器以上阶品的重宝都有。
<br/>　　不过，这些重宝大多是武者手中武器，乃是战斗之物。
<br/>　　故即便有此效果，武者也不会用来藏纳物品，大多是将身上东西放于乾坤戒。
<br/>　　可八龙焚火炉不同，虽也是重宝，但萧逸并未用它来当战斗之物。
<br/>　　每每使用，都仅仅是用作炼药罢了。
<br/>　　故将这些金光力量吸摄存放于炉内空间，并不成问题。
<br/>　　另外一点，这片空间萧逸也感知过。
<br/>　　空间之内，火焰熊熊，温度惊人。
<br/>　　一团团火焰气息，如同亘古火焰，燃烧不尽。
<br/>　　那便是龙炎吐息。
<br/>　　当然，这是在东域时，剑姬前辈与他说的话。
<br/>　　这些火焰到底是否真的是龙炎，萧逸可不敢确定。
<br/>　　且在他的对比中，这些火焰确实强，甚至远比顶尖火焰强得多。
<br/>　　但较之紫晶灵炎这等强悍火焰，却明显差了些许。
<br/>　　如果真的是龙炎，理应不会比世间强悍火焰差才是。
<br/>　　不过到底怎么回事，萧逸也不清楚。
<br/>　　他一直身怀这八龙焚火炉，大多时候的作用，还是用于炼药。
<br/>　　炉内空间，本就火焰熊熊，温度惊人，故在炼药时，效果极好，速度也极快。
<br/>　　特别是他每每炼药，都是炼制大量丹药，用八龙焚火炉，自然是最好的选择。
<br/>　　现今，他同样感知着炉内空间。
<br/>　　道道金光，便是吸摄入这片熊熊火焰空间之内。
<br/>　　金光的进入，即便是现今汹涌而入，却仍旧未有任何异动。
<br/>　　金光与熊熊火焰之间，并无半分冲突。
<br/>　　火焰，并未燃烧金光；金光，也未让火焰的熊熊之势产生半分影响。
<br/>　　“看来应该问题不大。”萧逸自语一声，脸上露出一抹自信。
<br/>　　既然金光能放于炉内火焰空间，那么接下来就是准备阵法与禁制的问题了。
<br/>　　待得八龙焚火炉吸摄够这些金光力量，直接以阵法封存之即可。
<br/>　　想罢，萧逸收回了对八龙焚火炉的感知。
<br/>　　接下来，手中一块块中品灵石取出。
<br/>　　灵石，以玄妙的方式，摆放于周遭。
<br/>　　毫无疑问，他在布置阵法。
<br/>　　真正强大的阵法，是需要布置的。
<br/>　　对比他以前随手而出的风雪大阵、火焰大阵等等，都是他本身就深谙此道，加之阵法层次不高。
<br/>　　故能随手拈来。
<br/>　　而现在，要布置的阵法可是要封下这些威力强猛的金光，自然层次不同。
<br/>　　在他得到的的第一块冰尊令上，他获得了冰尊者一身的阵法手段。
<br/>　　其中，有一阵法名唤‘冰玄封界阵’，乃是冰尊者阵法传承中有数的几个最强阵法。
<br/>　　此阵法非是杀阵，无杀敌之效。
<br/>　　但却有着极强的封印以及围困之效，乃是一上古封印大阵。
<br/>　　此阵，萧逸借助大量灵石，以及自身就修炼寒冰一道的情况下，可以布下。
<br/>　　当然了，这需要耗费不短的时间。
<br/>　　萧逸现今的阵法本事虽不弱，但也远远不及那些阵法大能，更别说上古阵法大家了。
<br/>　　这一封印大阵，想必要费他不少功夫。
<br/>　　不过，对比了一下八龙焚火炉此刻吸摄金光的速度；要彻底吸满整个火焰空间，想来时间也不短。
<br/>　　恰好，待得火焰空间被吸摄满金光力量，自己的阵法也差不多该完成了。
<br/>　　一旦将这些金光力量吸摄足够，便立刻收回八龙焚火炉，再以冰玄封界阵封炉，必能封下这些金光力量。
<br/>　　萧逸这般想着，已做好了打算。
<br/>　　整整一个时辰后。
<br/>　　萧逸周遭，已玄妙地放置好一块块灵石。
<br/>　　阵法还未完全布下，只是大概布置好阵基，接下来便是完善整个大阵。
<br/>　　周遭灵石分布，囊括了近百米范围。
<br/>　　萧逸盘膝坐到中间，手掌虚张，一丝丝寒冰力量涌现。
<br/>　　极致的寒冰气息，被萧逸精准地达到一块块灵石之上。
<br/>　　冰玄封界阵，乃是冰属性封印大阵。
<br/>　　萧逸要布置之，元力并不够，只能借助灵石的力量。
<br/>　　现今打出的极致寒冰气息，不过是‘引子’。
<br/>　　这所谓的‘引子’，会引出灵石内的力量，既而大幅度增加这一道道极致寒冰气息的力量。
<br/>　　又是整整一个时辰后。
<br/>　　“呼。”萧逸轻呼出一口气，擦了擦额头上密布的汗水。
<br/>　　要布下一个真正的强悍大阵，可绝对不是一件简单之事。
<br/>　　“阵法已成，只等八龙焚火炉吸摄够金光力量，即可立刻发动。”萧逸略微煞白的面庞上，露出一丝喜意。
<br/>　　手中，天机圣盘已经出现。
<br/>　　不错，萧逸打算以天机圣盘来催动这个大阵。
<br/>　　冰玄封界阵，本就极强。
<br/>　　再加上阵法至宝天机圣盘的增幅，必能威力恐怖。
<br/>　　集两大天地至宝之力，再加一上古封印大阵，萧逸就不信这些金光力量还跑得掉。
<br/>　　这时，他只需等待着。
<br/>　　又是半个时辰后。
<br/>　　萧逸放出感知。
<br/>　　八龙焚火炉内，那个庞大的火焰空间，在这般极速的吸摄速度下，渐渐被金光力量所充斥。
<br/>　　要不了多久，就能彻底吸摄满。
<br/>　　十数分钟后。
<br/>　　萧逸眼中一道精光迸发而出，“满了，大阵，启…”
<br/>　　萧逸暴喝一声。
<br/>　　火焰空间，已装满金光力量；现在，正是发动大阵封印之时。
<br/>　　不过，萧逸的动作，忽然一滞。
<br/>　　不知道为什么，他总感觉，有些莫名的不对劲。
<br/>　　这股不对劲，来源于八龙焚火炉，可到底哪里不对劲，萧逸又说不出来。
<br/>　　萧逸冷冽的目光，直视八龙焚火炉。
<br/>　　半晌，脸色一惊，明白了过来。
<br/>　　他终于发现了哪里不对劲。
<br/>　　那便是此刻的八龙焚火炉，置身浑厚金光力量内，炉内空间又装满了金光力量，可它的炉身，无丝毫变化。
<br/>　　一如平时的古朴、平静，一如萧逸平时用来炼药时的模样。
<br/>　　即便是冷焱剑靠近这些金光力量，都金光熠熠，气息大增。
<br/>　　可八龙焚火炉，却无丝毫变化。
<br/>　　这些金光力量，对它而言，似乎毫无作用。
<br/>　　半晌，正当萧逸还在思索着时，再次脸色一惊。
<br/>　　眼前，巨大坑洞之上，原本漂浮着的八龙焚火炉，忽然剧烈抖动。
<br/>　　这般剧烈抖动，他略微感到有些熟悉，好像在哪里看到过。
<br/>　　惊骇下，连忙放出感知。
<br/>　　萧逸的脸色，蓦地变得怪异起来。
<br/>　　炉内火焰空间中，那些浑厚金光力量，正不断冲击着炉内的封印。
<br/>　　冲击着的，正是八龙焚火炉的第二道封印。
