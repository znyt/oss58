　　第31章
<br/>　　“你们把我家小姐怎么了？吃了吗！”说到这里，小姑娘抹着眼泪呜呜哭了出来。
<br/>　　程大雷几人目瞪口呆，程大雷问：“这个小姑娘是谁呀？”
<br/>　　“小蝶，苏家小姐的丫鬟，从落叶城找来的。”林少羽道。
<br/>　　苏樱从屋子出来，吃惊道：“小蝶！”
<br/>　　小蝶眼泪猛地顿住，战战兢兢道：“小姐，你还活着，没有被他们吃掉。”
<br/>　　苏樱也很无语，挥挥手道：“你过来，我们屋子里说话。”
<br/>　　小蝶随苏樱进入房间，小蝶扯扯苏樱的衣服，捏捏她的胳膊。终于长松一口气：“小姐，你还活着，真是太好了。”
<br/>　　忽然，她一顿，痛苦道：“那么……是他糟蹋了你？”
<br/>　　苏樱脸上一红：“胡说什么呢，我没事。”
<br/>　　屋子外，程大雷问林少羽这是什么情况。
<br/>　　却原来，林少羽今天在山下值班，也就是打劫。好不容易等到一个人，却是一个十四五岁的小姑娘。据对方所说，她叫小蝶，是苏樱的丫鬟，特地来青牛山找苏樱。
<br/>　　却原来苏樱被掳走的事情已经传开，同样传开的还有蛤蟆寨击败黑石城一千精兵。在传言中，蛤蟆寨的蛤蟆大王，生得虎背熊腰，青面獠牙，每顿饭都要喝一斤人血，十斤人肉。所以小蝶第一句话才会那么问。
<br/>　　程大雷琢磨着这件事，问：“你们觉得，我们要是管苏家索要一笔赎金，应该要点什么？”
<br/>　　“大当家，舍得放她走！”徐神机吃惊道。
<br/>　　“什么话，我一切以山寨的利益为先，岂是能被女色迷惑的人。”程大雷坚定道，这自然不是他的真心话。可实在是，程大雷已经将能用的法子都用了，也没有打动苏樱的心，再留在这里，也是看不见吃不着。
<br/>　　“这个……估计很难要到赎金。”黄三元道：“苏家的情况有些复杂。”
<br/>　　“怎么讲？”曾经黄三元是韩虎踞派到落叶城联系这门亲事的人，自然对苏家的情况也有所了解。
<br/>　　黄三元道：“四海商行的老板名叫苏四海，苏四海没有儿子，倒是有很多女儿，七个还是八个我也记不得。苏四海年纪已经大了，家业只能由女儿继承，而在这些女儿中，苏樱是最有资格的。但问题是，苏樱的生母早早就去世了，现在苏家真正掌权的，是苏四海的小妾。她自然希望自己的女儿继承家业，而不是苏樱。我觉得，她估计都在祈祷苏樱被大当家吃掉。”
<br/>　　程大雷无语，你一开始说得好好的，最后一句话是什么意思。
<br/>　　“放我离开，你们想要的赎金，我有办法给你们送来。”
<br/>　　苏樱似乎听到院子里的谈话，突然走出屋子，站在门口说出这句话。
<br/>　　“你真的想走？”
<br/>　　程大雷问出这句话，便知道自己说了一句废话。黄三元说过，人但凡有点活路，也不会选择落草为寇，更何况苏樱，这位四海商行的千金小姐。
<br/>　　“来人，备纸笔。”
<br/>　　程大雷高喊一声，心中却有些悻悻然，妄自己煞费苦心，苏樱却依旧不动如山。
<br/>　　呵呵，女人！
<br/>　　程大雷让人把纸笔拿来，琢磨着应该向苏樱索要些什么，既然留不住对方的人，那就索性多要点财物，以安慰自己受伤的脆弱心灵。
<br/>　　最后，在山寨所有人的出谋划策下，程大雷定下赎金的数目，包括：粮食五百石、布匹若干、宣纸若干、盐、酒……还有一些书籍。
<br/>　　把清单交给苏樱，程大雷板着一张脸：“你确定，能给我送来！假若敢骗我，可莫忘了，我是会吃人的！”
<br/>　　苏樱端详着清单，最后抬起头道：“你没有狮子大开口，等我回去后，解决家里的事情，便会将东西给你送来，在你给的数字上翻倍。”
<br/>　　程大雷一听，嚯，口气还很大。
<br/>　　在一个风和日丽的下午，程大雷放走了苏樱。落叶城距离青牛山大概有七八十里，一路之上山贼众多，苏樱和小蝶如果独自上路，其实十分危险。所以程大雷还要安排一个人保护他们。
<br/>　　最后程大雷让林少羽和黄三元陪着二人一起上路，林少羽可以负责沿途的安全问题，而安排黄三元，程大雷是为了让他在落叶城采购一些猪崽，家禽。对于山寨来说，种植业要搞，养殖业也要整起来。
<br/>　　看着几人套好马车，程大雷一直送到蛤蟆岭下，冲黄三元和林少羽道：“是你们送她来的，这次还是你们把他送走吧。”
<br/>　　说到这里，程大雷一阵阵止不住的心酸，自己的压寨夫人就要被自己送走了。他交给苏樱十斤辣椒：“听说你们家有酒店，拿着这些试一试，看能不能打开市场。”
<br/>　　苏樱睁大雪亮双眼，轻轻点点头。
<br/>　　越看对方好看，便越觉得不舍，今日一别，何处再寻这样水灵的压寨夫人。可自己已经想尽方法，还是没能打动这个女孩的心。难道，她的心真是石头做的么……
<br/>　　临别之前，程大雷还想最后努力一把，他凝视着苏樱的眼睛，缓缓道：“陌上花开，可缓缓归矣。”
<br/>　　苏樱正要上马车，听到这话竟然回过头来，站在程大雷面前。
<br/>　　“我能不能单独和你说几句话？”
<br/>　　程大雷一听赶忙将黄三元几人赶开，等只有二人单独站在马车前时，他才紧张的问：“你要对我说什么？”
<br/>　　“这些天你对我说了很多好听的话，做了很多想让我开心的事。我不知道是为什么，大概是想让我喜欢你。可是，我能分辨出一个人是真的，还是假的。”苏樱看着程大雷的眼睛，缓缓道：“而你这些天对我做的一切，都是假的。”
<br/>　　程大雷一怔，下意识避开苏樱的眼睛。
<br/>　　“如果你给出的东西都是假的，怎么能期待用假的换来真的。”苏樱语气平缓，却字字千钧：“而其实，你是真心与否对我并不重要，我有没有真心给你也不重要。”
<br/>　　“那么，你想怎样？”程大雷觉得脸皮有些发烫。
<br/>　　“今日一别，不知何日才能相见，我想问一句？”苏樱道：“分别之前，你能不能给我一句真话？”
<br/>　　真话，什么是真话？
<br/>　　程大雷望着面前的苏樱，有那么一刹那的恍惚。红衣若火，肤白胜雪，你说她是林黛玉，她身上却生机勃勃，你若她是王熙凤，她却也有独属这个年纪的天真。
<br/>　　真的？自己看见她第一眼时的感觉是如何呢？
<br/>　　程大雷顿了顿，忽然跨近一步，嘴巴贴在苏樱耳边，缓缓说出一句话。
<br/>　　“一看见你，我就……硬……了。”
<br/>
