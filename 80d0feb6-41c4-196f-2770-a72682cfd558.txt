　　M134机枪六根枪管所发射子弹组成的“死神之鞭”挥过，楼顶上影子部队士兵就像是田地里被收割机割断的麦子，成片的倒下。
<br/>　　在M134面前，人类是渺小的，生命是脆弱的，没有任何生物可以在它的面前存活！
<br/>　　联合行动司令部中，看到通过Bravo小队队员身上携带的战场视频系统传送来的现场画面，房间里的所有人都是爆发出了欢呼声。云熙原本紧握的双手，这个时候也是松了开来。
<br/>　　听到从外面天空中传来的直升机螺旋桨所发出的轰鸣声，罗昊如释重负，对着身边的李然伸出拳头，说道：“总算是赶到了。”
<br/>　　李然用拳头跟罗昊撞了一下，说道：“标准的美式英雄出场方式么？不到最关键的时刻就是不出现，急不死丫的！”
<br/>　　“然后我们的风光就全都被外面那几个美国佬抢完了？”慕容飞有些不爽的说道。
<br/>　　“如果说现在是拍电影，我可以很肯定的告诉你，的确是这样。”李毅峰在慕容飞肩膀上拍了一把，说道：“但现在不是在拍电影，所以我们的风光，那几个美国佬抢不走。”
<br/>　　危机被解除，退守到楼梯中的四个人都是忍不住松了一口气，相互之间开起了玩笑。
<br/>　　这个时候，罗昊背包边小袋中的卫星电话响了起来。
<br/>　　罗昊拿出卫星电话，展开天线接通电话，“蛰龙，你那边情况怎么样？”
<br/>　　“行动顺利。大部分人都被营救出来了。”
<br/>　　树林中，蛰龙看了眼被营救出来的KSK和SAS队员，拿着卫星电话走到一旁，说道：“有几个被用了刑伤势太重，没办法带走，就留在了原地。”
<br/>　　这些坐在树林中休息，身上带着各种伤痕的KSK和SAS队员注意到蛰龙看向自己这边，眼睛中流露出愤怒的神色。尤其是SAS的队员，从他们眼睛中流露出来的眼神，恨不得要把蛰龙焚烧殆尽，可即便在怎么愤怒，他们也不敢发泄，完全就是敢怒不敢言。
<br/>　　能让高傲的列颠国绅士低下他们高昂的头颅，是因为他们对蛰龙和他手下小队队员的作战能力信服，对蛰龙的铁血手段感到恐惧。
<br/>　　在把SAS队员从反政府武装势力营地中营救出来的时候，有两名SAS队员执意要把几个完全失去行动能力，即便救走也未必能活下来，反而还会拖累整个小队的SAS队员带走，蛰龙不答应，只丢给那两个SAS队员一句话，“要么你们跟着我走，要么你们留下来陪你们的队员一起死。”
<br/>　　那两个SAS队员想要将蛰龙一军，执意留下，蛰龙二话不说，带着他小队中的另外两个队员转身就走。离开时，告诉其余的SAS队员，“活命的机会已经给你们了，能不能抓住，就看你们自己。不要试图对我使用你们西方国家那套道德绑架，首先我不在乎道德绑架，其次，那对我根本没用。因为我从一开始就没打算救你们，你们只是运气好，正巧被我遇见而已。”
<br/>　　最后，除了那两个一开始执意要带走重伤队员的SAS队员，剩余的SAS队员全都选择了跟随蛰龙离开。
<br/>　　当然，那两个想要发扬他们绅士风度的SAS队员不是不想要离开，而是遭到了反政府武装势力的包围，无法逃脱。而那些最开始犹豫，随后才跟随撤离的SAS队员也有三个没能顺利逃脱，期间蛰龙根本没有带人回来救援，直到那些SAS队员撤入了丛林，才帮忙打退了追来的反政府武装人员。
<br/>　　楼房中，罗昊听到蛰龙的话，没有对他的做法感到意外，罗昊清楚，蛰龙是那种说得出口就做得到的人。
<br/>　　“你后面打算怎么办？”罗昊对蛰龙问道。
<br/>　　“带着他们继续寻找其他被关押的特种部队。”蛰龙看了眼坐在地上的KSK和SAS队员，说道：“他们的命是我救的，他们现在归我指挥，要么服从我的命令，要么死！”
<br/>　　罗昊知道自己无法说服蛰龙，所以只是叮嘱他小心后，便挂断了卫星电话。
<br/>　　蛰龙跟罗昊之间的通话，李毅峰站在旁边全都听见了，当他听到蛰龙带着他的小队摧毁了反政府武装势力的两个营地，并且成功营救了KSK和SAS被关押人质的时候，心中的震惊不言而喻。
<br/>　　一个只要五个人组成的小队居然同时摧毁了两处有超过百人的反政府武装营地，这怎么可能！？
<br/>　　“蛰龙的小队只有五个人，他是怎么做到的？”李毅峰终究是没有能忍住心中的疑惑，想要从罗昊这儿得到答案，因为他对蛰龙，还有蛰龙那支小队充满了好奇。
<br/>　　“你听说过‘战略行动组’吗？”
<br/>　　这个时候，旁边的李然突然对李毅峰询问道。
<br/>　　李毅峰脸上露出震惊之色，诧异地看向李然说道：“你说，蛰龙的小队是‘战略行动组’！？”
<br/>　　“没错！”李然回答道：“战术行动组和战略行动组，虽然只是一字之差，但是两个行动组之间的差别很大，战略行动组是可以左右一场战争的存在，也是各国特种部队争相想要打造出来‘超级利器’。比如美国的……”
<br/>　　“海豹六队！”李毅峰回答道。
<br/>　　李然点头道：“但是海豹六队并不是所有的行动小组都是战略行动组，在海豹六队中战略行动组也只是占了很小的一部分，而且他们的身份是严格保密，就连海豹六队内部都很少有人知道他们的身份。打造一支战略行动组的难度太大了。”
<br/>　　李毅峰指了指罗昊，说道：“他当年怎么没有进入战略行动组？”
<br/>　　“因为他太全面。”李然解释道：“战略行动组的队员，不单是出色的特战队员，而且还需要有一方面特别出众，万里挑一，比如蛰龙，他精通暗杀和匿踪。还要一点……”
<br/>　　“我做不到蛰龙那样冷酷！”罗昊自己开口说道：“蛰龙是那种关键时刻可以牺牲一切的人，但是我做不到。这就是我跟他最大的差别所在，所以我不可能成为战略行动组的一员。”
