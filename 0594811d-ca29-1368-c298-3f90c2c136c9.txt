　　看到面前这一个人的高的灵石，白萱的眼神都瞪得豆大，满脸震惊。
<br/>　　虽然是在这颜古龙帝的洞府中，她跟随楚风眠看见了不少的宝物，但是看到眼前这灵石的一刻，她还是被震惊住了。
<br/>　　这一块灵石，跟之前楚风眠曾经所得到过的那块王品灵石，大小极为接近。
<br/>　　但是在上面的灵力，却是还要更加精粹许多。
<br/>　　皇品灵石！
<br/>　　一块皇品灵石的价值，足以比拟五块相同大小的王品灵石。
<br/>　　可以说眼前的这一块皇品灵石，比起楚风眠所收获的那块王品灵石价值还要高上五倍。
<br/>　　更何况皇品灵石，本身便是有价无市之物，在外面就算是花费巨大无比的财富，想要得到其中一下块，都需要机缘。
<br/>　　可现在的这一块，足有一人高，眼前的这块皇品灵石，比起楚风眠在颜古龙帝洞府中收取的任何一件宝物，价值都要高的多。
<br/>　　灵石不同于丹药，灵器，是绝对的硬通货。
<br/>　　皇品灵石，对任何层次的武者，都有着大用，只要是楚风眠愿意卖，只怕是会有着无数强者争抢来买，可以换取无数多的奇物。
<br/>　　更何况这皇品灵石，楚风眠自己吞噬了，都有着极大的好处，其中的灵力精粹到极致，根本没有任何的杂质。
<br/>　　一旦吞噬，直接可以转化为自己的力量，是无数诸天奇物中的佼佼者。
<br/>　　楚风眠扫视了周围一圈，打量了其中皇品灵石上的禁制，察觉到这禁制跟外面其他宝藏的禁制没有任何的不同之处后。
<br/>　　楚风眠便是凝聚灵力，右手化为战龙之爪，直接对着那皇品灵石，抓了过去。
<br/>　　“住手！”
<br/>　　一声惊喝，突然响起，只见一道身影，猛然来到了楚风眠的面前，一身灵力暴起，将楚风眠跟那皇品灵石隔绝开来。
<br/>　　“这皇品灵石，被我们地魔门看重，是我们地魔门的了，不想死赶紧滚！”
<br/>　　这是一尊年轻男子，一身红衣，身上蕴含滔天魔意，一眼望去便是地魔门的弟子。
<br/>　　这年轻男子开口的话，嚣张无比，看见这皇品灵石，直接是出手阻止楚风眠，想要来强抢灵石。
<br/>　　这等霸道嚣张，令楚风眠看到，都是眉头一皱。
<br/>　　“白骨公子！”
<br/>　　看到这年轻男子，白萱的脸上都露出了几分忌惮。
<br/>　　“此人的实力极强，实力足以媲美我们北邙学院的种子弟子，据说就连生死境二重的强者，都被他斩杀过！”
<br/>　　“哦？居然是你，白萱？”
<br/>　　听到白萱的话，那白骨公子，也是突然看了过来，一眼认出了白萱的身份。
<br/>　　“没想到在这里遇见了你！”
<br/>　　“上一次本公子让你做我的侍妾，没想到被你逃走了，今日让本公子再次遇见了你，倒是你我之间的缘分了！”
<br/>　　白骨公子看向白萱，大笑着开口道。
<br/>　　“正好，今日先是得到宝物，在得到美人，的确是本公子的大喜之日，乖乖降伏吧，这一次我可不会让你在逃了！”
<br/>　　白骨公子一身的灵力，猛然暴起，这一身灵力，顿时化为了无数白骨，围绕在他的周围。
<br/>　　这一道道白骨，可并非是寻常骸骨，其中的许多，都是坚韧非凡，乃是圣者的骸骨，甚至是生死境强者的骸骨，许多妖神的骸骨。
<br/>　　这些白骨，足足是化为了一尊巨大无比的手掌，猛然压迫下来。
<br/>　　这白骨手掌，只怕是比起地级灵器都要坚韧的多。
<br/>　　“不好，快退！”
<br/>　　白萱面对这白骨手掌，一边催动灵力，一边呼唤，想要先逃。
<br/>　　但是她的话音还未说完，楚风眠的身形，却是已经猛然冲了过去。
<br/>　　“一个地魔门的小魔头罢了，也敢在我的面前放肆？不知道天高地厚！早晚有一日，我会灭了你们这些魔头！今日我就先斩杀了你！看你还如此嚣张！”
<br/>　　楚风眠眼神中露出几分杀意，身上的战龙之力，猛然凝聚。
<br/>　　“战！”
<br/>　　一个“战”字，令楚风眠身上远古战龙血脉完全沸腾起来，一股股澎湃灵力，充斥在他的身上。
<br/>　　他猛然一拳，对着那白骨手掌轰击过去。
<br/>　　“轰！”
<br/>　　这白骨手掌，在楚风眠的一拳之下，全然破碎，其中的无数白骨，尽数断裂。
<br/>　　这一招白骨手掌，被楚风眠一拳打的崩溃。
<br/>　　“什么？居然可以打破本公子的白骨手掌？”
<br/>　　白骨公子脸上露出不可置信的表情。
<br/>　　楚风眠，他是第一眼就看见了，但是白骨公子却是完全没有将楚风眠放在眼中过。
<br/>　　一个神力境四重的武者罢了，在他看来都是如同蝼蚁一般的存在，可现在就是这蝼蚁，居然是打破了他的白骨手掌。
<br/>　　不过很快，白骨公子脸上的震惊却是消散，露出了几分阴冷的神情。
<br/>　　“没想到本公子还是有些看走眼了，还是一位高手，不过小子，敢跟本公子作对，你也是在自取灭亡！今日本公子就先杀了你，在降伏白萱！”
<br/>　　白骨公子一声长啸，他身边的无数白骨，顿时凝聚起来，化为了一根根坚固无比的白骨长矛。
<br/>　　“小子，去死吧！”
<br/>　　“嗖！嗖！”
<br/>　　无数白骨长矛，凝聚出的一刻，便是疯狂的向着楚风眠刺了过来。
<br/>　　这一根根白骨长矛的力量，都丝毫不弱于生死境二重的全力一击，每一根长矛打出，都是令周围震荡一阵。
<br/>　　“小小魔功，也敢放肆？”
<br/>　　楚风眠眼神中闪过一丝光芒，他长啸一声，身形一动，顿时化为了远古战龙的样子。
<br/>　　巨大的身躯猛然爆发出无数金光，化为一道屏障，这一根根白骨长矛打在这金光屏障之下，几乎是瞬间毁灭。
<br/>　　无法撼动其中一分一毫。
<br/>　　“龙族武技？”
<br/>　　白骨公子看向楚风眠，眼神中的杀意渐浓，一脸阴冷到了极致。
<br/>　　“今日就算是来一条真龙，本公子也要屠了他！懂得几分龙族武技就觉得可以抗衡本公子，给我去死！”
<br/>　　白骨公子，连连打出无数武技，顿时漫天白骨，纷纷落下。
<br/>　　这一道道白骨，组成无数兵器，从天而将，向着楚风眠轰击而来。
