　　第48章
<br/>　　程大雷来到这个世界后，心心念念就想着攀科技树，奈何他一个文科生，知识储备实在不够，在文理分科的时候，仅有的一点理科知识也马不停蹄的还给老师了。
<br/>　　况且，想要拿出真正横扫大陆的科技，这个时代也没有这个土壤。就像达芬奇在文艺复兴时期就设计出简单的直升机，可直升机真正出现已经是数百年后的事情。
<br/>　　科技发展固然有某一二天才的横空出世，推波助澜，但在他们推波助澜之前，大势已酝酿许久。
<br/>　　不过，今天的经历突然启发程大雷，这个时代落后的不仅仅是科技，其实也包括饮食。
<br/>　　蒸，煮，腌，烧烤（炙）是仅有的几种烹调方法，炒也很罕见，炸就别提了。这几种烹调方法不约而同的特点就是省油少盐。盐铁是国家私有，以前帝国垄断，如今各城实际控制者垄断，抓到贩卖食盐的人是要砍头的。至于油，在填饱肚子都困难的年代，怎会有人耗费大量粮食去压榨植物油，常用的还是动物油居多。
<br/>　　常见的肉类大多是羊肉，猪肉以及各种禽类，首先大型牲畜宰杀后不易保存，杀头猪三天卖不完，馊了卖给谁去，所以这些肉类比较昂贵些。相比之下，家禽类如鸡鸭鹅就稍便宜些。
<br/>　　落叶城靠海，是有海鲜可以吃的。
<br/>　　程大雷攀科技树够呛，但在饮食方面还是有些把握的，纵然他不是一个专业的厨师，可至少也吃过不少东西。
<br/>　　实在不行，麻辣烫总知道是如何做的吧，火锅要不要尝尝乏……满汉全席，一百零八道大菜，再加上各种街边摊……总也知道个大体的制作方法。
<br/>　　如今，程大雷还没有具体的执行方法，但却已经有了大体的思路，确切这件事是大有可为的。
<br/>　　“对了！”滔滔不绝的店小二忽然压低声音，有种惊天秘密要揭晓的感觉：“本店刚到一种食材，只是数量稀少，不是重要客人不会拿出来，看二位也是老饕，不知是否想尝尝？”
<br/>　　程大雷也来了兴趣：“什么食材？”
<br/>　　“二位……”小二的声音越来越细：“可听过辣椒？”
<br/>　　程大雷刚咬一块羊肉，徐神机刚闷一口酒，听到这话差点都吐出来。刚才走入四海楼时还有些怀疑，现在几乎可以确定，四海楼就是四海商行的生意。
<br/>　　“啊……这倒是闻所未闻？”程大雷瞪了徐神机一眼，阻止他笑出声。
<br/>　　“不是我小看二位，实在这食材以前从未见过，但味道之最实在令人流连忘返。就像我们落叶城的城主，吃过之后，生生索要走半斤，啧啧，本店总共得到的也不过十斤呐。”
<br/>　　程大雷记得苏樱临走前，是让她带走十斤辣椒粉的，看来是不差了。
<br/>　　“只是不知道，这么珍贵的食材你们四海楼是从何处得来？”程大雷问。
<br/>　　“这就多亏我家夫人了，是一位行海的客商从海外带来的，亏得夫人慧眼识珠，才没有错过这样的宝物啊。”小二沾沾自喜道：“我敢打赌，这食材在整个帝国也是独一份，二位要不要尝尝由它烹饪的菜。”
<br/>　　“不啦，不啦，我最近有些上火。”程大雷又掏出十枚铜钱，打发这小二下去。
<br/>　　估计小二也不太明白辣椒和上火有什么关系。
<br/>　　等他走后，程大雷和徐神机对视一眼，都觉得事情有些不对。四海楼确定是四海商行的生意不错，但四海商行的夫人肯定不是苏樱，这辣椒是苏樱带走的，如今却为何张冠李戴。
<br/>　　这里面铁定有问题！
<br/>　　“欧耶，你知道我这次进城，为何别人不带，偏偏要带你嘛？”程大雷压低声音。
<br/>　　徐神机愣了愣，才意识到程大雷在唤自己，他笑着捋了捋胡子：“自然是因为老夫神机妙算，大当……公子要让我出谋划策了。”徐神机及时改口。
<br/>　　“这个……有自信总归是好事吧。”程大雷白他一眼，心道：我还不是怕了你的算无一策属性，这要是我三五天不在山寨，你在家里一口又一口的毒奶，我好不容易攒下点东西，都得被你祸害光。
<br/>　　程大雷定了定神，道“你可要记得，这次咱们上城不是为了吃喝玩乐，而是带着任务来的。一是看看苏家的情况，欠我们的赎金不能不给，二是看看落叶城哪里有卖奴隶的地方，我们买一些人回去。”
<br/>　　“事情宜快不宜迟。”程大雷道：“下午咱们就要分头行动，军师，你觉得自己该去执行那个任务？”
<br/>　　“老朽自然是捡难的来，刺探苏家的任务就交给我了。”徐神机拍着胸脯。
<br/>　　“滚蛋！”程大雷呵斥一声：“你下午去街上转转，看他们交易奴隶的地方在哪里，只要确定下数目，咱们就立刻回山寨取钱。”
<br/>　　“那好吧。”徐神机闷闷不乐，叹口气道：“我也想在城里给灵儿扯两匹花衣。”
<br/>　　程大雷心底明白，徐神机虽然百无一用，但在父亲这个位置上，他还是够格的。即便进城，也惦记着自己女儿，如此父亲对女儿的感情，也是让程大雷心中一暖。
<br/>　　“好啦，账目上准了，事情确定下来后，你就在客栈等我。我这边不知情况怎样，今夜可能就不回来了，你也不必担心。”
<br/>　　二人商量一定，徐神机取了一些钱离开客栈。徐神机刚走不久，程大雷就把那小二又叫了回来。
<br/>　　“客官，您有什么吩咐？”
<br/>　　如今也是恶名在外的程大雷，脸上竟然浮现一抹涩然的绯红，交叉着双手道：“这个，那个……”
<br/>　　小二眼珠一转，忽然大声道：“喔，明白啦，您要找青楼！”
<br/>　　“嘘，噤声。”程大雷狠狠瞪他一眼，然后低声道：“落叶城最好的青楼是那家？”
<br/>　　“那自然要数回燕楼了！”小二一翘大拇哥。
<br/>　　好不容易进次城，就不要废了大好年华，现在酒也喝了，和小二也聊了，不去青楼走一遭，总觉得这江湖少侠一日游缺少那个环节。
<br/>　　打听清楚回燕楼的位置，程大雷意气风发，腰缠数贯趁着夜幕时分往回燕楼去了。
<br/>　　果然是灯火酒绿的好所在，刚走到门口，就有姑娘拉着程大雷往里走。
<br/>　　“公子仪表堂堂，风流倜傥，可把奴一颗心都暖化了。”
<br/>　　程大雷走进回燕楼，眼前莺歌燕舞晃得他眼晕，忽然他见到前方一个背影隐约有些熟悉。快走两步，来到对方身后，一拍对方肩膀。
<br/>　　前面那人回过头来，与程大雷目光相对，二人都是目瞪口呆。
<br/>　　“欧巴！”
<br/>　　“欧耶！”
<br/>　　
<br/>(本章完)
