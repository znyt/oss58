　　说到这，道风停顿了一下，缓缓说道：“众生如棋，他们手中的棋子而已。古往今来，极少有人能从他们手中跳出来。”
<br/>　　叶少阳在心底默默思考了下这句话，道：“你不是已经跳出棋盘了吗？”
<br/>　　“所以我要先躲在这里，在我能左右棋局之前，不让人把我捡回棋盘里。”
<br/>　　叶少阳挠了挠头，道：“你们有文化的就是好，说话都这么深奥。”
<br/>　　道风抬手要敲他，叶少阳眼疾手快，一把抓住道风的手——这本来就是下意识的反应，结果，居然把他的手腕抓住了。
<br/>　　叶少阳懵了一下，立刻就跳起来，“哎呀呀呀，我才想起来，你现在实力没恢复，连我都不是对手是吧？”
<br/>　　道风警惕地看着他，道：“怎样！”
<br/>　　“怎样，嘿嘿……我跟你说，我等这一天等了好久了！”突然扑上去，把道风按倒在床上，双腿骑了上去，道风试图反抗，但两只手腕都被叶少阳抓在手里，使不上力气，一张脸顿时变得红里透黑。“小王八蛋，你作死是吗！”
<br/>　　“哎我说，从来都是你欺负我，你让我欺负一下又咋啦！”叶少阳很享受这种难得的反杀的感觉，道风的表情却是痛苦不堪，不断叫骂着。
<br/>　　“啊，辣眼睛！”芮冷玉摇了摇头。
<br/>　　“他现在打不过我，机会难得啊，小玉你来一起欺负他，他没法反抗的。”
<br/>　　“我平时经常欺负他，而且……”
<br/>　　叶少阳没听到她说什么，按着道风，一个劲地让他叫哥哥。
<br/>　　突然一道青光从道风眉心处射出，本来是正对着叶少阳的脸，关键时刻道风还是稍微低头，让青光射中了叶少阳的腹部。
<br/>　　顿时，一股热流朝着四周蔓延起来，又疼又痒，这感觉……就像是被马蜂叮了一口。
<br/>　　芮冷玉耸了耸肩，“我本想提醒你的，你浪过头了。”
<br/>　　叶少阳捂着肚子在床上翻滚起来，突然被道风按住，骑在他后背上，冷冷道：“你真当我现在实力不济，就没有手段对付你了是不是？”
<br/>　　瓜瓜在一旁哈哈大笑。
<br/>　　芮冷玉捂嘴笑道：“你们两个，一个是人间道神，一个是斩了三尸的绝世强者，你们这样公然亲热真的好吗！”
<br/>　　道风从他身上下来，盘膝坐好，脸色微微潮红。
<br/>　　叶少阳半天才缓过来，也不闹了，坐到道风对面，干咳两声说道：“那个，说正事啊，除了这几个大佬，另外那些强者的实力又怎么样，跟我相比，差了多少？”
<br/>　　道风想了一会说道：“阴司几个大佬，李浩然，镇元子，鸠摩罗什，帝释天，阿修罗王……这一批，实力是在一个层次，他们是活生生的强者，你能遇到的最强大的对手，差不多就是这些人了。”
<br/>　　“我家小九，有后卿，女魃这些？”
<br/>　　芮冷玉听到后卿的名字，眼皮跳了一下。
<br/>　　“后卿若能完全炼化将臣之血，怕是有机会达到顶层，九尾天狐，一旦证道，比东皇太一也不会差多少。他们算是准二流强者，再往下，是女魃这一批二流强者，包括清长风、轩辕山三大神僧，他们之中，女魃稍强，但整体实力相当。”
<br/>　　叶少阳深吸一口气，道：“你这个分法好高端啊，女魃才只是二流强者，那我是第几流？”
<br/>　　“最多就是准三流，或者四流。”
<br/>　　叶少阳嘴角抽搐，没等开口，道风说道：“你别不服，你在人间无敌，但三界之中，比你强的人太多了。南宫影，杨宫梓，他们是三流到准二流。”
<br/>　　叶少阳无奈一笑，喃喃道：“这么说，我还有很长的路要走了。”
<br/>　　“你以为呢？”
<br/>　　芮冷玉插了一句，道：“少阳你别灰心，道风说的这些，都是按照最强者来的了，如果算是第一集团的话，你已经勉强能进得去了，千百年来，人家法师，又有几个能窥探到这个世界。”
<br/>　　叶少阳笑着点点头，问道风：“人类能达到的高度，最多是几流？”
<br/>　　“他们大多数，曾经都是人类。”
<br/>　　“我是说，活着的时候，不是说死后接着修炼的。”
<br/>　　道风想了一下道：“二流吧，古往今来，法师生前能够达到的高度，最多就是二流，连准一流都到不了。”
<br/>　　“好啊，那我就是第一个了。”叶少阳拍着胸脯，自信满满地说道，“我不要做二流，当就当一流！”
<br/>　　道风望着他，并没有嘲讽，而是淡淡一笑，“若你没有这份野心，你没资格做我师弟。”
<br/>　　瓜瓜也兴奋说道：“老大加油，我相信你可以！”
<br/>　　“一起努力吧。我也不是非要争什么第一，我只是觉得，我还有上升空间。”
<br/>　　这就是叶少阳的心声。
<br/>　　道风上次跟他说的那些话，他仍然记得的：要当就当下棋的人，绝不当别人的棋子！
<br/>　　道风刚刚排了强者的等级坐次，叶少阳非但没觉得前途渺茫，反而因为确定了目标，更加有动力了。
<br/>　　“你现在，距离提高一个阶段，只剩下一步之遥，如果能将你体内所有罡气都炼化，你必然能进入三流……”道风为他规划着修炼方向。
<br/>　　叶少阳也将自己修炼中遇到的问题提出来，让道风帮自己解惑。
<br/>　　两人越聊越投入。一边，芮冷玉开始还认真听着，后来发现听得不太懂——这对师兄弟所讨论的，对她目前来说还是有些高深了，假如叶少阳是四流强者，那她压根就是不入流了。
<br/>　　瓜瓜也听得无聊，想起一件事，很得意地对芮冷玉说道：“嫂子，我给你说，我找到了一个娘亲……”当下把陈悦的情况说了一遍。
<br/>　　听者有心，芮冷玉觉得这件事很奇怪，记在心里。
<br/>　　道风如今已经证道，虽然实力还没恢复，但是眼界什么的远远不是当时可比，有关修行方面的很多东西，他都能一眼看透，对叶少阳指点了很多。
<br/>　　听他说完，叶少阳有一种恍然大悟的感觉，对未来的修行之路，也更加有了明确的方向：先将体内隐藏的戾气搜寻出来完全炼化……按照道风的说法，到那时候，自己与黎山老母那群人有望一战。
