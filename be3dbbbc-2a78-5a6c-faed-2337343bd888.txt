　　吼吼吼吼。

<br/>　　“完了！这下扛不住了！”
<br/>　　“要死在这里了吗？”

<br/>　　这个时候林峰忽然道了句，上前一步，手中就是洋洋洒洒一片银光。

<br/>　　那在丧尸头顶上飞奔的老师丧尸顿时首当其冲，直接被爆头。

<br/>　　林峰道。

<br/>　　“这，这办公室还真是神奇啊，什么都有，都快演变成武器库了。”
<br/>　　陈勾愣了愣，不相信也得相信。
<br/>　　林峰一次性甩出去二十多枚飞镖，手里只剩下最后一枚了。
<br/>　　“你们在这呆着，我去前面探探路！”
<br/>　　林峰拿着武士刀朝前探路。
<br/>　　路过一个丧尸时，拔出了对方脑袋中的飞镖，飞镖就那么一些，自然要循环利用。
<br/>　　噗嗤。
<br/>　　林峰回收自己的飞镖，顺便还在找那个掉在第六层的空投。
<br/>　　很快，林峰发现了，他在高一九班的教室内。
<br/>　　林峰摸了过去，整个高一九班，已经没人了，只有几个被咬的稀巴烂的学生尸体。

<br/>　　咔擦。
<br/>　　空头箱内入眼的就是一把尼泊尔军刀和一把精钢长枪。
<br/>　　“两把武器？来的还真是及时。”
<br/>　　林峰把他们全部丢入背包栏里，到时候这两个武器给孙畅和陈勾用，也算是增加了他们的战斗力。
<br/>　　叮！
<br/>　　恭喜宿主获得灵器追风靴。
<br/>　　追风靴：增加速度，宛如追风。
<br/>　　“追风靴？”
<br/>　　林峰立即把鞋子换上。
<br/>　　顿时感觉整个人轻盈了许多。
<br/>　　“好东西，还可以加速度？”
<br/>　　林峰眼睛一亮，速度快了，就可以保命。
<br/>　　叮！
<br/>　　恭喜宿主获得灵器火凤肚兜。
<br/>　　火凤内兜：内涵火凤阵，一旦遇到危险，会自动激发，只能使用三次，三次过后，火凤阵失去效果。

<br/>　　林峰从箱子内拿出了一个明显女人穿的肚兜，上面还画着一条火凤凰，似乎就好像是真的一样，栩栩如生，只是这形状也太性感了吧。

<br/>　　林峰看了眼这火凤内兜。
<br/>　　“妈的，可不可以自己穿？”
<br/>　　不过他试了下，这玩意只能女人穿。

<br/>　　林峰把火凤肚兜放在了物品栏里。
<br/>　　拿起武士刀林峰离开了教室。

<br/>　　“给！这个你们拿着！”
<br/>　　林峰把刚才得到的尼泊尔军刀和精钢长枪给了他们。

<br/>　　陈勾顿时眼睛都亮了。
<br/>　　二话不说就把那精钢长枪拿在手里。
