　　海剑宗的构成，也跟当初楚风眠去的圣龙秘境，无比相近。
<br/>　　有着完整的贡献点制度，只有为宗门做事，获取贡献点，才可以享受宗门之中的种种好处。
<br/>　　几乎不管是哪一方星域的宗门，都是有着类似的制度，这样制度的宗门，才可以长久的传承下去。
<br/>　　而且想要从海剑宗之中，赚取贡献点，也是无比困难的一件事，海十四都已经被赐予了海姓，想要换取参悟一次剑道图腾的机会，都需要积攒很久的贡献点，
<br/>　　海十四的本命，并非是海十四。
<br/>　　在海剑宗之中，核心弟子的前二十位，都会被赐予海姓加上排位的数字。
<br/>　　海十四便是海剑宗之中排位第十四位的内门弟子。
<br/>　　而当初楚风眠碰到的那江展，虽然也是海剑宗的核心弟子，可没有排进前二十，根本没有得到海姓的资格。
<br/>　　“能够见到就好，至于贡献点，只要是完成海剑宗交代的任务就可以了吧。”
<br/>　　楚风眠开口道。
<br/>　　只要是能够近距离接触剑道图腾，楚风眠就有着一次夺取剑道图腾的把握。
<br/>　　这海十四既然是可以有机会去参悟剑道图腾，那楚风眠就可以借助海十四参悟剑道图腾的机会，藏在海十四的身躯之中，混进去。
<br/>　　等到夺去了剑道图腾之后，楚风眠就直接从海剑宗之中逃走。
<br/>　　“主人，看守剑道图腾的，一般都说两位以上的半步天人境长老，不好对付。”
<br/>　　听到楚风眠的话，海十四急忙的提醒道。
<br/>　　海十四现在被楚风眠种下了诸天生死印，一切的想法，都说为楚风眠考虑，自然是将一切的危险，说了出来。
<br/>　　“无妨，我来对付就是。”
<br/>　　楚风眠说到这里，突然想到了一件事，继续开口道。
<br/>　　“对了，海剑宗之中，有着几位天人境强者？”
<br/>　　半步天人境强者，也就只有半步天人境巅峰的人物，现在对楚风眠才有威胁。
<br/>　　不过对楚风眠最大的威胁，当然是天人境强者。
<br/>　　楚风眠也正是因为忌惮海剑宗之中的天人境强者，现在才不敢去硬闯，只能够是智取。
<br/>　　“只有一位，是我们海剑宗一位无比古老的老祖，号称临剑海帝，也是沧海星域最强的几位老祖之一，数万年前就踏入了天人境，羽化登仙的存在。”
<br/>　　提到这临剑海帝的时候，海十四的语气之中都是有些颤抖，显然是对着临剑海帝，无比的惧怕。
<br/>　　“只有一位，那还好说，临剑海帝，看来是以剑入仙的人物了，绝代剑仙。”
<br/>　　按照古老古籍的记载，一旦是踏入天人境，就有资格称之为仙了，以剑入仙，便是剑仙。
<br/>　　不过还有一些更加古老的记载之中，却是说只有玄天境，位列仙侯的强者，才有资格称为仙。
<br/>　　而天人境，只能够算是仙人，不算真正的仙。
<br/>　　当然这个时代之中，天人境的强者都无比罕见了，自然也没有人在意了。
<br/>　　“平日里临剑海帝都是在闭关潜修，不过一旦是主人尝试去夺剑道图腾，必然会惊醒临剑海帝的。”
<br/>　　海十四开口道。
<br/>　　“恩。”
<br/>　　楚风眠点了点头。
<br/>　　这临剑海帝，可不是狂龙帝那种，刚刚踏入天人境的强者，而是已经踏入了天人境数万年的人物了。
<br/>　　他的实力只怕是比起狂龙帝还要强横的多，楚风眠也必须要好好计划才行。
<br/>　　“找个计划，先潜入海剑宗之中再说。”
<br/>　　楚风眠也要了解了海剑宗之中，在准备动手。
<br/>　　他将海十四，收入到了境王布袋之中，旋即目光看向了海公主跟奇阳师。
<br/>　　奇阳师身上的气息，已经是恢复了不少了，毕竟他之前的伤势如此严重，也是因为身上的寒毒。
<br/>　　现在寒毒一旦是被楚风眠化解，剩下的那些伤势，对奇阳师这样一位九劫古帝来说，也不算什么，很快就可以恢复了。
<br/>　　“多谢楚公子救命之恩。”
<br/>　　奇阳师看到楚风眠的目光看了过来，急忙的起身，恭敬的行礼道。
<br/>　　这一次也是多亏了楚风眠，不然的话，他早晚都会因寒毒陨落。
<br/>　　之前在他身上的寒毒，已经是遍布五脏六腑了，几乎算是无药可救，只有楚风眠的吞天秘术，才可以救了他一命。
<br/>　　“举手之劳罢了。”
<br/>　　楚风眠说了一句，便是看向了海公主，仔细的打量了一眼，好奇的说道。
<br/>　　“玄鲸血脉？我倒是都没有看出来。”
<br/>　　刚刚从海十四的话中，楚风眠才知道了，海公主居然是玄鲸血脉。
<br/>　　玄鲸，乃是先天神兽之一，海族之祖。
<br/>　　虽然许多说法之中，海族之祖，乃是海神，但是海神只是海族文明的开创者。
<br/>　　真正创造出万千海族的，便是玄鲸。
<br/>　　天下所有海族之祖，都是玄鲸。
<br/>　　这玄鲸血脉，也是先天神兽血脉的一支，不亚于始祖天龙学血脉，吞天兽血脉。
<br/>　　楚风眠跟海公主也算是见识很久了，不过一直以来都是没有察觉到海公主身上的玄鲸血脉。
<br/>　　本来在楚风眠看起来，海公主的血脉只是普通的海族血脉，可听到了海十四的话，楚风眠仔细探察一番，才看的出来，这海公主的血脉之中，还隐藏着一种可怕的力量。
<br/>　　正是玄鲸之力。
<br/>　　不过楚风眠看得出，这海公主身上的玄鲸血脉，还没有觉醒。
<br/>　　曾经楚风眠在妖古大陆之时，曾经杀过的那火家少主，也是拥有腾蛇血脉，却没有觉醒。
<br/>　　先天神兽血脉，若是没有觉醒，看起来就是寻常血脉，只有特殊手段，才可以察觉到。
<br/>　　海公主身上的玄鲸血脉，虽然是没有觉醒，可是消息却是散布了出去，所以才会受到追杀，被破离开了沧海星域。
<br/>　　而且哪怕是离开了沧海星域百年时间，这追杀，也一直没有停过。
<br/>　　“你。”
<br/>　　听到楚风眠说出玄鲸血脉这几个字，那奇阳师的脸色猛然变化，一脸警惕的看着楚风眠。
<br/>　　他虽然知道楚风眠跟海公主关系不错。
<br/>　　可在先天神兽血脉的面前，别说是朋友，哪怕是亲兄弟，父子，都可能反目成仇。
