夏妍妍看向前方的夏夕绾，她苍白的脸上迅速露出了几分凄楚，“绾绾，这些都是我的错，我现在向你道歉，你能不能原谅我这一次，有什么事情你就冲着我来，不要迁怒我身边的家人朋友，更不要伤害他们好不好？”
<br/>
<br/>        夏夕绾真的没有想到首度露面的夏妍妍竟然会是这副苍白可怜的样子，“夏妍妍，所以你现在是在玩苦肉计，将自己装成一个无辜的受害者？”

<br/>        “现在全网嘲我最多情，那全网也知道我事先并不知道你替嫁给了陆总，成为了陆太太，可是…可是我不知情，你是知情的啊。”
<br/>
<br/>        “绾绾你为什么不事先就告诉我你是陆太太，好让我打消念头，爸给我们办了生日party，一场天灯闹得人尽皆知，你放任我满腔的少女心思，无情站在角落里嘲笑我这个痴情的傻瓜，然后…然后你才给了我致命一击，全城示爱让我陷入尴尬狼狈的境地里，你就这么…这么恨我么？”

<br/>        夏妍妍这番话说的声泪俱下，数度哽咽，她的脸已经被泪水模糊，就连肩膀都在颤抖，这可怜的模样当即博得了所有的同情。

<br/>        本来都在看戏的吃瓜群众都窃窃私语，

<br/>        其实妍妍挺可怜的。

<br/>        对啊，陆总谁不爱啊，夏夕绾明知道妍妍喜欢陆总却不说明真相，而是在生日party上上演了一出大戏，现在想来这手段啧啧…真是太狠了！
