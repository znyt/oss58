　　好好的日子竟然硬生生被自己过成一个魔方，钦陵苦涩的笑着，感叹着命运的作弄。
<br/>　　亲卫不懂是因为无知，可谁又能说无知不是福呢？
<br/>　　人生烦恼识字始，这话似乎说的没有什么错，如果自己也能像那个亲卫一样，大字不识一筐的话，应该就没有这么多的烦恼了吧！
<br/>　　什么理想，什么价值，这一切其实都是扯蛋，浑浑噩噩的活着未必不是一件好事。
<br/>　　想着当初自己从高原离开，就是为了向老头子证明自己，证明自己不是一个任由命运摆布的魔方。
<br/>　　可最后还是无力改变什么，除了把自己的日子过的一团糟之外，竟然什么都没有改变。
<br/>　　好不容易想通了一切，琢磨着回到高原再重活一次，可该死的命运，再次把他的人生弄的一团糟，跑到这个鸟不拉屎的地方，再次体现什么人生价值。
<br/>　　这么一看，他自己与手中的魔方似乎并没有什么区别，命运就是那个扭‘魔方’的人。
<br/>　　……
<br/>　　这人啊，一闲下来就会胡思乱想，所以为了不让那些大头兵乱想，每天便要给他们安排无数的例行训练，直到把这帮家伙累成狗，然后就可以什么都不想了。
<br/>　　可大头兵能这样，钦陵却不行，他已经是这里的最高长官，没办法让自己什么都不想。
<br/>　　还是就这样继续下去吧，早晚有一天会有结果的，每天钦陵都在这样安慰自己，然后第二天重复前一天的感觉。
<br/>　　五月的天气已经入夏，黑水河虽然已经是钦陵所知道的极北之地，可是夏天却并不凉快，除了早晚温差大之外，似乎比大唐的南方也差不了多少。
<br/>　　所以钦陵每天就多了一个习惯，那就是去河边洗澡。
<br/>　　这一天，钦陵照例带着三、两个亲卫来到了河边，脱的全身上下只剩一条裤衩之后，一个猛子扎进水里，从头到脚的清爽感让他的精神为之一振。
<br/>　　不过，当他从水里浮上来的时候，却发现岸上的亲卫正不断的向他招手，脸上的神情似乎很紧张的样子，而且那些亲卫手中似乎还拿着什么东西。
<br/>　　心中不解之下，钦陵返身游向岸边，等游了一会儿才发现，那亲卫手中拿的竟然是一套衣服，而那衣服的款式明显不是军中之物。
<br/>　　河中还有其他人！是来暗杀自己的？
<br/>　　陡然间，钦陵意识到了问题的严重性。
<br/>　　但很快他又放松了下来，在河中向岸上摆了摆手，示意岸是亲卫不用紧张，同时让他们散开。
<br/>　　这里是他每天都要来的地方，如果说靺鞨人想要刺杀他的确说的过去。
<br/>　　可是有什么刺客在刺杀之前把自己的衣服都脱了呢？而且还放在那么显明的地方，这不是明摆着告诉别人这里有人，不要过来么。
<br/>　　意识到这一点之后，钦陵的精神放松了下来，但好奇心却不可抑制的浮起。
<br/>　　毕竟日复一日的等待实在有些无聊，给自己找些事做也是不错的。
<br/>　　于是钦陵打出手势之后便与几个亲卫配合着，向上游一处水草茂盛之地游了过去。
<br/>　　毕竟这里河面虽然宽，但能藏人的地方却不多，除了那一片芦苇之外，其它地方完全就是一览无余，根本不可能有人能够藏的住。
<br/>　　当然，如果水里的人能够憋气半个时辰，一直在水底游到众人看不到的地方上岸，这种可能也是有的，但那人一定不会是靺鞨人，甚至不会是地球人。
<br/>　　……
<br/>　　时间一点一点过去，钦陵和几个护卫离那丛芦苇越来越近。
<br/>　　就在他们马上就要接近到芦苇丛附近的时候，一个声音突然从从芦苇丛里传了出来：“不要过来，再过来我，我就自杀！”
<br/>　　女的？自己一群大男人，大张旗鼓的折腾了半天，就为了抓一女的？而且还是……还是一个正在洗澡的女的？这特么不是扯王八犊子么！不知怎么，钦陵的心里突然冒出这样一句话。
<br/>　　于是钦陵连忙对岸上几个同样在发呆的亲卫打了几个手势，示意他们回避，而自己也转身向其它方向游去，口中尴尬的说道：“姑娘不要误会，我们不是坏人，马上就会离开，你的衣物还放在原来的地方！”
<br/>　　什么？你说钦陵一个贵族，什么大场面没见过，面对一个洗澡的女人不应该如此紧张？
<br/>　　呵呵，是，如果他自己穿着衣服他当然不会紧张，问题是丫现在也光着呢！
<br/>　　这特么如果继续搜下去岂不是要‘坦诚相见’！
<br/>　　钦陵虽然见过世面，可也是‘见’过，而不是‘露’过，这样的情况他自己也从来没有遇到过，紧张一些自然在所难免。
<br/>　　……
<br/>　　一刻钟之后，钦陵等人终于见到了这次‘事故’的肇事者，一个年龄大概在二十左右的靺鞨族姑娘。
<br/>　　“你是什么人？为什么会出现在军事管制区？难道你不知道这里是大唐军营的控制范围么？”钦陵经过这么长时间早已经穿好了衣服，恢复了冷静，看着那刚刚从水里出来换好衣服的女子沉声问道。
<br/>　　不想那女子竟然硬气的紧，狠狠剜了钦陵一眼：“要杀就杀，那么多废话干什么？”
<br/>　　“呃……”这下反倒让钦陵不知道怎么办才好了，想了半天才说道：“姑娘不要误会，我并没有为难你的意思，只是好奇这里从来都没有外人来过，为何你……”
<br/>　　“活不下去了呗，还能怎么样！我们部落常年生活在这里，可是你们一来立刻占据了这里，你让我们如何生存下去。”那女子倒也泼辣，一张小嘴跟吃了炫迈似的，根本停不下来。
<br/>　　活不下去了么？钦陵皱了皱眉，再次上下打量了那女子一眼，皮肤红润有光泽，好像并不像是什么活不下去的样子。
<br/>　　不过那女子却误会了钦陵，以为他是见自己孤身一人起了什么歹心，立刻眉毛一竖冷声喝道：“色狼，你看什么！”
<br/>　　“如果你能解释为什么你会来这里，我可以考虑放过你，否则我会让我的士兵把军营周围百里方圆全部划为禁区。”
<br/>(本章完)
