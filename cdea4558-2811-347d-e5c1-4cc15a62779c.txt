　　如今天道不全，天缘不现，四象星宫还未现世，四象神兽也还未成为星神兽。元始不怕它不心动。果然，
<br/>　　“好，我交换。”朱雀看完这道信息后，兴奋道。终于可以突破了，不容易了。至于这段信息的真假，它还是能够看得出的。因为冥冥之中他总有一种感应，与太古星辰有关，想必就是这信息之中提到的朱雀星域了。
<br/>　　朱雀神兽也不再废话，酝酿许久，张口一吐，便飞出一个闪烁着赤金色的小朱雀。
<br/>　　这正是朱雀本源精血！
<br/>　　这道精血闪烁着耀眼光芒，出现的瞬间，便有一道惊人的威压散布四周，血液之上散发着炙热的朱雀神火，其形似小了无数倍的朱雀真身。
<br/>　　“唧唧唧唧！”
<br/>　　在朱雀精血出在虚空之中的同时，神兽朱雀一瞬间就变得萎靡不堪，悲声鸣叫，好似受到极大伤害一般。
<br/>　　元始看见这团精血，大手向那团精血一抓，精血便消失不见，却是被元始收了起来。
<br/>　　“既然交易已经达成，那我便不再打扰。况且此次你受伤颇重，还是好好养伤为好。”元始对着神色萎靡，身体虚弱不堪的朱雀淡言道。
<br/>　　“好！”朱雀身形一闪，便消失不见，却是回到了岩浆下的朱雀神宫之中，疗养伤势去了，此番耗费精血，它实力大降，却是不准备就此前往朱雀星域。
<br/>　　目的已经达成，元始亦不再多留，身形一动，便已不见，等再次出现，却已到焚炎谷之上。他心神一动，面向西方。运起腾云之术，向着极西之地飞去。
<br/>　　……
<br/>　　接下来，元始分别到了极西之地、极北之地以及极东之地，找到了除却朱雀之外的其余三大神兽白虎、玄武、青龙，向他们交易神兽本源精血。
<br/>　　其中白虎性属金，掌杀伐之力，性格桀骜不训，不肯交易，但这对于元始来说，并不算什么，给你好处，你不交易，那就别怪我不客气。元始只好亲自动手给他上了一课，强行取走他一团精血，使之受损严重，比之其余三大神兽还要严重。
<br/>　　至于其余两大神兽，玄武镇北冥海眼，性属水，防御极强，元始也颇费一番功夫，才达成交易的。
<br/>　　倒是青龙神兽好说话，在元始说明目的，听到交易内容之后，便毫不犹豫地就答应了，这倒出乎元始的意料，他还以为也要耗费一番功夫呢？不过，这样也好，省得元始麻烦。
<br/>　　过程虽有曲折，但最终目的还是达到了，元始准备找个地方闭关一下，是时候让《玄黄不灭》突破了。
<br/>　　“咦！这个山谷不错！”这一日，元始走到一处山谷，看见山谷四面环山，山峦俊秀挺拔，百草丰茂，树木丛生，仙气缥缈，灵气浓郁，小溪潺潺，仙鹤飞舞，玄猿鸣啼……
<br/>　　这简直就是一座仙家福地啊。
<br/>　　元始对此十分满意，决定就在此地突破。
<br/>　　“嗯？此地竟然有先天阵法笼罩，莫不是其中有先天灵宝？”
<br/>　　元始刚入山谷，就发现此山谷之中有一座先天阵法笼罩其中，让人不得寸进。既然有阵法笼罩，那其中就必定有蹊跷。元始猜测这其中可能会有某种宝物。
<br/>　　这种猜测让元始心动不已，要知道元始到如今依旧是穷光蛋一个，拿得出手的先天灵宝都没有一件，全身上下只有一座以前遗留的因果莲台、一件开天之初抢来的后天攻击至宝鸿蒙量天尺，以及一件他灭了黑暗魔神后得到的黑暗天幕。
<br/>　　这三样宝物，不是他自己炼制的，就是他打劫抢来的，还没有一件是他自己捡到的。这让元始不止一次地怀疑自己不是盘古元神所化，并没有得到盘古遗留的气运福泽。要不然怎么不会出现出门遇灵宝，随地捡灵根这种现象呢？
<br/>　　他也不想想，他在不周之巅悟道，得到两大先天灵根，一混沌灵根，那不是气运又是什么？
<br/>　　好在元始并没有太过纠结这些，他自己对于灵宝并不是太过渴求，元始相信，自身实力才是最重要的，至于宝物，够用就行，没必要弄很多在身上。
<br/>　　如今元始遇到这先天大阵，猜测里面会有宝物，他决定一探究竟，就算得到宝物自己不用，日后不是还可以送给弟子门人等人，或者与别人交换对自己有用的宝物啊。
<br/>　　本以为要花费百年时间来破阵，没想到破阵时仅仅花费了不到十年的时间，这不是这阵法太弱，而是元始对道的领悟更深了，万法唯道，总有一些相同之处，阵法也一样。
<br/>　　元始之前就学会过笼罩混沌树的混沌阵法，那可是一个能组成一小型混沌世界的阵法，级别也特别高，再经过混沌之中百万年时间的闭关悟道，现在对于破除这等阵法，已经不需要多长时间了。
<br/>　　元始并没有完全破除，阵法依旧笼罩其外，但元始人却已身在阵中，他想救助此阵法的保护，在此阵中闭关突破，没有什么地方比这更好的了。
<br/>　　进入阵中，入眼所见，没有宝光纵横的先天灵宝，亦没有灵气四溢的先天灵根，有的只是一座美轮美奂的洞府宫殿。
<br/>　　洞府是一座华丽高贵的宫殿，这座宫殿由先天蓝玉筑成，极为奢侈。
<br/>　　玉分先天和后天，后者是后天慢慢形成的，历经千万年，前者是有先天元气进入玉坯之中，衍变出灵性的宝玉。这先天蓝玉，更是灵玉之中的极品，其中不仅蕴含这丰富的灵气，而且修道者若长时间在其中修炼，还能达到净心凝气，减少心魔的作用，它甚至可以堪比一些先天灵宝了。而此地却用先天蓝玉炼成宫殿，不得不说，其主人还真是大方啊。
<br/>　　不过也是，这先天蓝玉对于他这等人物已经没有太大用处了，食之无味，弃之可惜。不拿来做宫殿，还用来干什么？要是元始，他也这么干。不得不说，他们两人在某些地方还真是一样的人。
<br/>　　宫殿之上，蓝光绽放，氤氲生烟，高贵而不失纯美，华丽而不失清洁。让人一看就不能忘怀。
<br/>　　宫殿之外，禁制遍布，但宫殿之上，却刻有“昆吾”两字。
<br/>　　“嗯？原来是早已被人占了，本座怎么说没看到灵宝灵根呢？不过，若是此处有人，那么其主人呢，本座破阵之时也没有出现，现如今进入阵中，也没有出现。莫不是其主人外出有事耽搁了？还是已经死了？要是有事耽搁，那就麻烦了，要是死了，那就便宜本座了。
<br/>　　不过，不管怎样，本座既然已经进来了，就说明此地与我有缘，那就断然没有空手而出的道理。”
<br/>　　说着，元始再次抬头看向宫殿，却见宫殿之上，“昆吾”两字剑气纵横，凌厉霸道，威慑众生。
<br/>　　“嗯？昆吾……昆吾。”元始突然好似想起了什么东西，自言自语道：“此地莫不是昆吾老祖的洞府？”
<br/>　　想了想，也没有什么线索，“罢了，不管它是谁的洞府，从今以后，它，便是本座的洞府了。正好本座缺一行宫，此宫殿正好。谁要是敢来，本座接着便是。”
<br/>　　说完，元始大手一挥，便将“昆吾”两字抹去，道光闪烁之间，却见那两字已经消失不见，取而代之的是另外三字。
<br/>　　“玉虚宫！”
