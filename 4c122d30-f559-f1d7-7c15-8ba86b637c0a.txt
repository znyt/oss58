　　女魃见后卿擒住了芮冷玉，脸色微变，追赶过去，后卿回到安全地段之后，也没有再走，女魃追上他，站在他身边，冲叶少阳喊道：“叶少阳，想要你的心上人吗，用东皇钟来换！不然我顷刻就杀了她！”
<br/>　　说完，用一只手去按芮冷玉的天灵盖，结果意外发生了：后卿空着的一只手，抓住了她的手，冷冷说道：“你不许伤她。”
<br/>　　女魃神情大变，也顾不得当着若干手下的面，大吼道：“后卿，你知道自己在做什么，为了一个人间女子，你真要致尸族的前途于不顾吗！”
<br/>　　后卿低声说道：“你闹什么，有她在手中，就算叶少阳得到东皇钟，也得听我们的。”
<br/>　　女魃怔了一下，道：“你说的也是没错，不过，你真的不是为了保护她才这么说？”
<br/>　　“一举两得的事，怎么了？”
<br/>　　女魃哼了一声道：“不过我还是想两个计划一起进行！”
<br/>　　当下吩咐手下继续去抢夺东皇钟，她的想法很简单，如果能抢到东皇钟就抢，如果抢不到，再用芮冷玉去威胁叶少阳不迟。
<br/>　　“东皇钟是属于空界的，一定不能让尸族得到！拦住他们！”
<br/>　　摇光仙子站在摇光峰的顶端，对着下方发号施令，立刻有空界的强者前去阻挡那些僵尸，双方展开了一场血战。但是僵尸却是源源不断地从两座山的缺口中间涌了出来，黑压压一群，在人数上立刻占据了优势。
<br/>　　女魃冷笑道：“我既然敢发动袭击，自然早有准备……”
<br/>　　事实上，她的尸族大军，第一波早就已经到了，但并没有从山谷进去强攻，而是趁着几方势力大战的工夫，而是迂回前进，埋伏在两座山的中间，专门等着关键时刻一起推进。
<br/>　　尸族大军庞大的数量一旦进入山谷，立刻将空间填满，空间诸强者、阴司四神率领的阴兵，还有捉鬼联盟，一起退到了东皇钟的周围，奋力抵抗。
<br/>　　叶少阳对这一切，视而不见，他与后卿对望着。后卿微微一笑，倒是不带什么敌意，只有得意。
<br/>　　芮冷玉，现在就在他的怀里。
<br/>　　“阿呆！我相信你！”叶少阳没头没脑地说了这么一句，后卿却是听懂了，目光闪动了一下，想起了自己作为“阿呆”的那段经历，虽然他救过叶少阳一次，甚至不惜影响了女魃的计划，也明着跟叶少阳说，已经还了他的救命之恩，不过情分毕竟还是有的。
<br/>　　更不用说，他对于芮冷玉的恩情。
<br/>　　因此叶少阳跟他打了感情牌。虽然他对这个结果十分嫉妒和不满意，但还是相信后卿绝对不会伤害芮冷玉。在说完这句话之后，他就不再看芮冷玉，而是把目光收回到小九的身上。芮冷玉目下毕竟没有危险，而小九……魂体却已经淡薄得近乎快透明了。
<br/>　　她的魂力，尽数溶解进了身下的东皇钟里。
<br/>　　叶少阳感到有一股强烈的吸引，从东皇钟里涌出来，不断往自己身体里钻，逐渐的，他体内有了一种奇妙的感觉——应该是错觉，周围的一切都不见了，自己仿佛置身在一片海洋之中，东皇钟的半径四周，仿佛升腾着温暖浪花，热气一层层地烘烤着自己。
<br/>　　渐渐的，他不是看到，而是在意识中感觉到了东皇钟的存在，仿佛与自身逐渐融合在一起。
<br/>　　这是祭炼吗？
<br/>　　叶少阳悚然心惊，自己为什么可以祭炼东皇钟？
<br/>　　“少阳，东皇钟已经被我的魂力解开了封印，我的魂魄，也会融入其中……你我毕竟魂晶相连，你既然是我的主人，那也是东皇钟的主人，未来路很长，我不能再陪着你，只能……把东皇钟送给你……”
<br/>　　“不，不不不不！我不要什么东皇钟，小九，我只要你啊！”
<br/>　　叶少阳在神识里声嘶力竭地喊叫起来，声泪俱下，但是小九的影像却是越来越淡薄了。
<br/>　　“少阳，我没后悔……我喜欢你。”小九的声音，就算是在神识里，也微弱得像蚊语一般。
<br/>　　“小九，其实，我也喜欢你……那次在西域，我抱着你的时候，我也很开心。”叶少阳生怕没有机会说了，忍不住飞快地说道，“当时，我动心了。如果还有机会，我很想很想跟你在一起！”
<br/>　　“谢谢。”小九近乎透明的魂体微微一笑，“虽然是安慰我，但我……信了。”
<br/>　　“狐姐姐，我送你一程！”
<br/>　　摇光仙子拿出长笛，吹出了一曲低回婉转的曲子：王昭君在去西域路上弹奏的那首《出塞》。虽然是明争暗斗的对手，虽然她一直看不上妖族，不过九尾天狐的所作所为，打动了摇光仙子，也打动了黎山老母等人，所有摇光峰上的伤者，一起站了起来，用自己的方式，对小九行了重重的一礼。为她送别。
<br/>　　笛声飘扬，与山谷中厮杀的声音交织在一起，虽然有绝大的反差，却更是凸显出了一种悲壮的情怀。
<br/>　　“你曾经跟我说过，相聚皆是缘，缘尽莫强求。少阳，缘尽了……”
<br/>　　小九最后一丝魂力也散去了，汇入了下方的东皇钟里。
<br/>　　“小九！”叶少阳发出撕心裂肺的哭声。
<br/>　　一代九尾天狐，天选之妖，就此香消玉损。
<br/>　　东皇钟上的四道符文，完全明亮了起来，绕着东皇钟一阵飞舞。
<br/>　　东皇钟的封印被打开了，原本漆黑的外层，发出了暗金色的光辉，灵力直冲云霄。
<br/>　　这件上古传说中的法器、东皇太一的专属，总算重见天日了。
<br/>　　不过，这一切跟叶少阳都没有关系了，他趴在东皇钟顶端，看着自己的眼泪一滴滴落在上面，然后流下去，心中充满了无尽的悲伤。
<br/>　　在他心中，世界已经不在了，因为小九已经死了，与东皇钟融和成了一体……
<br/>　　“少阳，少阳！事情还没结束，振作起来！”
<br/>　　一个急切的声音，呼唤着叶少阳，强行拍打着叶少阳的肩膀，将他拉回到现实。叶少阳抬头看去，是广宗天师。
<br/>　　“关我什么事，小九都死了。”叶少阳心灰意冷地喃喃说道。如果不是芮冷玉还在后卿手上，如果不是小九拼着牺牲自己才救下自己，他真想一头撞死在东皇钟上，为小九殉情。
