　　清微的语气非常凝重，此事不管真假，若是传出去，某些有心人绝对会利用此事算计蜀山。
<br/>　　见着清微极其难看的脸色，慕清霄觉得后者没必要在这种事情上撒谎，而且他说的也非常有道理。
<br/>　　若是蜀山真的有三皇神器，怎么可能是区区仙门之首，哪怕蜀山想一统六界恐怕也不成问题吧？
<br/>　　但是，清微在这件事情上是真的一无所知，哪怕他是这一代的蜀山掌门。
<br/>　　当然，他不知道，却不代表蜀山就真的没有。
<br/>　　天璇宫中的青色丹鼎，绝对是真武殿中《上古神器志》上记载的神农鼎。
<br/>　　信息中记载，神农鼎，伏羲剑与女娲血玉都被藏在摇光宫的三皇台上，但神农鼎却出现在了天璇宫中。
<br/>　　此事，恐怕与太清真人脱不了干系。
<br/>　　但是，太清真人飞升后，偏偏没有带走神农鼎，若说看不上，那绝对是不可能的。
<br/>　　太清真人当初能够镇压六界，但却没有一家独大，看来真武殿中记载的信息也不完全是真实的，可能信息就被改动过。
<br/>　　毕竟，历史是由胜利者书写的。
<br/>　　也就是说，太清真人并没有将蜀山有三皇神器的事情说出来，以至于蜀山根本不知道三皇神器的存在。
<br/>　　“掌门可知道，三皇台？”
<br/>　　话语刚落，清微一脸迷茫，望着慕清霄道：“小子，你究竟想说什么？三皇可是传说中的存在，不是我们能随口言论的。”
<br/>　　“我只是想证实一些猜想。”
<br/>　　慕清霄目光闪烁了几下，若是六界书中呈现的画面不错，或许这里只是整个宇宙中衍化而出的一方小世界。
<br/>　　他的心中有许多疑惑，并且所有的疑惑都会让他联想到蜀山的祖师太清真人。
<br/>　　“此事在这里说说也罢，切勿不可乱传，多事之秋，蜀山不能在出任何的状况。”
<br/>　　清微语重心长的说道，出了邪剑仙这件事情，蜀山已经是无力在处理其他的事情，他身为掌门，不得不为蜀山考虑。
<br/>　　点了点头，慕清霄盘坐在蒲团上，道：“你是长辈，在修行方面比较有话语权，有些问题需要掌门给在下解惑。”
<br/>　　“长辈？你小子可没一副把老夫当作长辈的样子。”
<br/>　　清微没好气的瞥了他一眼，但拿他也没办法，无奈的摇了摇头，道：“有什么问题，你尽管问吧。”
<br/>　　“我在沉睡的期间，境界直接突破到了九重巅峰，能够清晰的感觉到自己卡在了瓶颈上，体内灵气似乎已经饱和，无法在做突破。”
<br/>　　醒来后，慕清霄明显感觉自己的境界已经到了大乘九重巅峰，在迈出一步便是渡劫期。
<br/>　　按照他在前期打下的扎实基础，想要突破应该不难，可是瓶颈给他的感觉就是牢不可破，而且体内灵气已经饱和。
<br/>　　“换做别的修士，想要突破一个境界需要几十年甚至更久，你小子只是睡了一觉就突破三个境界，居然还不知足？”
<br/>　　望着慕清霄一脸困惑的模样，清微气的吹胡子瞪眼，郁闷至极。
<br/>　　“你现在的情况我了解，但你可知道，渡劫乃是凡人境界的最后一个境界，挡在你前面的壁障可不是想破就破的，需要机缘，倘若悟性不够，或许十年，或许百年内都不一定能突破。”
<br/>　　挥了挥拂尘，清微再度恢复和煦的模样。
<br/>　　“小子，道不可说，老夫能给你说唯有经验，况且每个人的道路各不相同，说多了对你来说并非好事，或许等你何时找到了自己的路，瓶颈就消失了。”
<br/>　　闻言，慕清霄陷入沉思。
<br/>　　“老夫建议你下山游历几年，此事不要多想，机缘到了自会水到渠成。”
<br/>　　“下山游历，妙丹堂由谁来打理？”
<br/>　　清微捋了捋胡子，道：“妙丹堂需要打理的事务不多，你不再的时候，就由草谷来打理，那丫头已经是二品炼丹师，那丫头凝聚出了丹火，炼制一些低级丹药是不成问题的。”
<br/>　　清微并不懂炼丹术，自然不可能区分出丹火与兽火，想当初还将神火误认为了丹火。
<br/>　　草谷是他捡回来的，当初将草谷送到慕清霄身边的时候，意思就是让后者将小丫头培养成炼丹师。
<br/>　　只是，让清微没想到的是，慕清霄会如此真诚的培养草谷，半年就培养出一名二品炼丹师，着实将他吓了一跳。
<br/>　　慕清霄微微点头，道：“掌门，天璇宫中有一座丹鼎，我手中正好没有合适的，不知……”
<br/>　　“那座丹鼎自老夫进入蜀山就一直存在，已经有千年了吧。”
<br/>　　说罢，清微眼中闪过一精芒，回忆起当年往事。
<br/>　　“据说，那座鼎是祖师太清真人传下来的，只是丹道衰落，在也能人能用得上丹鼎，也就一直将它搁置在了天璇宫中。”
<br/>　　清微自然知道慕清霄对丹鼎起了心思，毕竟是炼丹师。
<br/>　　“你若是需要，便拿去吧，放在蜀山，只会令明珠蒙尘罢了，太清祖师在上，想必也会同意的。”
<br/>　　“多谢掌门。”
<br/>　　慕清霄微微行礼，心中满是喜悦。
<br/>　　若是清微老头子知道那座鼎是传说中的神农鼎，不知道会不会答应的如此爽快。
<br/>　　“无须谢老夫，自你进蜀山的第一天起，老夫就知道你的心不在蜀山，是老夫谢你才对，肯给蜀山留下丹道的火种。”
<br/>　　慕清霄微微一愣，果然是人老成精，没想到被看出来了，他来蜀山本就是抱有目的的，根本不可能待在这里。
<br/>　　“你也无需惊讶，你让草谷和常胤打听灵溪谷的事情，岂能瞒得了老夫，虽然不知道你找灵溪谷有何用意，向东两百里的地方就是灵溪谷，去吧……”
<br/>　　说罢，清微闭上双眼，盘坐在蒲团上，如同一个普通老者。
<br/>　　慕清霄微微行礼，转身径直离去，清微对他也算是有点小恩情，自然受的起他一拜。
<br/>　　待慕清霄离开，清微睁开双眼，掐指一算，两条白眉皱在一起。
<br/>　　“未来依旧是看不清虚实，如果老夫记得不错，水灵珠似乎就在灵溪谷中的女子身上，难道赶在长卿前面取走灵珠的，真是这小子不成？”
