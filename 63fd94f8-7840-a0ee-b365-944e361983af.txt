　　吴巧哪里敢开口，低头不说话。
<br/>　　三足金蟾只当她是害羞，哈哈大笑，上去解吴巧的衣服，很快就脱去了外衣。
<br/>　　吴巧挣扎了几下，往床上缩。三足金蟾哪里给她这个机会，直接扑了上去，压在吴巧身上，按住她的肩膀，朝她嘴上亲了过去。
<br/>　　吴巧躲闪了几下，内心猛然下了某种决心：眼前这个丑陋的男人，不管是人是神，绝对不值得自己付出贞洁，并且服侍他三年。
<br/>　　在三足金蟾又一次想要亲她的时候，这一次她没有躲闪，半推半就地跟他亲了起来。
<br/>　　三足金蟾还当她是放弃抵抗了，心中大喜，一张粗大的舌头往她嘴里伸进去。吴巧也给与回应，突然，从吴巧的嘴里吐出了什么东西出来，三足金蟾下意识地以为是她的舌头，轻轻咬了一下，发现是个圆鼓鼓的硬东西，想要吐出去，但是已经晚了。
<br/>　　这东西在触碰到他口水的一刻，立刻就融化了，然后一股难以言说的气味，在他口中蔓延开来。
<br/>　　“你，喂我吃的什么东西！”三足金蟾厉声喝道，想伸手去抓吴巧，突然一阵爆裂的声音，在他喉咙里炸开，整个人好像筛糠一样抖动起来，向后倒在了地上，双手抠着嘴巴，在地上翻滚不停，不时发出凄厉的嚎叫。
<br/>　　就在这时，一到人影破窗而入，挺剑直上，砍向五通神。
<br/>　　正是一直埋伏在外面的叶少阳。
<br/>　　“是你！”三足金蟾一眼看到叶少阳，立刻辨认出来，想要扑上来抓他，但浑身战栗，眼看叶少阳挺剑到了面前，为了求生，这才爆发除了一股拼命的力量，张大嘴巴，吐出一条猩红色的舌头，扇动妖风，朝叶少阳卷过来。
<br/>　　叶少阳立刻感到昏天暗地，急忙用将长剑顶住，这时候道渊真人和毛小方也冲了进来，从两边进攻三足金蟾。
<br/>　　“我是五通神，尔等不敢杀我！”
<br/>　　三足金蟾双手擒着两股黑光，分别对左右二人展开了攻势，口中念叨什么，在妖风被叶少阳挡住之后，舌头一卷，继续朝叶少阳袭击。
<br/>　　他认定了叶少阳就是这一切的主谋，必然杀之后快。
<br/>　　叶少阳现在的法力，其实是三人之中最弱的，被三足金蟾绝命一击的妖风吹得七荤八素，刚把妖风挡住，冷不防一条舌头卷过来，刚要反抗已经来不及了，被舌头一下子卷住。三足金蟾真身本来就是一只蟾蜍，靠的就是一条舌头觅食。
<br/>　　这舌头凶悍无比，裹住叶少阳之后，好像蟒蛇一样立刻收紧，叶少阳差点就吐出来，急忙暗中运气，抵御这股庞然大力。
<br/>　　“你个法师，我今天让你死！”
<br/>　　随着三足金蟾一声大喝，叶少阳顿时感到舌头裹住自己的力量又加强了几分，立刻觉得两眼发黑，差点当场吐出来。
<br/>　　“去死！”
<br/>　　三足金蟾怪吼一声，舌头再度收紧……然而就在下一瞬间，一把宝剑，从后面斩断了他的尾巴。
<br/>　　噗的一声，三足金蟾狂喷出一口血，硕大的身体倒在了地上，缠绕叶少阳的舌头也解开了，叶少阳挣脱出来，定睛看去。三足金蟾已经倒在了地上，现出了蛤蟆精的真身：一头浑身长满了肉瘤的巨大的绿皮蛤蟆，歪着嘴躺在地上，浑身抽搐着。
<br/>　　叶少阳长出了一口气，看了一眼道渊真人，说道：“好险。”
<br/>　　叶少阳走到三足金蟾面前，居高临下地俯瞰下去，说道：“我说过要杀你，就一定能做到，现在你还怀疑吗？”
<br/>　　三足金蟾绝望地看着他，眼中尽是一副不可思议的神情。
<br/>　　“你……你敢屠神……”声音听上去十分虚弱，甚至带着一丝恐惧。
<br/>　　“敢啊，没什么不敢的。做错了事，就要接受处罚，神也一样，你好歹是一尊山神，不会这个道理都不懂吧？”
<br/>　　三足金蟾看着他，颤巍巍地说道：“你到底……是谁？”
<br/>　　“茅山天师，叶少阳。”
<br/>　　“叶少阳……”三足金蟾重复了一下他的名字，从他看自己的眼神里，叶少阳知道他已经相信自己会杀了他。
<br/>　　三足金蟾喃喃道：“你给我吃的什么？”
<br/>　　“三花灭妖丸。”
<br/>　　三花灭妖丸是以茅山内门法术炼制的一种丹药，与灭尸丸类似，以雄黄、桃蕊、百合心调配而成，专门针对妖精，叶少阳之前去县城，就是去买这些东西。
<br/>　　在三花灭妖丸的基础上，叶少阳又加了一味自己的血，调配成形之后，一旦被妖精吞下，沾上妖气，立刻融化，就算是万年修为也顶不住，不过因为必须让人吞下才行，因此实际使用起来的机会很少，叶少阳就知道三足金蟾这个老色鬼会轻薄吴巧，让美华提前来找吴巧，将用灵符裹着的三花灭妖丸裹在中间，等三足金蟾进屋之前，再撕掉灵符，藏在口中。
<br/>　　三花灭妖丸是针对一切妖物的，对人体没什么用，外层有一层蜡，也不会融化，只有到了三足金蟾口中，才会显示出它的药力……
<br/>　　这中间还有一件叶少阳所不知道的事，在得到三花灭妖丸之后，吴巧其实并没有下定决心要帮他们，倒不是怕他们杀了五通神，反而是害怕杀不死五通神，到时候自己作为同谋，会遭到五通神的报复，不光是她，还有她的父母家人，都会受到牵连。
<br/>　　甚至一度她想过放弃，主动告诉五通神这件事，以换取他的好感，看能不能放了自己，但是在五通神真的轻薄她的时候，那一瞬间的强烈反感让她没有多想，不顾一切地把三花灭妖丸送进了他口中……
<br/>　　这才造成了三足金蟾的溃败。他好色无度，贪图享乐，最后还是败在了姑娘身上。想到这一点，五通神心中也是唏嘘不已。
<br/>　　不过他已经没有精力唏嘘了，他抬头望着叶少阳，近乎哀求地说道：“你不能杀我，你若杀我，炼尸缸将重现人间。”
<br/>　　炼尸缸？？
<br/>　　叶少阳三人面面相觑。叶少阳对这三个字有点陌生，但是毛小方和道渊真人都知道炼尸缸的典故，当场失色。
