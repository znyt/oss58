　　一来是本门的传承，二来也是以示区别，真正的道门弟子，只要看道袍的样子，一眼就能知道对方是哪个门派的。叶少阳自然也不例外。虽然这是一百年前，不过龙虎山道袍的样式却是没有变化。
<br/>　　叶少阳站在远处看了一会，心中一动，走了过去。
<br/>　　“这位小哥，算卦还是摸骨？”龙虎山道士看了他一眼，叶少阳没穿道袍，之前穿越的时候穿的是当代的衬衫，最近翠云给他做了一身当代的衣服，反正也是自己纺的土布，不值钱。叶少阳现在的打扮，跟民国时期的人没什么区别。
<br/>　　叶少阳看了一眼招牌说道：“先生是龙虎山弟子，为什么挂个麻衣神相的牌子？”
<br/>　　道士一愣，上下打量叶少阳，皱眉道：“你怎知贫道是龙虎山弟子？”
<br/>　　叶少阳捏了个子午诀，道：“见山不是山，见水不是水。”
<br/>　　道士又是一怔，盯着他说道：“山水有相逢，同为蓬篙人。”
<br/>　　叶少阳又道：“山外有山，水外生水，请教了。”
<br/>　　道士还礼道：“不敢，贫道龙虎山第三十八代外门弟子陈有信，流落江湖，混口饭吃。世人都知道麻衣神相，也只好借用名字了。不知道友何来？”
<br/>　　叶少阳心中盘算了一下，他是龙虎山第三十八代传人，比自己也就长一辈，这是因为龙虎山当代弟子，一向是比茅山晚两辈，自己那个时代的龙虎山掌教张无生，还得叫自己一声师叔。所以就算是来到一百年之前，自己所遇到的这个道士，辈分上也不过是早自己一辈而已。
<br/>　　叶少阳当下拱手行礼，道：“茅山第三十九代弟子叶少阳，这厢有礼。”
<br/>　　因为被他看穿了身份，又对上了切口，陈有信对他本没有怀疑，一听叶少阳自称是茅山第三十九代弟子，愣了一下，然后失笑，有些不屑地说道：“我本以为遇到道友了，原来是错了，小哥你这切口是从哪个道士朋友那里偷学的？”
<br/>　　叶少阳一怔，问道：“这话怎么说的，我哪里说错了吗？”
<br/>　　陈有信摆手说道：“你倒是没说错什么，但你毕竟不是道门弟子，不了解内情，茅山比我龙虎山长两辈，目前最年轻的弟子，也是第三十六代，怎么会有三十九代弟子，三代人，恐怕得排到百年之后了。”
<br/>　　叶少阳顿时无语，心想大哥我就是百年之后来的啊，但这话当然说不出。没想到竟然是这方面出了问题，叶少阳真心不好解释，只好硬到底，表示自己的确是三十九代传人。
<br/>　　陈有信笑道：“你说你是茅山传人，那你师父是谁？”
<br/>　　“这……”叶少阳考虑要不要说出青云子的名字，想想就算说了也没用，这个时代青云子肯定没出生呢。
<br/>　　陈有信也没想过他能答得上来，又问道：“那你再说说，茅山的掌教是谁？”
<br/>　　“伏明子？”
<br/>　　陈有信一听，倒是有些惊讶，道：“你能知道伏明子的名字，对茅山还真有些了解，不过伏明子虽然是内门弟子，能不能当上掌教还不一定，现在的掌教可是他师父云秋生呢。”
<br/>　　“对对对，云秋生！”叶少阳敲了敲脑门，茅山历代祖师，尤其是近代的，他都说的出名字，只记得伏明子是青云子之前的掌教，好像是民国时期的，也是当了几十年的掌教，听青云子说七十年代才咽气，所以顺口就把他说了出来，却没想到这个年代他还每当上掌教，这时候掌教还是云秋生。
<br/>　　叶少阳问道：“敢问现在龙虎山掌教是那位仙长？”
<br/>　　“家师紫云真人，如今坐观。”陈有信对他拱了拱手，“小哥既然不是道门中人，贫道也不方便透露太多，这就轻便，贫道这还要做生意。”
<br/>　　人家下了逐客令，叶少阳只好走了，怔怔地走了几步，又转回头，道：“请教了，我只问一个问题：龙虎山的道渊真人，你认识不？”
<br/>　　道渊真人，在叶少阳那个时代，号称法术界的活化石，据说有一百多岁，现在离 2016年有九十多年，如果传言属实，这个时候道渊真人已经出世，并且已经至少有二十岁，说不定已经拜在龙虎山门下。
<br/>　　陈有信一听道渊真人的名字，道：“你也听过道渊那小子？”
<br/>　　“那小子”三个字，让叶少阳嘴角抽搐了下，不过对方这么说，说明自己没有问错，顿时有些激动，点头道：“不知道道友认不认识？”
<br/>　　“认识，龙虎山百年之中，出了这个奇才，怎么能不认识呢。”陈有信嘴上这么说，用的却是讥讽的口吻。
<br/>　　叶少阳当然听出来了，问道：“这话怎么说？”
<br/>　　陈有信不说。
<br/>　　叶少阳只好问他道渊真人在哪。陈有信被问得烦了，只好告诉他，道渊真人犯了劫，被关在后面面壁思过，已经有一两年了。
<br/>　　陈有信见叶少阳缠着不放，耽误他做生意，看他像是本地人，也不敢惹，最后烦得不行，自己起身挑着招牌走了。
<br/>　　叶少阳没有去追赶他，怔怔地站在原地。此刻，他的内心是激动的。道渊真人……自己在这个世界，总算有了一个认识的人！
<br/>　　不过……这个道渊真人，听陈有信的意思，也就是个年轻人，而且既然能被他一个外门弟子嘲讽，在山上的地位肯定不是很高，至少比那个时代坐镇龙虎山太上掌教的他差远了。所以，这两人其实并不是一个人。也就是说，这个世界的道渊真人，肯定不认识自己。
<br/>　　叶少阳顿时又感到有些失望。不过思前想后，他决定还是去找道渊真人，毕竟，道渊真人是自己在这个世界遇到的第一个熟人，就算不是同一个人，自己也要走一趟见见他，把自己的情况跟他说，假如能说服他相信，或许可以帮自己一把。
<br/>　　至少，自己也有个可以商量的人。
<br/>　　主意打定之后，叶少阳立刻回到翠云家。翠云正在院子里洗菜，见叶少阳进来，说道：“让你买豆腐买了吗？”
