　　“好大的力气。”
<br/>　　萧逸看了眼自己的拳头，又看了眼那头妖兽的身体。
<br/>　　自己全力一拳轰出，竟连一丝拳印都不曾留下。
<br/>　　这几头妖兽，在吸收了数量庞大的武道冰晶后，绝对已达到武道大能以上的水准。
<br/>　　想来也是。
<br/>　　妖兽，之所以强，是因为它们魂兽的庞大妖元，以及惊人的妖兽肉体。
<br/>　　而人类武者，相对弱得多。
<br/>　　所以人类武者，需要修炼功法，参悟天地武道。
<br/>　　而武道力量，乃是天地之力，自是强大莫测，仅仅是一丝，就足以让人类武者的寿命达到动辄数百年以上。
<br/>　　圣境之后，武者的大部分实力，也来自于武道力量的掌控。
<br/>　　现今，武者的这些武道力量，被妖兽吸收。
<br/>　　对于妖兽的肉体，那是极大层次的提升。
<br/>　　武道力量，是天地之力，连武者都无法正常使用；需得靠领域的方式，从而增幅自身。
<br/>　　而这些邪修，却是通过武道冰晶，吸收了武者的武道力量，再强行加诸在妖兽身上。
<br/>　　这才造就了面前这数头凶兽。
<br/>　　吼…吼…吼…
<br/>　　萧逸在思索着，打量着这些妖兽。
<br/>　　可这些妖兽，可是已狂性大发，暴躁不堪，可不会给萧逸打量的时间。
<br/>　　周遭邪修已退去。
<br/>　　原地，只剩萧逸这个目标。
<br/>　　自然的，萧逸便是它们攻击的对象。
<br/>　　不过，萧逸也不太在乎。
<br/>　　手中紫炎，凝聚惊人，轻松地与这些妖兽鏖战着。
<br/>　　而周遭，整个花家府邸，早就被他用屏障禁锢，这些邪修一个也别想逃掉。
<br/>　　吼…
<br/>　　数头妖兽，或巨大的身躯下，重重朝萧逸袭来。
<br/>　　或锋锐的利爪，直取萧逸身体，欲要将萧逸撕成碎片。
<br/>　　嘭…嘭…嘭…
<br/>　　萧逸一边躲避，一边不断打出火焰。
<br/>　　霎时间，整个花家府邸，一片狼藉，火焰飞舞。
<br/>　　数头妖兽的气息，仍旧在上涨。
<br/>　　萧逸想看看，这些妖兽最终能达到什么实力。
<br/>　　数十分钟后。
<br/>　　萧逸还未急。
<br/>　　周遭邪修，却已脸色大变，“该死，邪道凶兽怎地还未击杀那易霄。”
<br/>　　“再这般下去…”
<br/>　　周遭邪修，脸色难看到极点。
<br/>　　这数头妖兽，若杀不了萧逸，他们便无法逃离这片屏障。
<br/>　　萧逸瞥了眼远处邪修的脸色，虽疑惑，但也暂时未理会。
<br/>　　绝对的实力下，拥有的，自然是绝对的信心。
<br/>　　此时，这数头妖兽的实力，大概达到9900道层次的武道大能水准。
<br/>　　这等实力，已经差不多超过当初在三风分部那，由三风大阵凝聚而出的邪道三风兽了。
<br/>　　萧逸也不得不祭出了紫炎领域，将地脉金火和十界灭生火的武道力量一并融入领域。
<br/>　　“不会达到绝世强者层次吧？”萧逸看着气息仍旧在上涨的妖兽，眉头紧皱了起来。
<br/>　　不过，片刻之后，萧逸终于发现了一丝不对劲。
<br/>　　那便是，这些妖兽气息在上涨的同时，气息也在变得更加紊乱不堪。
<br/>　　这些妖兽虽然狂暴至极，狰狞异常，但脸上带着的，则是异常痛苦的神色。
<br/>　　十数秒后。
<br/>　　这些妖兽，忽然停下了肆虐的身影。
<br/>　　身上的气息，陡然紊乱到极点。
<br/>　　“难道说…”萧逸陡然脸色大变。
<br/>　　“不好。”
<br/>　　萧逸身影连连一退，双手齐出，漫天紫炎顷刻打在了屏障之上。
<br/>　　几乎是他做好这些的瞬间。
<br/>　　轰…轰…轰…
<br/>　　一连数道滔天爆响。
<br/>　　数头妖兽，竟是瞬间自爆。
<br/>　　恐怖的威力，让得整个花影城震颤不停，宛若地裂。
<br/>　　惊人的气势，冲天而起，整个花影城上空，空间破碎，狼藉一片。
<br/>　　待得爆响声落下。
<br/>　　整个花家府邸，已夷为平地。
<br/>　　妖兽、邪修，已悉数全无，显然是在惊天爆炸中化作了齑粉。
<br/>　　全场，唯萧逸毫发无损。
<br/>　　但面具之下的脸色，仍旧是一白，嘴角一丝鲜血溢出。
<br/>　　刚才那数股自爆，若非他反应快，及时以紫晶灵炎稳固了原本封锁花家的屏障的话，恐怕整个花影城都会被夷为平地。
<br/>　　9900道层次的武道大能水准，接近绝世强者的层次自爆，其威力，相当可怕。
<br/>　　而一个大城之内，人口可不是一个小数字。
<br/>　　当然，也幸好他早有准备，在四周布下了屏障。
<br/>　　只不过，后果便是他维持屏障，需得一人承受下这数股可怕的自爆威力。
<br/>　　这才让得他也受了不轻伤势。
<br/>　　这时便看出一个大城，或者一个地域之内，拥有强者庇护的好处了。
<br/>　　中域，强者云集，又杀戮无数。
<br/>　　若无强者庇护一方，对于普通武者来说，强者的交战，将是一场灾难。
<br/>　　上古八殿的分殿，遍布中域各城，也有这方面的因素。
<br/>　　中域虽是强者为尊，虽是无规则律例可言，但终究有一份底线。
<br/>　　否则，这跟千万年前妖兽为祸，涂炭生灵之时，有何区别？
<br/>　　“呼。”这时，萧逸轻呼出一口气，稳下了身上伤势。
<br/>　　伤势，不轻，但也并不怎么重，之后稍微休息一番便能痊愈。
<br/>　　看着周遭夷为平地的景象，萧逸摇了摇头。
<br/>　　邪修的手段，果真诡异。
<br/>　　那些武道冰晶，确实也极强，但也不是无限增幅妖兽实力的。
<br/>　　当妖兽的肉体支撑不住时，自然便也自爆了。
<br/>　　刚才萧逸没看错的话，那几头妖兽，足足吞下了数百颗武道冰晶。
<br/>　　这才有了刚才如此惊人的自爆威力。
<br/>　　这个邪修分部，算是毁了。
<br/>　　这里的邪修，也死尽了。
<br/>　　但…线索和情报也全断了。
<br/>　　萧逸之前屠的几个邪修分部，虽也是火焰滔天，将邪修分部焚毁殆尽。
<br/>　　可那是在他的操控之下，以他的控火手段，足以精准而完美地留下他需要的东西，如乾坤戒和情报。
<br/>　　可刚才这几头妖兽的自爆，可是无差别爆发。
<br/>　　整个邪修分部内，别说乾坤戒，就是一花一草都没有留下，悉数被炸成了齑粉。
<br/>　　“情报又断了，该死。”萧逸咬了咬牙。
<br/>　　恰在此时。
<br/>　　嗖…嗖…嗖…
<br/>　　花家府邸上空，一道道身影激射而来，显然是来得极急。
<br/>　　“何人赶在我花影城闹事？”
<br/>　　“嗯？花家府邸被夷为废墟了，该死，好大胆的贼人。”
<br/>　　萧逸只看了一眼上空，如果他没猜错的话，这一道道身影，便是花影城内的强者了。
<br/>　　如花影城城主，以及这里的各分殿负责人。
<br/>　　“敢在我花影城杀戮？本城主饶不了你。”一个中年人，怒喝一声。
<br/>　　“慢。”一个老者，拦下了中年人。
<br/>　　老者，直视萧逸，不，准确来说，是直视萧逸的面具，“敢问阁下，可是易霄分殿主？”
<br/>　　“老夫猎妖殿花影城分殿分殿主。”
<br/>　　萧逸淡漠地点了点头，“这里是邪修分部。”
<br/>　　“邪修分部？”高空中一道道人影，面露惊色。
<br/>　　但下一秒，又眉头一皱，“你说是便是？证据呢？”
<br/>　　“易霄，即便你是猎妖殿分殿主，也不能平白杀戮…”
<br/>　　说话的，是天机殿分殿的分殿主。
<br/>　　上古八殿，各殿武者的服饰，是有区别的。
<br/>　　萧逸一眼就能认出这一个个老者来自于哪座分殿。
<br/>　　“是与不是，你们自己查。”
<br/>　　“若连这点本事都没有，易某也没兴趣与你们浪费时间。”
<br/>　　话音落下，萧逸紫炎火翼一振，御空飞离。
