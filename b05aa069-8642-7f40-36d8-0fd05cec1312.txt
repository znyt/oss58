　　天空之上。
<br/>　　一道剑光，猛然划过空间，瞬息之间，逼近楚风眠。
<br/>　　这剑光，锋利无比，似乎是要将面前的一切，都给直接斩断。
<br/>　　这一道剑光，几乎瞬间，便是来到了楚风眠的面前。
<br/>　　楚风眠站在原地，似乎都有些没有反应过来，还有些呆滞。
<br/>　　看到这一幕，司空弘更是心中冷笑起来，他的这一道剑光，让那些度过生死大劫的天才来，都无法抵挡得住。
<br/>　　更何况他全力之下，对付一个神力境武者。
<br/>　　他已经看到了下一刻，楚风眠毫无还手之力，被这一道剑光，直接抹杀的一幕。
<br/>　　这种速度，就算是如今寒月漓想要救，都不可能。
<br/>　　“雕虫小技。”
<br/>　　在这剑光，距离楚风眠只差一直距离的一刻，楚风眠的口中，才淡然吐出四个字。
<br/>　　只见楚风眠突然大手一挥，猛然一抓，这一道剑光，却是被楚风眠，直接是抓在了手中。
<br/>　　锋利的剑光，不断的切割这楚风眠的手，但是楚风眠的手上，却是没有任何一点伤痕，仿佛是这剑光，都无法伤及到楚风眠。
<br/>　　“碰！”
<br/>　　随着楚风眠手一用力，这一道剑光，便是直接在楚风眠的手中，被硬生生的捏碎。
<br/>　　周围看到这一幕的人，脸色都是一愣，似乎有些不可置信，如今楚风眠的强大。
<br/>　　尤其是那冰漓长老，更是一副见了鬼的样子，她猜测过楚风眠的实力，却是想不到，楚风眠居然足以做到这一步。
<br/>　　徒手接下一尊剑修的剑光。
<br/>　　这简直是疯狂到了极点的行为。
<br/>　　更何况，这司空弘，可不是什么寻常剑修，是真正的天才，百绝榜中的人物，如今居然是他的剑光，都被楚风眠直接接下。
<br/>　　“嗯？”
<br/>　　司空弘的脸色，也是一愣，眼神中闪过一丝惊讶，不过他的脸色，看起来却还是一如既往的平静。
<br/>　　“好强大的肉身，能够接下我的剑，看来你的肉身，的确是达到了一种不可思议的地步！”
<br/>　　“不过，就凭借这肉身，却是根本不足以跟我抗衡！你的肉身再强，我就不信，可以强的过我的剑！”
<br/>　　司空弘心神一动，他手心的剑，却是消失。
<br/>　　取而代之的，却是另一把剑，猛然出现在了他的手中。
<br/>　　这一把剑上的力量，却是跟刚刚，截然不同。
<br/>　　刚刚司空弘手中的剑，只能算是一件寻常的地级灵器，但是如今，他手中的剑，其中弥漫的气息，却是天级灵器的气息。
<br/>　　这灵剑出窍的一刻，锋利的剑芒，仿佛就是要将楚风眠，直接斩杀，斩成碎片。
<br/>　　“司空大罗剑！这是司空家的至宝！”
<br/>　　冰漓长老看到这剑，都是有些大惊失色道。
<br/>　　“这把剑，怎么会在你的手上！”
<br/>　　这司空大罗剑，可是一件天级灵器，一件天级灵器，在生死境的战斗中，足以有着决定性的作用。
<br/>　　司空弘的实力，本就在楚风眠之上，如今在加上这司空大罗剑，楚风眠就连任何一丝战胜司空弘的机会，都没有了。
<br/>　　司空景胜嘴角露出一抹冷笑说道。
<br/>　　“冰漓长老，不用大惊小鬼，这司空大罗剑，一直都是由我们司空家的家主掌控，如今我这孙儿，已经是被立为司空家下一代的家主，这司空大罗剑在我孙儿的手中，倒也正常。”
<br/>　　对这一战，司空景胜倒是无比自信，他不觉得，楚风眠这样一个就连北域百绝榜都没有上的人，能够跟司空弘抗衡。
<br/>　　在他看来，不足三招，楚风眠必败，甚至是很有可能，直接陨落。
<br/>　　“小子，你就不是仗着一身肉身的力量，才有自信跟我抗衡吗？现在我倒是要看看，是你的肉身硬，还是我手中的剑锋利！”
<br/>　　司空弘看着楚风眠，冷笑连连道。
<br/>　　“现在跪下，直接认输，我可以饶过你性命，放你离开。”
<br/>　　司空弘对着楚风眠开口，这语气，好像就是在对一个语气施舍。
<br/>　　如今司空大罗剑一出，司空弘便是已经觉得，他已经是彻底制住了楚风眠，一个人的肉身再强，也不可能跟天级灵器抗衡。
<br/>　　现在的楚风眠，在他看来，便是一直想要如何玩弄就如何玩弄的蝼蚁。
<br/>　　但是他的话音未落，楚风眠的脸上，却是露出了几分嘲笑。
<br/>　　“这把剑，是好剑，不过用剑的人，却是一个废物，这把剑在你的手中，也没有任何的用处，给我拿来吧！”
<br/>　　楚风眠冷笑一声，猛然一抓，已经是向着司空弘抓了过去，要将他的手中的剑，直接夺走。
<br/>　　“想要夺剑？找死！”
<br/>　　司空弘长啸一声，他手中的剑，一下子落下。
<br/>　　这一剑落下，足以分化出了千道剑气，同时冲着楚风眠斩杀过去。
<br/>　　一剑千重，这司空弘的剑术，已经称得上是不错了，在北域之中，同境界武者中，几乎是最强者。
<br/>　　以他的资质，将来很有可能也会成为一尊极为可怕的剑道宗师。
<br/>　　只可惜，如今这司空弘碰到的人，却是楚风眠，他既然招惹上了楚风眠，那他也就没有什么将来了。
<br/>　　司空弘的剑术，虽然不错，但是在楚风眠看来，却是破绽百出。
<br/>　　“龙！”
<br/>　　楚风眠的右手，突然之间，龙血凝聚，直接化为了一道巨大无比的龙爪。
<br/>　　这龙爪的五根手指，指尖上，都是一股剑气，猛然凝聚出来，凌空一抓。
<br/>　　这天空上的千道剑气，在这一抓之下，直接层层破碎，没有任何一道，还能残留。
<br/>　　这一剑，居然便是如此，被楚风眠这样轻而易举的给破掉了
<br/>　　司空弘手持天级灵器，凝聚剑意的一剑，在楚风眠的一抓之下，全部消散。
<br/>　　“什么？居然可以破开我的剑术？怎么可能！”
<br/>　　司空弘脸色猛然一变，再也没有之前的冷静，而是脸色上，隐约之间都露出了几分惨白。
<br/>　　不管是他的剑术，还是他的剑，几乎都是在同境界中，无懈可击。
<br/>　　可如今，就这样被楚风眠如此轻易的破开了，而且楚风眠根本不是以力破开的，而是直接寻找到了他剑术的破绽，将他的剑术，彻底击败。
<br/>　　在剑术上，司空弘，居然败了。
