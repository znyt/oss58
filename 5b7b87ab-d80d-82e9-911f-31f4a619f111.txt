　　半月后，苏重已把铠甲事宜，处理的差不多。铠甲已发给土豪“酋长”。
<br/>　　而经过一系列手续后，他也获得了七十二万的打赏金。到这里事情算是告一段落，苏重已实现赚钱目标。
<br/>　　收到钱后，他就向家里打了十万块。苏重不敢打太多，一个刚毕业的学生，突然得到这么一笔钱，实在让人放心不下。
<br/>　　果然，他刚打完钱半小时，老妈的电话就打了过来。
<br/>　　“小重，你往家里打钱了？”母亲语气有些严肃。
<br/>　　“嗯，是我往家里打了十万块钱，那不是诈骗。”苏重很轻松回道。
<br/>　　他打算直言相告。这笔钱来的快，看着有些走捷径，在老一辈人眼里，这就是歪门邪道。不过放在现在，这也并不是什么大惊小怪的事情。
<br/>　　“你从哪里弄来那么多钱！”母亲焦急的快速道。
<br/>　　“妈，您别着急。这钱虽然赚的快，但来路正。早就知道你会担心，这笔钱的来源，我已经拍照记录，稍后我给你发过去。你也别急，听我慢慢告诉你，这些钱的来历……”
<br/>　　好半晌，苏重才把整个过程讲清楚：“总的来说就是，我做了一个机械模型，因为比较高档，所以卖了一个好价钱，足有三十多万。这回你放心了吧。”
<br/>　　苏重最后还是撒了个谎，只说自己赚了三十万，甚至不到实际数目的一半。
<br/>　　“你不是卖了三十多万吗，给家里打来十万，手里留那么多钱干什么？家里再凑个三四十万，再加上你的钱，足够在县城买个学区房了。等交了钥匙再装修完，怎么也得一两年。到那时候正好给你结婚用。那些钱可不能乱花！”刚刚还在担心钱的来历，现在又开始担心苏重乱花钱了。
<br/>　　苏重无奈一笑，这才刚毕业，老妈就开始考虑结婚了。自己还是个单身狗呢。
<br/>　　“买房子不着急，我看好一个投资项目，准备试试手。钱来的容易，花起来不心疼，也不怕赔本。万一真赔了，大不了我再多做几个模型。”在这之前，苏重心里就已有打算。
<br/>　　“模型这么赚钱，你还去投资其他项目？”苏妈不理解。
<br/>　　“当然是其他投资更赚钱啊。”苏重只能继续扯谎，目前投资什么都还没影呢，哪里谈得上赚钱不赚钱。
<br/>　　“而且，模型做多了就不值钱了。等什么时候缺钱，我就做一个。毕竟物以稀为贵吗。”
<br/>　　“哦，你心里有数就好。你不趁着最近有时间回家一趟？”苏妈问道。
<br/>　　苏重知道，老妈这还是不放心，打算把自己叫到面前仔细盘问。苏重虽然有些无奈，却并不厌烦，而且他之后确实会很忙。
<br/>　　之前不断穿越世界，为的不就是回到本源世界和父母家人团聚。如今有了时间，苏重非常在乎。
<br/>　　“好。我把这边的事情收一下尾，两三天之后就回去。”苏重答应的爽快。
<br/>　　之后几天，苏重快速处理收尾事宜。把厂房租赁给退了。制作模型剩下的材料没多少，直接扔在厂房。
<br/>　　只是把图纸搜集整理起来封存，怎么说也是自己的努力成果。
<br/>　　一些小工具还是刚买的，苏重打包起来，发了个邮件给家里。自己以后用不到，家里平时切割修补，说不定用得着。毕竟刚买的就卖二手，不划算，扔了就太可惜。
<br/>　　回家后，苏重在家里呆了半个多月，整天除了吃就是睡。什么本源、穿越都不想。刚减下来的重量，就再次给吃了起来，过的那叫一个潇洒。
<br/>　　之后苏重就再次离开家踏上征途。
<br/>　　他非常想住在家里。但为了本源，苏重必须不停穿越。穿越时身体会卧床不动，就像植物人。
<br/>　　而且由于世界不同，他每次醒来的时间也不一样。有时可能一两天，有时可能会十天半个月。
<br/>　　父母在旁，自己一下子晕过去半个月，肯定会引起恐慌。他不得不离开家，找一个更加隐蔽的地方。
<br/>　　之前租房住，也不是一个好办法。万一哪天房东有事来找，苏重晕着没法回应，说不定也会引起麻烦。
<br/>　　他需要一个不受打扰的隐秘之地，但苏重又不想直接往深山里钻。
<br/>　　这里是本源世界，苏重只是一个身体比常人更健康的普通人。没有超凡力量，山林里对他来说，非常危险。想要独自一人进山避世，根本不现实。
<br/>　　他原本还在发愁隐居地，但出乎苏重预料，这个问题很快就得以解决。
<br/>　　在为了应付老妈询问，查找所谓的投资项目时，他意外发现一个好去处。
<br/>　　苗家村是一个偏远山村。这里有山有水，空气清新风景优美。是一个少有的无污染纯天然乡村。
<br/>　　但这里太偏僻，只有一条蜿蜒小路绕山而行，弯弯曲曲的通往山外。本来是个极好的农家避暑旅游之地，却因为交通原因，始终无法发展起来。
<br/>　　两年前，有一个外商包下苗家村后山，要采用散养方式，养殖高质量的家禽。可惜，这么长时间过去了，一直不温不火。没怎么赔，却也没赚。勉强维持收支平衡。
<br/>　　到了现在，终于经营不下去，有意向外转让。
<br/>　　苏重看到这则广告的时候，立刻就心动了。包下一座山养鸡，他可以用防止疫病的缘由封山。这样一来，即便他数月不露面，也不会担心被人发觉问题。
<br/>　　而且，这事是有先例的，甚至被商家作为卖点四处宣传。苏重一点儿都不担心别人怀疑。
<br/>　　而且苗家村地处偏僻，除了村里的留守老幼，根本就不会有外人来这里。苏重几乎瞬间就决定，要把这个苗家村养殖场接过手来。
<br/>　　骑着一辆二手摩托，辗转数个小时，被颠的七荤八素的苏重，终于到了苗家村。
<br/>　　村子偏僻，少有外人。他这辆排气如放炮的摩托进村时，瞬间就引起众人注意。一群坐在大树下聊天下棋的老人，纷纷好奇打量苏重。
<br/>　　一个黑瘦中年人看到苏重，兴奋的从人堆里站起来，高高兴兴的跑到苏重面前。
<br/>　　“是陈小哥吗？我这已经等候多时了。”中年人热情的和苏重握手。
<br/>　　眼前这黑瘦中年人，就是苗家村养殖场的老板——沈松。之前苏重就曾经和他进行过电话联系。
<br/>　　“陈小哥，之前电话里听你声音，就觉得你年纪不大。没想到竟然这么年轻。”沈松惊奇的看着苏重。嗫嚅半晌犹豫道：“小哥，你真想好要接手我这养殖场？”
<br/>　　“嗯，我已经想好了。这钱是我自己的，家里不管。你放心就好，不会给你添麻烦。”苏重面色不变，心里却忍不住吐槽，脸嫩怎么办，我也很无奈啊。
<br/>　　“那就好。”沈松半信半疑，不过他也急于脱手，不再细问。而且转手之后，他就要离开这里，说不定永远也不会回来，他也不怕这些麻烦。
<br/>　　“陈小哥，既然你实心实意接手。我也不骗你，这山我一口气包了十五年。每年三千块。这四万五的租金，之前两年的不要，剩下的你得给我补齐。我这里有合同，村大队的人也能作证。”
<br/>　　苏重点头表示同意，这是应有之意。
<br/>　　“山上还有不少设备和一些鸡，杂七杂八算下来，我给你算五万块。合起来总共八万六，你看怎么样。”沈松有些紧张的看着苏重。
<br/>　　苏重想了想，这五万块，肯定有水分。不过苏重不想扯皮，盯着沈松看了好一会儿，直到对方被看的满头冒汗，苏重才淡淡点头表示同意。
<br/>　　之后就在村长等几位老人见证下，两人签订了转让合同。从这天气，苗家村养殖场，就属于苏重了。
<br/>　　“陈小哥，虽然有些不地道。但我还得给你说清楚。这里山清水秀，山上草木丰茂。养出来的鸡非常好，鸡蛋具有极高营养价值。只是……只是这里太偏僻了，太深入山林了！”沈松一脸的懊恼。
<br/>　　“自从养殖开始，山上的鸡就不断减少。鸡蛋也引来不少动物，杀都杀不完，简直蛇鼠成灾！虽然鸡和鸡蛋质量好，但架不住这种损失啊，所以我才转手。小哥，你可要有个心理准备。”
<br/>　　沈松说完，也不看苏重脸色，转身就走。好似身后有狼撵着一般。他心里虚，签了合约才说，着实坑了苏重一把。
<br/>　　苏重却没怎么在意。他又不是真的来养鸡的。签完合同，苏重就拿着沈松给的钥匙，上山去看看。
<br/>　　沈松这个时候已经走了。早在苏重来之前，他就已收拾好包裹。此刻签完合同，忍着心里不舍，直接就走。
<br/>　　就连给苏重引路介绍都不想干了，怕自己反悔，毕竟这是他两年多的心血。
<br/>　　山坡下有一圈铁网，把山头揽了一圈，一个铁栏杆大门，正对着村里唯一一条上山路。
<br/>　　打开大门，苏重缓步上山。在半山腰一处平台，找到两间小屋。是用空心砖块，快速搭建起来的瓦房，墙皮都用水泥抹平，外貌还有这七八成新。
<br/>　　虽然有些简陋，但也足够坚固。又绕着周围转了一圈，看了看养殖场里的各项设备。
<br/>　　发现沈松报的五万块价格虽然有些虚高，但也不算太离谱。
<br/>　　设备房屋等用具很新，更多的还是咯咯叫着到处跑的黑羽鸡。
<br/>　　这是一种独特品种，能产绿皮鸡蛋。怪不得养殖场被山林动物祸害，沈松依旧还能坚持两年。这里的养殖产品，还真是高级货。
