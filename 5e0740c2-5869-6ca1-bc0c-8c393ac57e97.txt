　　匕首落空，杨伏微微一惊，在他的预料中，以一个凡境二重的速度，绝不可能躲开。
<br/>　　萧逸趁着杨伏愣神，脚步立马挪动，快速拉开距离。
<br/>　　“哼，本想让你死得痛快些，既然你自己找不痛快，那就别怪我了。”杨伏不想浪费时间，没有再保留实力，全力朝萧逸攻去。
<br/>　　全力出手的凡境九重，速度快得吓人，比凡境二重快了十倍都不止。
<br/>　　“破绽来了。”萧逸双眸一凛，露出了冷笑。
<br/>　　杨伏虽然自信，但身在萧家，心存忌惮，不得不全力出手。
<br/>　　全力出手，速度飙升到极致，代表着他将没有退路。
<br/>　　当然，在他看来，他不需要退路，对付一个区区的凡境二重罢了，他绝对能做到秒杀。
<br/>　　然而，萧逸也有着他的保命底牌——冰鸾剑武魂。
<br/>　　对于那把神秘的冰鸾剑，萧逸有着足够的信心，确信它必然极为强大。
<br/>　　以萧逸过人的战斗经验和丰富的见识来看，绝对有把握拼着重伤的代价，将这杨伏击杀。
<br/>　　虽然代价有些大，但毕竟凡境二重与九重的差距太大了。
<br/>　　恰在此时，房门忽然被推开，依依走了进来，怯生生地说道，“少爷，我给你热了下饭菜，你自个儿吃，依依放下就走，不会影响你的心情。”
<br/>　　依依不安地低着头，怕惹萧逸不高兴。
<br/>　　气氛骤然停格。
<br/>　　“额。”萧逸愣了愣。
<br/>　　杨伏也愣了愣，停下了攻击。这里毕竟是紫云城三大家族之一的萧家，他也担心会忽然出现一个凡境以上的强大武者。
<br/>　　当发现只是一个没有实力的侍女时，他松了口气，立马再次朝萧逸刺去。
<br/>　　这时，依依也抬起了头，发现了这里的状况。
<br/>　　“少爷。”依依大惊，瘦弱的身躯却爆发出极快的速度，立马朝萧逸跑去。
<br/>　　“依依，快跑。”萧逸顿时大急。
<br/>　　“小子，先顾好你自己把。”杨伏一匕首刺下。
<br/>　　萧逸暗暗咬牙，依依忽然出现，他一分神，被杨伏抢到了出手的先机。
<br/>　　“小子，死吧。”杨伏露出了狞笑。
<br/>　　“嗤”的一声，毒匕刺中了。
<br/>　　不过，刺中的却是依依的身体。
<br/>　　“依依。”萧逸大惊。
<br/>　　依依的出现太过突兀了，突兀得萧逸有些反应不过来。当萧逸反应过来时，依依已经挡在了自己面前。
<br/>　　幸好他眼疾手快，堪堪拉住了依依，这才导致毒匕没有刺中依依的要害，仅仅是刺中腰背。
<br/>　　“混蛋。”萧逸惊怒交加，心头涌现出一股强烈的愤怒与杀意。
<br/>　　“给我死。”萧逸瞬间祭出了冰鸾剑。
<br/>　　一抹耀眼的紫芒突兀出现，一把绝世神兵瞬间凝聚于手中，狠狠地朝杨伏刺去。
<br/>　　“哼。”杨伏冷哼一声，一把从依依背后将毒匕抽出，而后刺向萧逸。
<br/>　　“一个凡境二重竟还敢和我正面较量，真真找死。”杨伏满脸不屑。
<br/>　　然而下一秒，杨伏脸上的不屑瞬间变为惊骇与恐慌。
<br/>　　当冰鸾剑与毒匕碰撞之时，剑尖对匕尖。
<br/>　　那由真气凝聚而成的毒匕瞬间崩溃。
<br/>　　“怎么可能？”杨伏大惊失色。
<br/>　　还未待他反应过来，毒匕已经破碎消失，而那抹紫芒，则带着锋利无比的气息，直直插入他的身体，洞穿了心脏。
<br/>　　凡境九重强大的肉身，毫无抵抗之力。
<br/>　　“这...这...”杨伏话没能说完，大脑已经失去意识，无力地倒在地上。
<br/>　　他临死都想不明白，自己一个凡境九重，仅次于后天境的强大武者，竟然会死在一个凡境二重的小子手上。
<br/>　　曾经在紫云城颇有凶名的毒匕杀手，就此死亡。
<br/>　　“依依。”萧逸刚打算看依依伤得严不严重时....
<br/>　　脑海中忽然‘轰’的一声，出现了一股陌生的记忆。
<br/>　　脑海中蓦然出现了五个大字，‘冰鸾剑传承’。
<br/>　　同一时间，体内的冰鸾剑武魂光芒一闪，一股古老的力量传递到双眼之中。
<br/>　　萧逸的眼睛忽然变了，左眼深邃漆黑如墨，透漏着一股无尽深渊的冰冷；右眼炽热如火，眸子通红，仿佛蕴含着一轮太阳星辰。
<br/>　　“好冷....好热。”萧逸对于眼中忽然出现的感觉很是不适。
<br/>　　“左眼太阴，右眼太阳，这就是冰鸾剑的传承吗。”萧逸心里呢喃了一声。
<br/>　　通过脑海中的记忆得知，太阴太阳，乃是冰鸾剑中本身蕴含的能力。
<br/>　　只不过，这个能力必须是冰鸾剑承认的主人，才能获得并使用。
<br/>　　换个说法，太阴太阳，便是冰鸾剑认主的标记。
<br/>　　从此，冰鸾剑将真正认萧逸为主。
<br/>　　得到这个能力，方能真正掌控冰鸾剑，并发挥出冰鸾剑百分百的威力。
<br/>　　在这双一黑一红的眼睛之下，萧逸看透了杨伏的身体，看到了他体内的黄阶武魂毒匕。
<br/>　　忽然，左眼和右眼中同时迸发出一个虚无的无形漩涡。
<br/>　　两个无形漩涡竟在主动吸收杨伏体内的毒匕武魂。
<br/>　　几乎是肉眼可见的速度，杨伏的毒匕武魂不断萎缩，最后直接消失。
<br/>　　而这些力量，透过漩涡，传递回太阴太阳之眼，最后传达到萧逸体内的武魂之上。
<br/>　　“吸收武魂力量，这...”萧逸目瞪口呆。
<br/>　　萧逸感受到，体内原本散发着赤色光芒的弱小控火兽武魂，瞬间变成了橙色武魂；而冰鸾剑武魂，却没有任何变化。
<br/>　　不，冰鸾剑也有变化，只不过极为微小。原本就深紫色的武魂，稍微深色了一丁点儿，几乎微不足道。
<br/>　　黄阶武魂的力量，对赤阶武魂来说，足够庞大。但对紫色武魂来说，却是显得无比弱小了，弱了百倍都不止。
<br/>　　“吸收别人的武魂力量，并加注到自身，提升自己的武魂力量，啧啧，冰鸾剑阿冰鸾剑，你的能力也太过变态了吧。”萧逸惊喜交加。
<br/>　　萧逸猜测，冰鸾剑应该是早就认可了自己这个主人了得。不过这是自己第一次将它凝聚出来，这才触发了冰鸾剑传承，并得到太阴太阳之眼这个能力。
<br/>　　下一秒，当两个无形漩涡消失，眼睛恢复正常的眼色，萧逸才立马反应过来。
<br/>　　“依依。”萧逸立刻检查起依依的伤势。
<br/>　　依依，早已意识模糊，腰背上的鲜血汩汩流着，流淌了一地。
<br/>　　“依依，别睡，快醒来。”萧逸一边叫唤着依依，一边为她捂住伤口。
<br/>　　萧逸以前常年在危险境地中行走，早已自学了一身不弱的医术。
<br/>　　“幸好，没有伤及要害，只要包裹好伤口....”萧逸刚刚松了口气。
<br/>　　忽然，依依的脸色转瞬间变黑，气息也越来越弱。
<br/>　　“不好，那是毒匕，毒性极强。哪怕止住了伤口，那毒性也能要了依依的命。”萧逸大急，这才想起来，杨伏的毒匕武魂拥有剧毒。
<br/>　　这时，在萧逸地不断叫唤下，依依艰难地睁开了眼睛，虚弱地呢喃了一句，“少爷”。
<br/>　　“少个屁阿。”萧逸呵斥道，“我不是跟你说了，有危险时不要老挡在我面前，你...哎..”
<br/>　　依依虚弱地笑了笑，道，“嗯，依依的命是少爷的，萧逸少爷风流倜傥英俊潇洒玉树临风绝世无双帅到掉渣，依依什么都听少爷的。”
<br/>　　依依的声音很低，说话也一字一顿，但萧逸却听得清清楚楚。
<br/>　　“你...你真的记下来了...”萧逸忽然想起来，今天晚饭之前，依依都魂不守舍地经常低声自言自语。
<br/>　　原来，她一整天都在回忆着他曾经说过的话，并试图记下来。
<br/>　　这时，依依再次闭上了眼睛。
<br/>　　毒匕武魂乃是黄阶武魂，它的毒性，对于依依这个没有任何修为的普通人来说，实在太强了。
<br/>　　“混蛋，睁开眼睛，老子不许你死。”
<br/>　　“你不是说什么都听我的吗，现在就给我醒来。”
<br/>　　“....”
<br/>　　不知道为什么，萧逸这个曾经的杀手界王者，一直冷酷异常的心，竟变得慌乱了起来。
<br/>　　“不能慌，我一定能想到办法救你的。”萧逸终究是心智过人，马上平伏下内心。
<br/>　　他很清楚，自己就算着急也没有任何卵用，倒不如去想救依依的办法。
<br/>　　“有了，形意五绝，鹤形，当年我受了再重的伤势都能治好自己。”
<br/>　　“不行，依依不会形意五绝。”
<br/>　　“我的医术....不行，这是武魂赋予的毒性，这个世界的炼药师或者能解，我的医术不可能奏效。”
<br/>　　萧逸头一次这么急，哪怕当年在地球孤身一人杀入大型佣兵团的老巢，独自面对无数敌人，他都能镇定如常，毫无慌张。
<br/>　　“有了。”萧逸忽然眼睛一亮，“太阴太阳之眼能吸收武魂力量。”
<br/>　　萧逸其实也不知道行不行，但他不会放过任何一个机会。
<br/>　　“凝。”萧逸低喝一声，霎那间，原本正常的眼睛，再次变为一黑一红，冰冷与炽热充斥其中。
<br/>　　通过之前脑海的记忆，他已经知道如何使用这个能力。
<br/>　　当他的眼睛看向依依时，顿时发现，依依体内正被一股黑漆漆的能量侵蚀着。
<br/>　　如果没猜错的话，这股力量，就是毒匕的毒性。
<br/>　　“给我吸。”萧逸低喝一声。
<br/>　　两个虚无的无形漩涡迸发而出，顷刻间将那股黑漆漆的能量吸收殆尽。
<br/>　　不过，令人奇怪的是，这一次，这些力量并没有被吸纳到他的武魂中，反而在无形漩涡中已经消失掉了。
<br/>　　这团黑漆漆的能量虽然也是武魂力量，不过已经转化成毒性了，萧逸如果吸收了，只会对自己的身体有害，而不会有任何益处。
<br/>　　这时，依依脸上的黑气已经散去，代表毒性已经消除。
<br/>　　“果然奏效了，毒性已除，接下来，只需包裹伤口即可。”萧逸笑了笑，松了口气。
