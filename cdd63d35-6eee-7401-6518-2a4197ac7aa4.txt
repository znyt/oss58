    两人持续朝前走了半小时，依然是一条没有岔路的通道，仿佛无穷无尽一般，陈兴终于沉不住气了。<br/>    <br/>    “这里到底有多大！”<br/>    <br/>    “我们没有听见枪声。”阿乔木说道。她的意思是，这里很安静，枪声能传播很远，但进来以后，他们没有听到其他队伍的动静。也就是说，这里的空间非常大，以至于他们和其他四个队伍都没有交集。<br/>    <br/>    “或许这是个独立空间。”陈兴说道。<br/>    <br/>    “你的意思是说，每一个雾门都通往不同的地方？”阿乔木问道。<br/>    <br/>    “差不多吧。”陈兴在通道中间坐下，然后招呼阿乔木过来，“我们分析一下，还有什么异常的地方。”<br/>    <br/>    “你发现没有，我们这一个小时内，没有再遇到活动的石像。”阿乔木在他身边坐下，说道。<br/>    <br/>    “嗯……”陈兴沉吟着，陷入了思绪。确实，他也注意到这一点，只是想不明白，为什么石像不再攻击他们。<br/>    <br/>    如果一个谜题无法解开，就要往更基础的地方寻找答案。<br/>    <br/>    比如说，一排石像之中，为什么只有几个能动？<br/>    <br/>    要解决这个问题，就要明确动机，再从动机逆推过程。动机是一切事件的缘由，可以从几个方面考虑，一是条件，二是利益，三是情感。<br/>    <br/>    在犯罪心理学上，首先要看嫌疑人有没有作案的能力和条件。如果没有，案件就不成立。但如果有，就要看利益，也就是嫌疑人在这件事情上有没有好处。如果没有好处，就要看情绪的影响，比如说报复、情杀，甚至是看不顺眼、心理变态等等。<br/>    <br/>    从条件的角度分析，巫术师们拥有成熟的技术，可以制造出大量的魔灵，所以在条件上没有问题。<br/>    <br/>    而从利益的角度分析，会动的石像肯定比不会动的石像造价高。于是，为了节省成本，巫术师们利用鱼目混珠的手法，让进攻者防不胜防。<br/>    <br/>    至此，动机和过程都变得清晰了，不需要再考虑情感方面的问题。<br/>    <br/>    那么，为什么混在其中的活动石像不再攻击他们？<br/>    <br/>    原因可以很多，但最直观的，就是石像无法对他们造成有效伤害。<br/>    <br/>    面对武力无法解决的对手，巫术师们是怎么考虑的？<br/>    <br/>    想要找到答案，就必须代入对方的思维。如果他是巫术师，他会怎么解决？<br/>    <br/>    他环视着四周，目光从一个个石像上扫过。它们面容一致，身材相同，仿佛同个模子里印出来的。唯一的区别就是手中的武器不同，但其实也只有剑盾、长枪、弓箭、双手斧四种。<br/>    <br/>    忽然之间，一个数学模型出现在他的脑海中。<br/>    <br/>    那是小学课本中，或者奥林匹克数学题中常见的“火柴模型”，就是移动其中一根，或是几根火柴，使公式或者图形成立。<br/>    <br/>    在他的眼中，高大的石像变成了一根根火柴。只要挪动其中几根，就能改变迷宫的走向。<br/>    <br/>    谜题解开，面对无法用武力解决的对手，巫术师们设下了第二重防御，利用石像的交替移动，把敌人困在迷宫里，拖延时间。<br/>    <br/>    “聪明的男人，你想到答案了吗？”似乎从陈兴的表情上看出了倪端，阿乔木问道。<br/>    <br/>    “你应该感到庆幸。”陈兴搭着阿乔木的肩膀，说道，“你站在了我的身边，而不是对立面。”<br/>    <br/>    阿乔木难得地露出了微笑，“能和你搭档，我一直感到很荣幸。”<br/>    <br/>    接下来，陈兴就把他的猜想告诉了阿乔木。在听的过程中，阿乔木的眼睛越来越亮，禁不住称赞道：“男人，你真的太聪明了！”她眼中幻彩迷离，就像个见到偶像的小女生，满脸崇敬之色。<br/>    <br/>    “那么，男人，我们接下来应该做什么呢？”阿乔木饶有兴致地问道。<br/>    <br/>    陈兴没有回答，而是取出了“夜枭”。<br/>    <br/>    “过来。”他命令道。<br/>    <br/>    阿乔木一脸疑惑。<br/>    <br/>    “躺下。”他再次命令道。声音平和淡漠，却充满了难以抗拒的力量。阿乔木的表情渐渐凝固，变得有些害怕。<br/>    <br/>    “我缺个枪架。”他用狙击枪比划了一下，说道。<br/>    <br/>    阿乔木只好趴了下来。<br/>    <br/>    “仰躺。”陈兴又提出了要求。<br/>    <br/>    阿乔木犹豫了好一会儿，动作僵硬地翻过身体，仰躺在地上，双手下意识地护住胸部。那棕色的眼睛里，闪烁着无助的光芒。<br/>    <br/>    “手拿开！”陈兴提高了音量，见她没动，又踢了下她的屁股。虽然力气不大，鞋尖却传来惊人的弹性，就像踢在皮球上。<br/>    <br/>    “快点！”陈兴皱起眉头，催促道。<br/>    <br/>    最后，阿乔木认命般地闭上双眼，挪开了双手。<br/>    <br/>    见她敞开胸腹，陈兴嘴角一勾，露出邪邪的笑容。在她身上趴下，下巴枕着她的腹部，枪管从中线穿过，架在胸骨上。<br/>    <br/>    这个姿势对上面的人来说十分舒服，但对下面人来说，无疑是巨大的煎熬。除了要承受上面的人的重量外，还要承受来自枪械的反作用力。不过以她准镇守级的身体素质来说，都是小问题，顶多有些疼痛罢了。<br/>    <br/>    陈兴移动枪口，瞄准二三十米外一个石像的腰部，逐渐压缩灵能，然后输入到子弹中。<br/>    <br/>    “别动。”<br/>    <br/>    胸腔的起伏带动了枪管，枪口上下浮动。阿乔木只好配合地屏住呼吸，让自己的身体成为合格的枪架。<br/>    <br/>    一分多钟后，蓄能终于完成，陈兴扣下了扳机。<br/>    <br/>    “嘭！”<br/>    <br/>    枪管向后一挫，湛蓝的火舌喷吐而出，前方石像立即拦腰折断，巨大的身躯向后倒塌，发出沉闷的轰隆声。<br/>    <br/>    子弹出膛的瞬间，阿乔木吓得大叫一声，胸口剧烈起伏。<br/>    <br/>    陈兴“咔嚓”的一声，拉动枪栓，退出冒着青烟的弹壳，重新塞入一发钢芯穿甲弹，瞄准倒塌石像旁边的一个，再次压缩灵能，灌入弹头，射击！<br/>    <br/>    打掉两个石像后，两人从缺口处进入另一条通道。陈兴再次要求阿乔木躺下，做他的人肉枪架。<br/>    <br/>    陈兴的计划很简单，摧毁挡路的石像，做直线运动。<br/>    <br/>    根据他以前看过的资料，人造空间会遭到主空间的排斥，就像在水中开辟一个真空点，无时无刻都要面对水的压强。为了维系真空点的存在，需要一个持续的作用力去抗衡水压。<br/>    <br/>    而空间和水不一样，水是物质，空间是时间和能量的结合体。主空间和人造空间会发生能量对冲，以此产生平衡。<br/>    <br/>    牛顿的能量守恒定律适用于整个多元宇宙，往往越大的人造空间就需要越多的能量支撑。所以，巫术师们不可能制造太大的空间，否则光维持其存在就会消耗非常大的能量。<br/>    <br/>    哪怕再大的能量，也有耗尽的一天。另外，在开辟人造空间时，需要一股非常庞大的能量，这股能量决定了空间的大小。如果这里的巫术师有能力制造出超大的空间，就不至于被毁灭了。<br/>    <br/>    由于姿势的问题，陈兴几次忍不住想把阿乔木就地正法。可他比较爱干净，没有“打野战”的习惯，总感觉这里太脏，邪火刚起来又灭下去。<br/>    <br/>    他唯一一次在野外，就是在前往龙石镇的路上，跟想要刺杀他的波娜。但那次他也洗了澡，帐篷里又垫了厚厚的毯子，干净舒服。<br/>    <br/>    至于跟苏娜在地穴里，一方面入口就在他的房间，他是洗了澡才过去的，另一方面，苏娜结了白色的大茧，里面很清洁，就像厚厚的蚕丝，不存在不干净的问题。<br/>    <br/>    他已经半个多月没洗澡了，全身黏糊糊的，实在没心情做那事儿，否则阿乔木早就被他干掉了。<br/>    <br/>    两个小时后，陈兴击倒了四十多个石像，进入一条新的通道。<br/>    <br/>    一开始他还没觉察，当子弹击穿目标、散落一地铜绿时，他才意识到，石像变成了青铜像。<br/>    <br/>    定眼看去，这些青铜巨像高矮不一、形态各异，而且表面存在不同程度的损坏，更像是青铜魔灵的仓库、陈列室，或是坟场。<br/>    <br/>    陈兴不敢靠近，保持着距离观察。他渐渐发现，这些魔灵大致上可以分为四种。<br/>    <br/>    像竹竿人一样，修长灵活的，像猩猩一样，敦厚壮实的，还有肚子圆鼓鼓、胖子模样的，以及八条尖腿，介于蜘蛛和章鱼之间，中间有颗独眼的怪物。<br/>    <br/>    陈兴不由得想起公国长老利安德尔·拉尔夫的两只机械傀儡，“安加玛索”和“奥加玛索”，安加玛索很像第一种，而奥加玛索很像第二种，只是它们的体型更为庞大。<br/>    <br/>    “玛索”在古语中就是“傀儡”的意思，和这些青铜魔灵是同类别的东西。<br/>    <br/>    在分类上，魔灵一般特指“使用混沌魔能驱动的傀儡”，而傀儡特指“使用非常规能源驱动的机器人”。简单来说，傀儡就是机器人中的一种，而魔灵则是傀儡中的一种。<br/>    <br/>    “休息一个小时。”观察了一会儿，陈兴退到后方，取出灵能药剂灌入口中。<br/>    <br/>    刚才的射击消耗了他一半的灵能，需要补充完再探索。<br/>    <br/>    可刚坐下没多久，不远处闪过一道红光，一只独眼魔灵动了。<br/>    <br/>    “咯啦啦……”<br/>    <br/>    随着关节的扭动，铜锈像雨点般落下。<br/>    <br/>    “小心！”陈兴话音刚落，独眼魔灵就转动着身躯，朝这边“望”了过来。<br/>    <br/>    下一个瞬间，独眼泛起橙红的光芒，“滋啦”的一声，射出一道拇指粗的激光。阿乔木迅速卧倒，而陈兴没感觉到危险，所以站在原地没动。激光从他头顶一扫而过，后方立即传来焦糊的味道。转头看去，一排石像被切成两截，齐齐倒塌。<br/>    <br/>    <br/>
