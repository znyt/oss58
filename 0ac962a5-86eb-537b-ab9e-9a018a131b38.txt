　　佟千雪用钥匙刚把手腕和脚腕上的镣铐解开，布克尔特就推开门从门外走了进来，看到已经解开镣铐恢复了自由的佟千雪，眼神陡然变得凌厉起来，他眼睛中的锐利，让佟千雪感觉到自己的皮肤都是有些刺痛。
<br/>　　“你们果然有事情瞒着我。”
<br/>　　布克尔特不认为佟千雪自己有本身能解开镣铐，她能解开镣铐只有娜塔莉娅给了她钥匙这一种可能。不过佟千雪有没有解开镣铐对布克尔特来说已经无关紧要了，他一把抓住佟千雪的手臂把她的双手反剪到身后，用塑料手铐捆住她的双手，把她带出了木屋。
<br/>　　“是佟千雪！！”
<br/>　　司徒空看到了被布克尔特从屋子里带出来的佟千雪，拉开反曲弓，试图一箭击毙布克尔特救下佟千雪；可是布克尔特却是早就有所准备，他一半身体被建筑所遮挡，并且把佟千雪挡在他身体的另一侧，让司徒空没有可以进攻的机会。
<br/>　　“妈的！这家伙走位角度实在太刁钻了。”
<br/>　　司徒空尝试了几次依旧是没有能抓到合适的进攻机会，箭镞中含有剧毒，如果不小心射中佟千雪，这次营救行动就彻底失败了。
<br/>　　司徒空把架在反曲弓上的毒箭送给了一个端着M4A1试图偷袭的武装毒贩，毒箭入体，武装毒贩瞬间毒发毙命！
<br/>　　佟千雪看到了村子中的战术中队，她挣扎，她反抗，试图摆脱布克尔特的控制，但是在力量上，佟千雪跟佟千雪完全不是一个级别的，不管她如何努力的抗争都不能从布克尔特手中逃脱出来。
<br/>　　“拦住他们！”
<br/>　　布克尔特对着正在抵挡着战术中队进攻的武装毒贩命令道：“一定要拦住他们，绝对不能让他们接近矿洞！”
<br/>　　这个村子后山有个矿洞，是以前居住在这里的村民留下的，布克尔特把这个村子控制之后，花了很大的人力将这个矿洞打通，贯通这座矿山通到矿山背后。
<br/>　　布克尔特在这个矿洞中留下了摩托车和武器，作为自己最后的逃生通道，穿过这座矿山，背后是一片山林，在那里布克尔特建造了安全屋，里面有食物、武器、药品和通讯工具。布克尔特在这里的确是经营了很长时间，也做了很多的准备用来应对各种紧急情况。
<br/>　　巴坦伦的死对布克尔特造成的影响很大，让布克尔特瞬间从优势的一方变成了劣势的一方，失去了巴坦伦手中军队的支援，布克尔特的武装力量根本不足以对抗龙隐部队和比洛多的机动部队。
<br/>　　而且巴坦伦一死，比洛多肯定会对巴坦伦以前的部队进行收编重整，等到比洛多完成这些事情的时候，布克尔特要面对的可就不仅仅只是机动部队，而是整个联合政府军了！所以布克尔特必须想办法尽快脱困离开南美洲，不抓紧时间跑的话，他连跑都没有时间跑了！
<br/>　　现在在布克尔特手中还剩下一张牌，最后一张牌，就是当时从西伦科特手中得到的那支P病毒！
<br/>　　西伦科特手中总共有三支改进型的P病毒，他把其中一支交给了布克尔特，自己带走了两支。在货轮上原本詹姆斯想要下杀手拿走那两支P病毒，但是因为蛰龙的突然出现，詹姆斯的计划破产，P病毒也被蛰龙带走。
<br/>　　目前为止，这种改进后的P病毒除了“暗月”之外，有可能只有中方掌握，美国政府很迫切的想要得到一支改进型的P病毒进行研究，尽可能的对其研究透彻，除了要找到如何对抗这种病毒，还要搞清楚这种具有超高感染性，感染之后死亡率超过百分之九十的超级病毒是如何研究出来的。这种致命的生物武器不能掌握在除了自己以外的人手中！
<br/>　　鹰酱自己手中拥有生物武器那是“合法”的，而别人拥有，那就是“恐怖主义行为”，需要受到制裁！这就是鹰酱的流氓思维和流氓理论！
<br/>　　“Boss，种植园那边全都完了，比洛多使用了燃烧弹，把我们的种植园全都烧毁了。”
<br/>　　前“绿色贝雷帽”特种部队队员，后来也是因为西伦科特的关系加入布克尔特贩毒集团的诺林顿带着娜塔莉娅走进了矿洞，对布克尔特说道：“另外，我们的加工工厂也遭到了攻击，厂房设备和仓库被导弹炸了。我们……”
<br/>　　诺林顿最后一句话并没有说下去，可即便他不说，布克尔特也知道他要说什么，无非就是我们在这里经营的一切全都完了。
<br/>　　这里完蛋对布克尔特来说的确是伤筋动骨，但还不足以让布克尔特到山穷水尽的地步，布克尔特在瑞士银行有一个存款金额不小于一亿美金的账户，他还有一支可以用来跟美国政府进行谈判的P病毒！
<br/>　　布克尔特把佟千雪推到矿洞角落中，在心中快速做着权衡，最后他还是决定用自己手中这最后一张牌跟美国政府进行谈判，用一支对方一直想要得到的生物病毒武器，来保全自己的安全，布克尔特相信美国政府是愿意进行这次谈判的。
<br/>　　从地上拿起一个头盔戴在头上，然后抓起佟千雪把她横放在一辆越野摩托车的车座上，对诺林顿说道：“这里布置一下，然后去安全屋汇合！”
<br/>　　“这个女人怎么办？”诺林顿对布克尔特问道。
<br/>　　“把她留在这里就行了。”布克尔特看了眼娜塔莉娅，回答道。
<br/>　　娜塔莉娅对于布克尔特已经不存在价值了，带着她只是给自己增加负担和累赘。
<br/>　　诺林顿把娜塔莉娅推进矿洞一个阴暗的角落中，然后利用存放在矿洞中的手雷和步兵地雷进行了一些布置，跨上矿洞剩下的那辆越野摩托跟上了已经先行离开的布克尔特。
<br/>　　随着比洛多的机动部队赶到，村子中的武装毒贩防御呈现出溃败的迹象，五分钟后，村子中的武装毒贩被尽数歼灭，机动部队开始在村子中搜索躲藏起来的毒贩，还有消失不见的布克尔特和佟千雪。
